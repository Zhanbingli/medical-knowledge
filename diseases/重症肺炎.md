# 重症肺炎(Severe Pneumonia)

> [!summary] 30 秒摘要
> **红旗**:意识改变 / RR ≥ 30 / SpO2 < 90% / 多叶浸润 / SBP < 90
> **首查**:**CURB-65 / PSI 评分** + 血气 + CXR + 血培养 + 痰培养(抗生素前)+ 病原学(流感 / COVID PCR / 尿抗原)
> **一线**:**1h 内首剂抗生素**(经验性)+ 氧 / NIV / 插管 + 复苏;严重 → ICU
> **升级**:CURB-65 ≥ 3 / 任何 ICU 标准 / 脓毒性休克 → ICU + 1h bundle

---

## 一句话定义

肺实质急性感染,涉及肺泡 / 间质 / 终末细支气管。重症定义:满足 **1 项主要标准**(机械通气 / 升压药)或 **≥ 3 项次要标准**(IDSA/ATS 2007/2019)。

来源:IDSA / ATS Community-Acquired Pneumonia in Adults 2019;中国成人社区获得性肺炎诊断和治疗指南 2016(中华结核和呼吸杂志 / 中华医学会呼吸病学分会);中国医院获得性肺炎与呼吸机相关性肺炎诊断和治疗指南 2018。

## 分型(决定经验性抗生素方向)

| 类型 | 定义 | 主要病原 |
|---|---|---|
| **社区获得性肺炎(CAP)** | 院外或入院 < 48h | 肺炎链球菌、流感嗜血杆菌、肺炎支原体、衣原体、军团菌、流感 / COVID;重症 + 结构性肺病 → 铜绿 |
| **医院获得性肺炎(HAP)** | 入院 ≥ 48h 后 | 革兰阴性(铜绿、肠杆菌科、ESBL、CRE)、MRSA |
| **呼吸机相关肺炎(VAP)** | 机械通气 ≥ 48h 后 | 同 HAP + 铜绿 / 不动杆菌为主 |
| **吸入性肺炎** | 误吸危险 | 厌氧菌 + 革兰阴性 / 革兰阳性混合 |
| **免疫抑制者肺炎** | HIV / 化疗 / 移植 / 高剂量激素 | 上述 + PCP / CMV / 真菌 / 结核 / 非典型分枝杆菌 |

## 重症定义(IDSA/ATS 2019)

### 主要标准(任一)
- **机械通气**(有创)
- **脓毒性休克需升压药**

### 次要标准(≥ 3 项)
- RR ≥ 30
- PaO2/FiO2 ≤ 250
- 多叶浸润
- 意识改变
- BUN ≥ 20 mg/dL(7.1 mmol/L)
- WBC < 4
- 血小板 < 100
- 体温 < 36℃
- 低血压需积极复苏

---

## 🚩 红旗(立刻警觉)

- **CURB-65 ≥ 3** 或 PSI IV-V 级
- 意识改变(老年常以谵妄起病)
- RR ≥ 30 / SpO2 < 90% / PaO2/FiO2 ≤ 250
- SBP < 90 / 心动过速 + 末梢灌注差
- **乳酸 > 2** 不能解释
- 多叶 / 双侧肺部浸润
- **已诊断流感 / COVID-19 + 影像迅速进展** → 警觉病毒性肺炎 / ARDS
- 免疫抑制 + 发热 → 警觉机会感染 + FN
- 老年 / 长期卧床 / 痴呆 + 误吸高危 → 吸入性病情进展快

### 立即(到院 1h 内)

- [ ] 生命体征 + qSOFA / NEWS2 / **CURB-65**

- [ ] **乳酸**(脓毒症怀疑时)

- [ ] **CXR**(立位 / 床旁)

## CURB-65(分诊核心)

| 项目 | 1 分 |
|---|---|
| **C**onfusion(意识改变) | ✓ |
| **U**rea(BUN > 7 mmol/L / 20 mg/dL) | ✓ |
| **R**espiratory rate ≥ 30 | ✓ |
| **B**lood pressure(SBP < 90 或 DBP ≤ 60) | ✓ |
| 年龄 ≥ **65** | ✓ |

**分层**:
- 0-1 → 门诊治疗(死亡率 < 3%)
- 2 → 短期住院 / 观察(死亡率 ~ 9%)
- ≥ 3 → 住院 / 考虑 ICU(死亡率 15-40%)
- 4-5 → ICU

来源:Lim WS et al. Thorax 2003;BTS / IDSA 沿用。

## 一线处理(经验性,1h 内首剂抗生素)

> [!danger] 重症肺炎 = 脓毒症的常见源
> 1h 内抗生素;不应等培养结果。同时启动 [[emergencies/脓毒症|脓毒症]] 1h bundle。

### 0. 复苏(重症)

- 晶体液(乳酸林格)
- 升压药维持 MAP ≥ 65(脓毒性休克 → [[drugs/去甲肾上腺素|去甲肾上腺素]])
- 监测尿量 / 乳酸 / 末梢

### 1. 经验性抗生素(按场景)

#### 社区获得性肺炎(CAP)

| 严重度 / 场景 | 经验首选 |
|---|---|
| **门诊**(无合并症) | 阿莫西林 / 大环内酯(中国常用)/ 多西环素;**已知大环内酯耐药 → 氟喹诺酮** |
| **门诊**(合并症) | β-内酰胺 + 大环内酯 或 单用呼吸氟喹诺酮(左氧 / 莫西沙星) |
| **住院非 ICU** | **头孢曲松 + 阿奇霉素** 或 单用呼吸氟喹诺酮 |
| **ICU(重症 CAP)** | **头孢曲松 / 头孢噻肟 + 阿奇霉素 / 氟喹诺酮**;**铜绿风险**(结构性肺病 / 既往气道菌)→ 哌拉他唑 / 头孢吡肟 + 氟喹诺酮;**MRSA 风险**(影像空洞 / 流感后)→ 加 **万古 / 利奈唑胺** |

> 来源:IDSA / ATS CAP 2019;中国 CAP 2016。

#### 医院获得性肺炎(HAP)/ VAP

- **覆盖铜绿 + ESBL** 是默认起点
- **哌拉西林他唑巴坦 / 头孢吡肟 / 美罗培南**(根据当地耐药)
- MRSA 风险 → **加万古 / 利奈唑胺**
- 重度 / MDR 高危 → 双覆盖革兰阴性

> 来源:IDSA / ATS HAP/VAP 2016;中国 HAP/VAP 2018。

#### 流感 / COVID-19

| 病原 | 处理 |
|---|---|
| **流感**(疑似 / 确诊) | **奥司他韦 75 mg PO bid × 5 天**(住院重症可延长 / 加倍) — 越早越好,< 48h 启动 |
| **COVID-19** | 按当前共识(瑞德西韦 / 帕罗韦德 / 类固醇按指南)[需查证: 当前年最新共识] |
| 病毒性肺炎 + 细菌叠加 | 同时覆盖 |

#### 吸入性肺炎

- 哌拉他唑 / 厄他培南 / 头孢曲松 + 甲硝唑(若怀疑厌氧)
- IDSA 2019 不强调常规加厌氧覆盖,除非脓肿 / 牙周病严重

#### 免疫抑制 / PCP 怀疑

- **复方磺胺甲噁唑(TMP-SMX)** 高剂量 + **类固醇**(PaO2 < 70 时)
- 真菌覆盖按风险

### 2. 支持治疗

- **氧 / 高流量氧疗 / NIV / HFNC** 按 SpO2 / PaO2/FiO2
- 不稳定 / NIV 失败 → 气管插管 + 机械通气(肺保护性 6 mL/kg PBW)
- VTE 预防、应激溃疡预防、营养
- 血糖控制 7.8-10 mmol/L

### 3. 类固醇(选择性)

| 情境 | 推荐 |
|---|---|
| 严重 CAP + 脓毒性休克 | **氢化可的松 200 mg/d × 7 天**(SSC 2021;部分研究死亡率改善) |
| COVID-19 重症需氧 | **地塞米松 6 mg PO/IV qd × 10 天**(RECOVERY trial 2020) |
| PCP + PaO2 < 70 | 泼尼松 40 mg bid × 5 天后递减 |
| 一般 CAP | **不推荐常规**(EAP-CAP 等 trial 异议) |

### 4. 病原确认后调整

- 培养出敏感菌 → **24-48h 内降阶梯**
- 完成 5-7 天疗程(轻 / 中度 CAP)
- 重症 CAP / VAP / 脓肿 → 按指南延长

## 🔺 升级标准

> [!danger]
> - **CURB-65 ≥ 3** → 住院考虑 ICU
> - 任一 IDSA 主要标准(机械通气 / 升压药)→ ICU
> - PaO2/FiO2 ≤ 250 / SpO2 < 90% 高流量不能维持
> - 脓毒性休克(SBP < 90 + 乳酸 > 2 / MAP 不能维持)
> - 多叶浸润快速进展(< 24-48h)
> - 影像出现空洞 / 脓肿 / 大量胸腔积液
> - 老年 / 免疫抑制 + 任何不稳定征象
> - 不响应 48-72h 抗生素 → 重新评估病原
> - 任何"病人在恶化"

## ⚠️ 常见坑

> [!warning]
> 1. **抗生素延迟 > 1h** → 重症 CAP 死亡率显著升;别等检查
> 2. **没抽血培养就给抗生素** → 病原诊断率低;重症必抽 ×2
> 3. **影像阴性就排除** → 早期(< 24h)/ 脱水 / 免疫抑制可阴性;高度怀疑 → CT
> 4. **老年肺炎仅以谵妄起病** → 没听诊 / 没拍片漏诊
> 5. **吸入性肺炎漏诊** → 误吸高危 + 右下肺浸润
> 6. **流感 + 细菌叠加(尤其金葡 / MRSA)漏诊** → 流感后突然恶化、空洞、休克 = 警觉
> 7. **没问免疫抑制状态** → PCP / 真菌 / 结核漏诊
> 8. **HAP / VAP 用 CAP 方案** → 革兰阴性铜绿 / MDR 漏掉
> 9. **常规给类固醇** → 仅特定指征(休克 / COVID-19 缺氧 / PCP)
> 10. **抗生素响应就停太早** → 重症 / 脓肿 / VAP 需足疗程
> 11. **结核漏诊** → 慢性 + 消瘦 + 上叶 / 空洞,反复"治不好的肺炎"
> 12. **影像不复查** → 重症 / 不响应 / 老年应在 6-8 周复查排除肿瘤;部分需更早
> 13. **胸腔积液漏穿** → 重症伴积液应早穿确认是否脓胸
> 14. **儿童 / 妊娠用药未调整** → 氟喹诺酮在儿童 / 孕妇有限制
> 15. **不查 HIV** → 任何重症肺炎 + 不典型 / 反复 → 降低门槛
> 16. **PCT / CRP 单独定治疗时长** → 应结合临床 + 影像;PCT 在某些研究指导停药
