# 脑出血(Intracerebral Hemorrhage, ICH)

> [!summary] 30 秒摘要
> **红旗**:突发头痛 + 局灶定位 + 意识改变;**头 CT 即刻确诊**
> **首查**:头 CT(< 25 min);CTA(警觉血管畸形)+ 凝血 + 评估抗凝史
> **一线**:**血压 SBP 140**(INTERACT-2 / ATACH-2);**逆转抗凝** + 颅高压管理 + 神外评估
> **升级**:任何怀疑立即神外会诊;**前 24 小时血肿扩大风险最高**

---

## 一句话定义

非创伤性脑实质内出血 → 占所有卒中 ~ 15-20%,**死亡率最高**(30 天 30-50%)。

来源:AHA / ASA Guidelines for the Management of Spontaneous Intracerebral Hemorrhage 2022;中国脑出血诊治指南 2019(中华神经科杂志);ESO ICH Guidelines 2022。

## 病因

| 类别 | 机制 / 部位 | 占比 |
|---|---|---|
| **高血压性 ICH** | 基底节 / 丘脑 / 脑桥 / 小脑深部小动脉(Charcot-Bouchard 微动脉瘤)| 50-60% |
| **脑淀粉样血管病(CAA)** | 老年 + 皮层 / 皮层下 浅叶出血 | 20-30%(老年) |
| **抗凝相关** | 华法林 / DOAC / 抗血小板 | 5-15%(抗凝时代上升) |
| **血管畸形 / 动脉瘤** | AVM / 海绵状血管瘤 / 动脉瘤破入脑实质 | < 10%(年轻者多见) |
| **肿瘤出血** | 转移瘤(黑色素瘤、绒癌、肾细胞)/ 胶质瘤 | < 5% |
| **静脉血栓形成** | 出血性静脉梗死 | 罕见 |
| **可卡因 / 苯丙胺** | 高血压 + 血管炎 | 罕见,年轻人 |
| **凝血障碍** | 血小板 < 30 / 凝血因子缺乏 / DIC | 罕见 |

> [!tip] **部位提示病因**
> - 基底节 / 丘脑 / 脑桥 / 小脑深部 → 高血压性
> - 浅叶(顶叶 / 颞叶 / 枕叶)→ CAA(老年)/ AVM(年轻)
> - 多发 / 微出血 → CAA / 血管炎
> - 不规则形 / 多灶 → 肿瘤出血

## 🚩 红旗(立刻警觉)

- 突发头痛(可不剧烈)+ 局灶神经定位 + 意识改变
- 高血压病史 + 突然剧烈头痛 + 呕吐
- 已抗凝 + 任何新发神经症状
- 老年 + 反复浅叶出血(CAA)
- 入院后 GCS 进行性下降(血肿扩大或脑疝)
- 库欣三联(高血压 + 心动过缓 + 不规则呼吸)→ 颅高压 / 即将脑疝
- 瞳孔不等大 + 一侧光反射消失 → 颞叶钩回疝

## 首查(到院 25 分钟内 CT)

- [ ] 生命体征 + GCS + NIHSS + 瞳孔
- [ ] 静脉通路 ×2 + 心电监护 + SpO2
- [ ] **头 CT 平扫**(确诊 + 部位 + 血肿体积估算 ABC/2)
- [ ] **CTA**(若怀疑血管畸形 / 动脉瘤 / 年轻 / 不典型部位)
- [ ] CBC + 电解质 + 肌酐 + 肝功 + 凝血(PT / INR / APTT) + 血型 + 交叉
- [ ] 抗凝药物史:华法林 / DOAC / 抗血小板 / 肝素 / 溶栓
- [ ] 血糖、毒物筛查(可卡因)、妊娠试验
- [ ] ECG + 肌钙蛋白(神经源性心肌损伤)

### 影像评估关键点

- **血肿体积 ABC/2 法**(估算):A × B × C / 2(三轴最大径,cm),> 30 mL 高死亡率
- **Spot sign**(CTA)= 活动性出血,血肿扩大风险极高
- 评估周围水肿、占位效应、中线移位、脑积水

## 一线处理

### 1. 血压控制(关键!)

> [!danger] **SBP 目标 140 mmHg**(AHA 2022 推荐;前提是基线 SBP 150-220)
> - **INTERACT-2 (NEJM 2013)**:积极降至 140 安全,可能改善功能预后
> - **ATACH-2 (NEJM 2016)**:SBP 110-139 vs 140-179 — 急性肾损伤增加,但血肿扩大无差
> - **现代推荐**:平稳降至 SBP 140(避免快速降至 < 130 致急性肾损伤);维持 24h

**用药**:
- **拉贝洛尔** 10-20 mg IV q10min(最大 300 mg) / 持续 1-2 mg/min
- **尼卡地平** 5 mg/h IV,每 5 min 加 2.5 mg/h(最大 15 mg/h)
- 避免硝普钠(增加 ICP)

来源:AHA / ASA ICH 2022;中国脑出血诊治指南 2019;具体本院药品规格 [需查证]。

### 2. 反转抗凝 / 抗血小板(立即!)

| 药 | 处置 |
|---|---|
| **华法林** | **维生素 K 10 mg IV**(慢推 ≥ 30 min)+ **4F-PCC**(凝血酶原复合物)25-50 U/kg 优于 FFP |
| **达比加群** | **idarucizumab 5 g IV**(REVERSE-AD 2017) |
| **Xa 抑制剂**(利伐沙班 / 阿哌沙班 / 艾多沙班) | **andexanet alfa**(若可得;ANNEXA-4)/ **4F-PCC 50 U/kg** |
| **肝素** | 鱼精蛋白 1 mg / 100 U UFH |
| **LMWH** | 鱼精蛋白部分中和(~ 60%) |
| **抗血小板**(阿司匹林 / 氯吡格雷) | **不能逆转**;PATCH trial 提示输血小板**反致预后差**,**不推荐常规输血小板** |
| **rt-PA 溶栓后 ICH** | 冷沉淀 10 U + 氨甲环酸;血小板可考虑 |

> 来源:AHA / ASA ICH 2022;PATCH NEJM 2016;REVERSE-AD;ANNEXA-4;具体本院可用性 [需查证]。

### 3. 颅高压管理

- 抬头 30°
- 镇静镇痛(芬太尼 / 丙泊酚)避免烦躁致 ICP 升
- **甘露醇 0.25-1 g/kg IV q6h** / **3% 高渗盐水 100-250 mL**
- 严重 / 即将脑疝 → 紧急神外:EVD / 去骨瓣 / 血肿清除
- ICP 监测(Glasgow 标准 / 严重出血)

### 4. 神外手术指征

| 情境 | 处置 |
|---|---|
| **小脑出血 > 3 cm 或脑干压迫 / 脑积水** | **紧急去骨瓣减压 + 血肿清除**(I 类推荐) |
| 幕上 ICH | 多数研究显示常规手术不优于内科;**MISTIE III(微创立体定向血肿引流)**特定病例可获益 |
| 脑积水 | EVD |
| 大面积 + 危及生命 | 个体化(去骨瓣 ± 血肿清除) |

来源:STICH I/II;MISTIE III;AHA / ASA 2022;中国脑出血指南 2019。

### 5. 抽搐 + 防 VTE

- 出现抽搐 → 苯二氮䓬 + 长期抗癫痫(左乙拉西坦)
- **不推荐常规预防性抗癫痫**(无明确发作时)
- **VTE 预防**:机械预防即起;**药物(LMWH)血肿稳定后 24-48h 启动**(权衡)

### 6. 血糖 + 体温

- 血糖目标 7.8-10 mmol/L(避免 < 3.9 或 > 11.1)
- 体温 > 38℃ → 退热

## 🔺 升级标准

> [!danger]
> - 任何 ICH → 立刻神外会诊
> - GCS 进行性下降 ≥ 2 → 急 CT(警觉血肿扩大 / 脑积水 / 脑疝)
> - **小脑出血 > 3 cm** → 紧急手术评估,可数小时内进展为脑干压迫
> - 库欣三联 / 瞳孔变化 → 即将脑疝 → 立即降颅压 + 神外
> - 抗凝 + ICH → 立即逆转抗凝(不等神外)
> - 大面积 + 不响应内科 → ICU + 手术评估
> - 任何"病人在恶化"

## ⚠️ 常见坑

> [!warning]
> 1. **降压目标 < 130 太快** → 急性肾损伤(ATACH-2)+ 灌注不足
> 2. **没逆转抗凝** → 血肿持续扩大;华法林 INR 不复测仍以为已纠正
> 3. **抗血小板出血常规输血小板** → PATCH trial 显示**反预后差**(不推荐)
> 4. **rt-PA 溶栓后 ICH 没区别处理** → 冷沉淀 + 氨甲环酸,与自发性 ICH 不同
> 5. **小脑出血"先观察"** → > 3 cm 或脑干压迫数小时内进展,**必须急诊神外**
> 6. **认为"幕上 ICH 都不需要手术"** → MISTIE III / 大面积 / 危及生命仍可个体化
> 7. **预防性抗癫痫** → 已不推荐;有发作再用
> 8. **早期 LMWH 启动 VTE 预防** → 增加血肿扩大,需等血肿稳定 24-48h
> 9. **CTA 没做漏诊血管畸形 / 动脉瘤** → 年轻 / 浅叶 / 不典型部位 → CTA 必做
> 10. **不查毒物筛查 / 妊娠** → 可卡因 ICH / 子痫 / HELLP 漏诊
> 11. **认为"GCS 15 不严重"** → 早期可保留意识,数小时血肿扩大后崩溃
> 12. **CAA 老年反复出血未识别** → 多次浅叶 / 微出血,慎用抗凝
> 13. **降压药选硝普钠** → 升 ICP,不推荐
> 14. **脑积水漏诊** → 第三 / 四脑室出血常并发,需 EVD
> 15. **入科后没复查 CT 监测扩大** → 24h 内复查;符号变化即重做
