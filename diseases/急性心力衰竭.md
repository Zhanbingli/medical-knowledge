# 急性心力衰竭(AHF / ADHF)

> [!summary] 30 秒摘要
> **红旗**:气短 + 端坐 + JVP↑ + 湿啰音 + 水肿;**先按"湿/干 × 暖/冷"4 象限分**
> **首查**:**ECG + BNP/NT-proBNP + CXR + 床旁超声(B 线 / EF / IVC)** + hs-cTn + 电解质
> **一线**:湿暖 → 利尿 + 扩血管 + NIV;湿冷 → **去甲肾上腺素 + 多巴酚丁胺**(不给硝酸甘油 / 大利尿)
> **升级**:心源性休克 / NIV 失败 / 急性瓣膜并发症 / STEMI 合并 → 立即 ICU + 心内 / 心外

---

## 一句话定义

**心脏不能维持机体所需供血** 的急性发病 / 急性失代偿。

**[[emergencies/急性肺水肿|急性肺水肿]] ≠ 急性心衰**:急性肺水肿是 AHF 的常见**表型**(湿暖型),但 AHF 还包括:
- ADHF(慢性心衰急性失代偿)— 最常见,~ 70%
- 急性肺水肿(快速发作)
- **心源性休克**(湿冷型)— 死亡率最高 30-50%
- 右心衰竭(PE / RV 梗死后)
- **高输出量心衰**(甲亢 / 贫血 / 脓毒症 / 动静脉瘘 / 妊娠)

来源:ESC Acute and Chronic Heart Failure Guideline 2021;ESC HF Focused Update 2023;AHA / ACC / HFSA Heart Failure Guideline 2022;中国心力衰竭诊断和治疗指南 2018(中华心血管病杂志)。

## 4 象限表型(决定治疗,**最重要的临床框架**)

| | **暖**(灌注好) | **冷**(灌注差) |
|---|---|---|
| **湿**(充血) | **湿暖型**(~ 70%)<br>急性肺水肿 / ADHF<br>→ 利尿 + 扩血管 + NIV | **湿冷型**(~ 20%)<br>**心源性休克**<br>→ 升压药 + 正性肌力 |
| **干**(无充血) | 干暖型(代偿期,不在急性) | **干冷型**(~ 5%)<br>低输出 + 容量不足<br>→ 谨慎补液 + 正性肌力 |

**灌注差(冷)征象**:皮肤湿冷 / 末梢花斑 / 神志改变 / 尿量↓ / 乳酸↑ / 脉压窄
**充血(湿)征象**:端坐 / JVP↑ / 湿啰音 / S3 / 肝大 / 下肢肿 / 腹水

来源:Stevenson LW, Eur J HF 1999(经典);ESC HF 2021 沿用。

## 🚩 红旗(立刻警觉)

- 端坐呼吸 + 不能平卧
- 双肺湿啰音 + 粉红泡沫痰
- JVP↑ + S3 奔马律 + 双下肢凹陷性水肿
- SpO2 < 90% / RR > 30
- **冷休克征象**(湿冷型)→ 心源性休克
- SBP < 90 + 灌注差 + 乳酸 > 2
- ECG 急性 ST 改变 / 新发心律失常
- **新发心脏杂音**(警觉急性 MR / VSR / 夹层)
- 妊娠晚期 / 产后 → 警觉围产期心肌病
- 化疗后 / 放疗后 → 警觉化疗相关心肌病

## 首查

### 立即(到床后 10 分钟内)

- [ ] 生命体征 + SpO2 + qSOFA / NEWS2
- [ ] **12 导联 ECG**(STEMI / 心律失常 / RV 应变)
- [ ] 静脉通路 ×2 + 心电监护
- [ ] **CXR**(肺水肿 / 心影 / 胸腔积液 / 主动脉)
- [ ] **BNP / NT-proBNP**:< 100(BNP)/ < 300(NT-proBNP)→ 强烈反对急性 HF;>500/>1800 → 强烈支持
- [ ] **hs-cTn**(0h + 1h / 3h)— 鉴别 ACS,但 AHF 本身可轻升
- [ ] **CBC + 电解质 + 肌酐 + 肝功 + TSH + 凝血 + 乳酸**
- [ ] 动脉血气(重症)
- [ ] **床旁超声(POCUS)**:
  - **B 线 ≥ 3 / 区域 ≥ 2 区域** = 间质 / 肺泡水肿
  - LVEF + 室壁运动(节段性 → 缺血)
  - 瓣膜(急性 MR / AS / TR)
  - 心包积液
  - **IVC**(扩张 + 不变化 → 容量过负荷)
- [ ] 妊娠试验(育龄女性)
- [ ] D-dimer + CTPA(怀疑 PE)

## 一线处理(按表型)

> [!danger] 错位表型 = 错位处理 = 患者死亡

### 湿暖型(最常见,~ 70%)

> "先消肿 + 降负荷"

1. **体位 + 氧 + NIV / CPAP**(早期启动,ESC I 类)
2. **IV 利尿** — [[呋塞米]]:
   - 利尿剂初次 / naive:**20-40 mg IV**
   - 已用口服利尿:**等效或加倍 IV**
   - 持续输注 5-20 mg/h 顽固水肿
   - 监尿量 + 电解质(K, Mg)
3. **IV 扩血管(SBP > 110 时首选)**:
   - **硝酸甘油 IV 10-20 µg/min**,每 5min 加 10-20,最大 200 µg/min
   - **硝普钠 0.25-10 µg/kg/min**(警觉氰中毒长期 / 大剂量)
4. 病因治疗:房颤控率 / 复律、ACS 再灌注、感染抗菌
5. **不再常规给吗啡**(ESC 2021 弱化,OPERA-HF 等观察提示死亡上升)

### 湿冷型(心源性休克,~ 20%,死亡率 30-50%)

> [!danger] 逻辑相反!**不给硝酸甘油 / 大利尿**

1. **升压**:**[[drugs/去甲肾上腺素|去甲肾上腺素]] 0.05-0.1 µg/kg/min 起**(OptimaCC 优于多巴胺)
2. **正性肌力**:
   - **多巴酚丁胺 2-20 µg/kg/min**(β1 激动)
   - **米力农 0.125-0.75 µg/kg/min**(磷酸二酯酶抑制;**警觉低血压**;不需 β1)
   - 老年慢性 HF 患者用 β 阻滞 → 多巴酚丁胺效应减,选米力农
3. **循环辅助**:
   - **IABP**(主动脉内球囊反搏)— 部分获益,IABP-SHOCK II 提示无显著生存改善
   - **VA-ECMO**(可逆性病因 / 移植桥接)
   - **Impella**(微轴流 LVAD,本院条件)
4. **机械并发症** → 心外科 / 介入紧急(VSR、乳头肌断裂、夹层累及瓣)
5. STEMI / NSTEMI 合并 → 紧急 PCI(SHOCK trial)

### 干冷型(罕见,~ 5%)

- 容量不足 → **谨慎补液**(小量 250 mL 晶体试)
- + 正性肌力 / 升压

### 共同注意

- **避免 NSAIDs / 大量补液 / 负性肌力 CCB**
- 监测:尿量目标 ≥ 0.5 mL/kg/h、肌酐、电解质、乳酸
- 病因 + 慢性管理过渡(GDMT)
- VTE 预防(机械 + 药物)
- **不在急性期 IV 给 ACEI / ARB / ARNI**(慢性 GDMT,稳定后启动)

## 长期治疗(GDMT 四联疗法,稳定后启动)

> ESC 2021 + 2023 update / AHA 2022 强推荐

| 类 | 例 | 备注 |
|---|---|---|
| **ARNI / ACEI / ARB** | [[沙库巴曲缬沙坦]] / 依那普利 | 优于 ACEI(PARADIGM-HF) |
| **β 受体阻滞剂** | 比索洛尔 / 卡维地洛 / 美托洛尔缓释 | 急性期心衰失代偿期暂停;稳定后启动 |
| **MRA**(盐皮质激素受体拮抗剂) | 螺内酯 / 依普利酮 | EMPHASIS-HF / RALES |
| **SGLT2 抑制剂** | [[达格列净]] / 恩格列净 | DAPA-HF / EMPEROR-Reduced |

仅 HFrEF(EF ≤ 40%)受益最多;HFpEF 部分获益(SGLT2)。具体本院药品 [需查证]。

## 🔺 升级标准

> [!danger]
> - 心源性休克(SBP < 90 + 灌注差 + 乳酸 ↑)
> - NIV 30-60min 不响应 → 气管插管
> - SpO2 < 90% / pH < 7.25 / pCO2 进行性升
> - STEMI / 高危 NSTEMI 合并 → 立即导管室
> - 急性瓣膜并发症(新杂音 + AHF)→ 心外科
> - 顽固水肿 + 大剂量利尿不响应 → CRRT / 超滤
> - 任何"病人在恶化"

## ⚠️ 常见坑

> [!warning]
> 1. **湿冷型给硝酸甘油 / 大利尿** → 灾难;先想心源性休克
> 2. **没排除 ACS** → STEMI 假装"心衰"再灌注晚
> 3. **NIV 启动太晚** → 等病人喘到不能配合;早 NIV 救命
> 4. **吗啡常规给** → 不再推荐(ESC 2021 弱化)
> 5. **房颤 RVR + AHF 不积极控率 / 复律** → 节律就是诱因
> 6. **慢性 HF 漏服药突发 ADHF,只利尿不重启 GDMT** → 出院再发
> 7. **老年女性 + 急性 AHF + 突发新杂音漏听** → 急性 MR 外科急诊
> 8. **BNP 单值定 AHF** → BNP 受年龄、肾功能、肥胖影响;NT-proBNP 在房颤升
> 9. **ARDS 当 AHF 给利尿** → 容量耗竭 + 不解决炎症,病情恶化
> 10. **嗜铬细胞瘤 / 肾动脉狭窄漏诊** → 反复 / 顽固高血压性 AHF
> 11. **围产期心肌病漏诊** → 妊娠晚期 / 产后 + AHF
> 12. **化疗相关心肌病漏问** → 蒽环类 / 曲妥珠单抗 / 免疫检查点
> 13. **干冷型当湿型给利尿** → 容量进一步耗竭
> 14. **β 阻滞剂在急性失代偿期不停 / 不减** → 已用者多数继续低剂量,新发不在急性期启动
> 15. **米力农用于低血压病人** → 强血管扩张,警觉
