# 癫痫持续状态(Status Epilepticus, SE)

> [!summary] 30 秒摘要
> **红旗**:抽搐 ≥ 5 min 或两次发作之间未恢复意识
> **首查**:**ABC + 血糖 + EEG 决策**;意识不清持续 → EEG 排 NCSE
> **一线**:**苯二氮䓬一线**(劳拉西泮 IV / 地西泮 / 咪达唑仑肌注);**5-20 min 内**给足
> **升级**:不响应一线 → 二线(左乙拉西坦 / 丙戊酸 / 苯妥英);仍不响应 → ICU 镇静(丙泊酚 / 咪达唑仑持续)+ EEG 监测

---

## 一句话定义(ILAE 2015 操作性定义)

癫痫持续状态发生时间:
- **t1(异常长发作)**:**惊厥性 SE 发作 ≥ 5 min** / 局灶性意识 SE ≥ 10 min;启动急救治疗
- **t2(长期不可逆后果)**:发作 ≥ 30 min 神经元损伤风险显著(惊厥性);不同类型 t2 不同

来源:Trinka E et al. Epilepsia 2015(ILAE 操作性定义);NCS / AES Treatment of Convulsive SE 2016;中国癫痫持续状态诊治专家共识 2018(中国神经精神疾病杂志)。

---

## 分型(决定治疗策略)

| 分型 | 描述 | 备注 |
|---|---|---|
| **惊厥性 SE(GCSE)** | 全身强直阵挛 / 局灶继发全身性 | 最常见急诊,死亡率 10-20% |
| **非惊厥性 SE(NCSE)** | 持续意识改变 / 行为异常 + EEG 持续放电,**无明显抽搐** | **EEG 才能诊断**;ICU / 重症常见,易漏 |
| **微小 SE(subtle SE)** | 惊厥性 SE 后期 → 仅微小肢体抽动 + EEG 持续放电 | 非常容易漏 |
| **难治性 SE(RSE)** | 一线 + 二线足量后仍持续 ≥ 60 min | 25-40% RSE 比例 |
| **超难治性 SE(SRSE)** | 持续 ≥ 24h 即使麻醉镇静 | 死亡率 30-50% |

## 病因(决定预后,不能光治抽搐)

| 类别 | 具体 |
|---|---|
| **既往癫痫患者药物不依从 / 漏服** | 最常见(中国成人),**问药 / 查血药浓度** |
| 急性中枢病因 | 卒中、ICH、脑炎、脑膜炎、TBI、低氧脑病 |
| 代谢 | **低血糖**、低钠 / 高钠、低钙、肝衰、肾衰 |
| 中毒 / 戒断 | 酒精戒断、苯二氮䓬戒断、可卡因、TCA、有机磷、茶碱 |
| 药物 | 异烟肼(B6 缺乏)、青霉素、喹诺酮、镇痛、化疗 |
| 自身免疫性脑炎 | NMDA 受体脑炎(年轻女性、新发难治性 SE 警觉)|
| 中枢肿瘤 / 转移 | 慢性 + 影像可见 |
| 缺氧后脑损伤 | CPR 后 / 心搏骤停 |

> [!warning] **新发难治性 SE(NORSE)**+ 年轻女性 / 无既往癫痫 → 警觉自身免疫性脑炎;送抗体 + 经验性免疫治疗

## 🚩 红旗(立刻警觉)

- **任何抽搐 ≥ 5 min** / 反复 + 中间不恢复意识 → 启动 SE 流程
- 即使抽搐已停,**昏迷 / 意识不清持续不能解释** → 警觉 NCSE(EEG)
- 出现"微小"肢体抽动(嘴角 / 眼睑 / 手指)+ 长期意识不清 → 微小 SE
- 高热 + 意识改变(感染性病因)
- 头部外伤 / 创伤后 SE
- 既往癫痫 + 漏药 / 加新药 / 系统性病
- 妊娠期抽搐 → 警觉 [[子痫]](独立诊治流程)
- 老年 + 心血管病 + 抽搐 → 警觉卒中后 / 缺氧后

---

## 鉴别诊断(看似 SE 但不是)

| 情况 | 区分点 |
|---|---|
| **晕厥后惊厥**(convulsive syncope) | 短暂(几秒-数十秒)+ 自限 + 心源性病因;不是 SE |
| **心因性非癫痫发作(PNES)** | 病史诱因、发作模式不规则、闭眼、骨盆推送、EEG 正常 |
| **运动障碍**(ticks、舞蹈、肌张力障碍) | 无意识改变 + 慢性 |
| **去大脑 / 去皮质姿势** | 颅脑严重损伤 + 持续姿势,EEG 不同 |
| **抗精神病恶性综合征(NMS)** / **5-HT 综合征** | 用药史 + 高热 + 强直 + 自主神经紊乱 |
| **破伤风** | 牙关紧闭 + 角弓反张 + 接种 / 伤口史 |

## 首查

### 立即(< 5 min)

> [!warning] **不要等检查再给苯二氮䓬**

- [ ] **ABC + 头偏一侧 + SpO2**(避免误吸)
- [ ] **指尖血糖**(< 3.9 → 立即葡萄糖)
- [ ] 静脉通路 ×2 + 心电监护
- [ ] **苯二氮䓬 IV / IM / 鼻 / 颊**(下方剂量)
- [ ] CBC、电解质(Na / K / Ca / Mg)、肌酐、肝功、血氨、ABG、乳酸
- [ ] 抗癫痫药物血药浓度(若既往用药)
- [ ] 妊娠试验(育龄女性)
- [ ] 毒物筛查(酒精 / TCA / 可卡因)

### 抽搐控制后

- [ ] **EEG**(尤其意识仍不清)— 排除 NCSE / 持续放电
- [ ] **头 CT 平扫**(任何首发 SE / 局灶定位)
- [ ] **MRI**(CT 阴性 + 不能解释的持续异常)
- [ ] **腰穿**(怀疑感染 / 自身免疫性脑炎)
- [ ] 自身免疫性脑炎抗体(NMDA / LGI1 / GABA-B 等)— 不典型 / 难治性 / 年轻女性
- [ ] 影像 + EEG 后病因导向

## 一线处理(ABCDE + 4 步抗惊厥阶梯)

> [!danger] 时间是关键
> 0-5min:稳定 + 苯二氮䓬
> 5-20min:**一线足量**(若仍发作)
> 20-40min:**二线启动**(若仍发作 = SE 已确立)
> > 40min:**ICU + 麻醉镇静**(难治性 SE)

### 阶段 1:稳定(0-5 min)

- ABC + 头偏 + 吸引 + 高流量氧
- 监护 + 静脉通路 + 血糖
- 体位:侧卧防误吸
- 适时插管(GCS 持续低 / 大量分泌物 / 低氧 / 不能保护气道)

### 阶段 2:一线(苯二氮䓬,5-20 min)

> [!danger] 一线必须足量;失败 = 后续疗效大幅下降

| 药 | 成人剂量 | 儿童剂量 | 备注 |
|---|---|---|---|
| **劳拉西泮(首选 IV)** | **0.1 mg/kg IV**(最大 4 mg/次,可重复 1 次)| 0.1 mg/kg | 起效 3-5 min;持续 12-24h;**ESETT trial 等支持** |
| **地西泮 IV** | 5-10 mg IV(可重复每 5min,总 ≤ 30 mg) | 0.15-0.2 mg/kg | 起效快;持续短,易复发;**易抑制呼吸** |
| **咪达唑仑 IM**(无 IV 通路时) | **10 mg IM**(成人 ≥ 40 kg) | 0.2 mg/kg IM(最大 10) | RAMPART trial:**IM 咪达唑仑 ≥ IV 劳拉西泮**(院前) |
| 直肠地西泮 | 10-20 mg PR | 0.5 mg/kg PR | 院外 / 儿童 / 无 IV |
| 鼻 / 颊咪达唑仑 | 10 mg | 0.2 mg/kg | 替代选择 |

来源:ILAE 2015;NCS/AES 2016;ESETT NEJM 2019;RAMPART NEJM 2012;中国 SE 共识 2018。

### 阶段 3:二线(20-40 min,若一线 2 次仍失败)

> ESETT trial(NEJM 2019):**左乙拉西坦 / 丙戊酸 / 苯妥英 三药等效**

| 药 | 成人剂量 | 备注 |
|---|---|---|
| **左乙拉西坦** | **60 mg/kg IV**(最大 4500 mg)over 5-10 min | **首选**(无肝代谢、相互作用少、安全) |
| **丙戊酸** | **40 mg/kg IV**(最大 3000 mg)over 10 min | 心血管抑制少;**孕期慎用 / 肝病禁** |
| **苯妥英 / 磷苯妥英** | 苯妥英 20 mg/kg IV(< 50 mg/min);磷苯妥英 20 mg PE/kg | 历史标准,心律失常 / 低血压 / 注射部位坏死风险 |
| 苯巴比妥(替代) | 15-20 mg/kg IV | 老年 / 呼吸抑制注意 |

具体本院剂型与配制 [需查证]。

### 阶段 4:难治性 SE(> 40-60 min,ICU + 麻醉)

> ICU + 持续 EEG 监测 + 气管插管 + 机械通气

| 药 | 起始 | 滴定目标 |
|---|---|---|
| **咪达唑仑持续** | 0.2 mg/kg bolus → 0.05-2 mg/kg/h | EEG 抑制爆发 / 临床发作停 |
| **丙泊酚持续** | 1-2 mg/kg bolus → 30-200 µg/kg/min | 同上;长期警觉**丙泊酚输注综合征(PRIS)** |
| **戊巴比妥** | 5-15 mg/kg bolus → 0.5-5 mg/kg/h | 重症;深度抑制 |
| **氯胺酮**(部分中心) | 1-3 mg/kg → 1-10 mg/kg/h | 替代 / 联合;血压不下降 |

持续 24h 后逐步减量,EEG 监测控制。

### 病因导向治疗

| 病因 | 处理 |
|---|---|
| 低血糖 | 50% 葡萄糖 + 维生素 B1 |
| 低钠 | 高渗盐水谨慎(避免 ODS) |
| 异烟肼中毒 | **维生素 B6 高剂量** |
| 酒精戒断 | 苯二氮䓬 + B1 |
| 子痫 | **硫酸镁**(独立流程,见 [[子痫]]) |
| 自身免疫性脑炎(NORSE / 不典型) | 类固醇 + IVIG / 血浆置换 + 二线免疫 |
| 卒中 / ICH / 感染 / 肿瘤 | 病因特异性 |

## 🔺 升级标准

> [!danger]
> - 一线苯二氮䓬 2 次足量后仍发作 → 立即启动二线
> - 二线足量后仍发作(> 40-60 min) → ICU + 麻醉镇静 + 持续 EEG
> - 微小 / 非惊厥性 SE 持续 → ICU + EEG 监测
> - 任何不能保护气道 / 缺氧 / 心血管不稳 → 插管 + ICU
> - 妊娠 + 抽搐 → 走 [[子痫]] 通道(硫酸镁)
> - NORSE / 难治性 + 年轻女性 / 不典型 → 神免疫科 + 经验性免疫治疗
> - 任何"病人在恶化"

## ⚠️ 常见坑

> [!warning]
> 1. **苯二氮䓬给量不足** → 怕呼吸抑制就给小剂量,反致 SE 持续 + 死亡
> 2. **没及时给二线** → 一线 2 次失败就该上二线,不要"再观察"
> 3. **抽搐停了就放心** → 30% 持续意识不清是 NCSE,**EEG 才能发现**
> 4. **没查血糖 / B1 / 电解质** → 漏诊代谢病因
> 5. **酒精病人没先给 B1** → Wernicke 加重
> 6. **异烟肼中毒不用 B6** → 异烟肼诱导 SE 唯一特效
> 7. **妊娠抽搐不用硫酸镁** → 子痫漏诊后果母胎双重
> 8. **苯妥英外渗** → 紫色手套综合征 / 组织坏死;首选磷苯妥英或左乙拉西坦
> 9. **难治性 SE 不上麻醉** → 神经元持续损伤,死亡率上升
> 10. **NORSE 漏自身免疫性脑炎** → 年轻女性 / 不典型新发 → 抗体 + 免疫治疗
> 11. **PNES 误为 SE 反复加药** → 病史 + 视频 EEG;不要在 PNES 上深度麻醉
> 12. **儿童剂量计算错** → 按公斤,有上限
> 13. **CT 阴性就排除** → MRI / LP / 抗体仍可发现病因
> 14. **既往癫痫漏问血药浓度** → 漏服 / 中毒不分
> 15. **持续 EEG 不可得** → 转中心或频繁床旁 EEG
