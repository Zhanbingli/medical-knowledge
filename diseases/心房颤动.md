# 心房颤动(Atrial Fibrillation, AF)

> [!summary] 30 秒摘要
> **红旗**:RVR + 血流动力学不稳 / 急性卒中 / 急性心衰 / 新发心绞痛
> **首查**:**12 导联 ECG**(确诊)+ 电解质 + TSH + 心超 + **CHA2DS2-VASc + HAS-BLED**
> **一线**:不稳定 → 同步电复律;稳定 → 控率(β / CCB)+ **抗凝评估**(决定卒中预防)
> **升级**:不稳定 / 急性卒中 / 心衰恶化 / 极快 RVR 不能控 → 立即上级

---

## 一句话定义

心房无规律快速电活动(350-600 次/分),房室结过滤后心室率不规则。

- **最常见持续性心律失常**;终生发病率 ~ 25%(欧美)
- **核心临床代价 = 心源性卒中**(占缺血性卒中 ~ 20-30%)+ 心衰恶化 + 心动过速性心肌病

来源:ESC Guidelines for the Management of Atrial Fibrillation 2024;中国心房颤动诊治指南 2018(中华心血管病杂志);AHA / ACC / HRS AF 2023 update。

---

## 分型(决定治疗与预后)

| 分型 | 持续时间 | 备注 |
|---|---|---|
| **首次诊断** | 任何持续时间 | 仅指首发,后续按发作分类 |
| **阵发性(Paroxysmal)** | 自发终止或干预后 ≤ 7 天 | 可反复发作 |
| **持续性(Persistent)** | > 7 天 | 需药物或电复律 |
| **长期持续性(Long-standing persistent)** | > 1 年仍考虑复律 | — |
| **永久性(Permanent)** | 接受不复律 | 仅控率 |

按瓣膜病背景:
- **瓣膜性 AF**(中重度二尖瓣狭窄 / 机械瓣)→ **必须华法林,DOAC 禁用**
- **非瓣膜性 AF**(其他)→ DOAC 首选

---

## 🚩 红旗(立刻警觉)

- **不稳定 RVR**(SBP < 90、急性肺水肿、心绞痛、意识改变)→ **立即同步电复律**
- 新发卒中症状 + AF → 心源性栓塞,**走卒中通道**(不擅自抗凝,先排除出血)
- 新发心衰恶化 + AF RVR → 评估是否原发心律失常诱发
- **房颤合并预激(WPW)+ RVR** → **禁用 AVN 阻滞剂(β / CCB / 地高辛 / 腺苷)**(可加快旁路传导致 VF)→ 不稳定电复律,稳定 IV 普鲁卡因胺 / IC 类
- 房颤新发首次诊断 + 甲亢 / 急性感染 / PE / ACS / 大量饮酒 → **寻找诱因**

## 鉴别诊断(其他不规则窄 QRS 心律)

| 心律失常 | 区分点 |
|---|---|
| [[心房扑动]] | 锯齿样 F 波、心室率多为 150 整数倍(2:1 = 150;3:1 = 100;4:1 = 75) |
| [[多源性房性心动过速]](MAT) | ≥ 3 种 P 波形态、PR 间期变化,常见 COPD / 严重肺病 |
| [[频发房性早搏]] | 提前的 P 波 + 不完全代偿间歇 |
| [[阵发性室上速 SVT]] | 规律窄 QRS,突发突止;AVNRT / AVRT |
| [[宽 QRS 心动过速]](VT vs SVT 伴差异) | 别误把房颤伴差异传导当 VT;但**宽 QRS 默认按 VT 处理** |

## 一线处理

### 0. 决策路径(ABC pathway,ESC 2024 强调)

> [!tip] **A**nticoagulation / **B**etter symptom control / **C**omorbidities / Cardiovascular risk factor management

### 1. 不稳定 → 同步电复律

| 情境 | 处置 |
|---|---|
| SBP < 90 / 心源性休克 | **DC 同步电复律 100-200 J 双相**起步,逐步加 |
| 急性肺水肿 | DC 同步电复律 |
| 急性心绞痛 / ACS | DC 同步电复律 |
| 神志改变 / 心搏骤停前征 | DC 同步电复律 |

### 2. 稳定 → 控率优先

| 选择 | 用法 | 备注 |
|---|---|---|
| **β 受体阻滞剂**(美托洛尔) | 5 mg IV q5min × 3 或口服 | 首选;EF 保留 / 降低均可 |
| **非二氢吡啶 CCB**(地尔硫卓 / 维拉帕米) | 0.25 mg/kg IV bolus,后 5-15 mg/h | 仅 EF 保留;**EF < 40% 禁用** |
| 地高辛 | 0.25 mg IV q2h(总 1 mg)× 2-3 | 心衰中可加;起效慢;**老年 / 肾差减量,警觉中毒** |
| 胺碘酮 | 150 mg IV over 10min → 1 mg/min | 重症 / 难治;**有复律作用,需评估抗凝** |

**目标心率**:
- 宽松控制:静息 < 110 bpm(RACE-II 显示等效)
- 严格控制:< 80 bpm(症状仍未缓解时考虑)

### 3. 复律决策(节律控制 vs 频率控制)

> [!warning] 抗凝时机决定复律安全
> 房颤 ≥ 48h 或时间不明 → **3 周抗凝后再复律**(或 TEE 排除左房血栓后复律)+ **复律后至少抗凝 4 周**(无论 CHA2DS2-VASc)
> < 48h 且无危险因素 → 可直接复律 + 复律后抗凝 4 周

| 选项 | 工具 |
|---|---|
| **同步电复律** | 100-200 J 双相,镇静下 |
| **药物复律** | **氟卡尼**(无器质性心脏病)/ 普罗帕酮 / 伊布利特 / 胺碘酮 |
| **导管消融** | 阵发性药物难治 / 心衰(EAST-AFNET 4 trial:早期节律控制改善预后) |
| **左心耳封堵** | 抗凝禁忌长期不能用者(WATCHMAN) |

### 4. 抗凝(决定长期预后,关键!)

> [!danger] **CHA2DS2-VASc 决定抗凝**
> - **男 ≥ 2、女 ≥ 3** → **建议长期口服抗凝**
> - **男 1、女 2** → 个体化(考虑年龄、卒中风险、出血风险、患者偏好)
> - 男 0、女 1 → **不抗凝**

| 抗凝选择 | 适应证 |
|---|---|
| **DOAC**(利伐沙班 / 阿哌沙班 / 达比加群 / 艾多沙班)| **首选**(非瓣膜性,出血少 + 不需监测 INR) |
| **华法林** | 中重度二尖瓣狭窄、机械瓣、CrCl < 15、孕妇(实际孕妇用 LMWH) |
| **LMWH** | 围术期桥接、孕妇 |

> [!warning] **阿司匹林单药不应作为 AF 抗凝替代**(出血风险接近抗凝但保护差);ESC 2020-2024 持续不推荐。

具体剂量按肾功能调整 [需查证: 本院 DOAC 剂量表 + 中华肾内合并 AF 抗凝共识]。

### 5. 急性卒中合并 AF

- 走 **卒中通道**;**先排除出血**(头 CT)
- 缺血卒中 + AF + 时间窗内 → **溶栓 / 取栓**(参考 [[急性缺血性脑卒中]])
- 抗凝**何时启动**:
  - TIA / 小卒中:2-3 天后
  - 中等卒中:5-7 天后
  - 大卒中:10-14 天后(出血转化风险)

### 6. 节律控制 vs 频率控制(选择)

- **EAST-AFNET 4 (NEJM 2020)**:早期节律控制(< 1 年)改善心血管结局
- **AFFIRM / RACE**:慢性 AF 节律 vs 频率,**总体生存等效**
- 临床判断:症状重 / 心衰 / 年轻 / 早期 → 倾向节律控制;高龄 / 无症状 / 长期 AF → 倾向频率控制

## 🔺 升级标准

> [!danger]
> - 不稳定 RVR → 立即电复律 + 上级到场
> - 新发卒中症状 → 卒中通道
> - 房颤 + 急性心衰恶化
> - 房颤 + 怀疑 ACS / PE
> - **房颤合并预激(WPW)** → 禁用 AVN 阻滞剂,谨慎处理
> - RVR 持续不能控制 > 6h
> - 任何"病人在恶化"

## ⚠️ 常见坑

> [!warning]
> 1. **把房扑当房颤** → 房扑心室率多规则(2:1, 3:1, 4:1 整数倍 150),处理略不同
> 2. **EF < 40% 用维拉帕米 / 地尔硫卓** → 加重心衰
> 3. **新发房颤 > 48h 直接药物复律不评估血栓** → 需 TEE 或先抗凝 3 周
> 4. **CHA2DS2-VASc 0/1 也常规抗凝** → 出血风险大于获益
> 5. **认为"控律比控率更好"** → AFFIRM/RACE 总体等效;以症状为导向
> 6. **抗凝忽视肾功能** → DOAC 多需按 CrCl 调整(达比加群 / 阿哌沙班 / 利伐沙班 / 艾多沙班各异)
> 7. **甲亢未筛查** → 治疗甲亢可恢复窦律
> 8. **预激合并房颤误用 β / CCB / 地高辛 / 腺苷** → 加快旁路传导致 VF
> 9. **阿司匹林替代抗凝** → 出血同等保护差,过时观念
> 10. **房颤合并急性卒中早期就给抗凝** → 出血转化;按面积权衡 2-14 天
> 11. **DOAC 患者溶栓** → 末次给药 < 48h 多视为禁忌
> 12. **复律后忘了抗凝** → 复律后 4 周内卒中风险显著
> 13. **导管消融期望过高 / 不评估复发** → 术后 3 月内空白期常复发,继续抗凝
> 14. **运动员 / 年轻 AF 漏问家族 / 结构性 / 长 QT 综合征**
> 15. **AF "假日心脏综合征"**(大量饮酒诱发)→ 第二天宿醉胸闷可能就是

## 关键评分

- **[[CHA2DS2-VASc]]**(0-9):卒中风险,决定抗凝
  - C:CHF / EF ↓(1)
  - H:高血压(1)
  - A:年龄 ≥ 75(2);年龄 65-74(1)
  - D:糖尿病(1)
  - S:既往卒中 / TIA / 血栓栓塞(2)
  - V:血管病(MI / PAD / 主动脉斑块)(1)
  - Sc:女性(1)
- **[[HAS-BLED]]**(0-9):出血风险
- **[[EHRA]]**:症状分级 I-IV
- **[[ABC pathway]]**:ESC 2024 整体管理框架
