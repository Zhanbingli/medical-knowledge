# 高血压急症(Hypertensive Emergency)

> [!summary] 30 秒摘要
> **红旗**:**SBP > 180 / DBP > 120 + 急性靶器官损害**(脑 / 心 / 肾 / 眼 / 妊娠)
> **首查**:ECG + 肌钙 + BNP + 肌酐 + 尿常规 + 头 CT(若神经定位)+ CXR + 眼底 + HCG
> **一线**:**IV 滴注**药物滴定;**1h 内降 SBP 25%**(夹层 / 子痫例外)+ 病因导向
> **升级**:任何怀疑 → ICU + 持续动脉测压(夹层 / 子痫 / 急性 LV 衰竭等独立流程)

---

## 一句话定义

**SBP > 180 mmHg 和 / 或 DBP > 120 mmHg + 急性靶器官损害(AOI)** = 高血压急症。

vs **高血压亚急症(Hypertensive Urgency)**:同样 BP 但**无急性靶器官损害** → 数小时-数天口服调整,**不静脉降压**。

来源:AHA / ACC Hypertension Guideline 2017;ESC / ESH Hypertension 2018 + 2024 update;中国高血压防治指南 2024(中华医学会心血管病学分会 / 高血压联盟)。

## 病因(找诱因 = 治疗一半)

| 类别 | 具体 |
|---|---|
| **原发性高血压控制不良** | 漏服 / 自停药 / 高盐 / 应激 |
| **二次性高血压** | 肾动脉狭窄、原发性醛固酮增多症、库欣综合征、嗜铬细胞瘤、阻塞性睡眠呼吸暂停 |
| **药物 / 中毒** | 可卡因、安非他明、MAOI + 食物相互作用、避孕药、皮质类固醇、NSAID、麦角胺、戒断(可乐定 / β 阻滞剂) |
| **妊娠** | 子痫前期 / 子痫 |
| **肾脏** | 急 / 慢性肾脏病 |
| **神经** | ICH / 卒中 / 颅内压增高 |
| **疼痛 / 焦虑** | 排除性 |

## 🚩 红旗(立刻警觉)

- 极高血压(SBP > 180 / DBP > 120)+ **任一新发症状**:
  - 神经定位 / 意识改变 / 抽搐 → ICH / 卒中 / 高血压脑病
  - 雷击样头痛 → SAH
  - 撕裂胸背痛 + 双臂血压差 → [[emergencies/主动脉夹层|主动脉夹层]]
  - 端坐 + 肺水肿 → [[emergencies/急性肺水肿|急性肺水肿]]
  - 胸痛 + ECG 变化 → ACS
  - 妊娠 + BP 升 + 头痛 / 视物模糊 → 子痫前期 / 子痫
  - 阵发性高血压 + 心悸 + 出汗 + 头痛 → 嗜铬细胞瘤
- III/IV 级视网膜病变(出血、渗出、视乳头水肿)
- 进行性 AKI / 血尿 / 蛋白尿

## 鉴别(高血压急症 vs 亚急症 vs 非紧迫)

| 情境 | 处理 |
|---|---|
| **高血压急症**(BP 极高 + AOI) | **IV 滴定** ICU 监测 |
| **高血压亚急症**(BP 极高 + 无 AOI) | **口服调整**,门诊 / 病房 24-72h 内目标降至基线 |
| 焦虑 / 疼痛 / 急性应激 → BP 升 | **先治疼痛 / 焦虑**;不当降压有害 |
| 白大衣高血压 | 24h 动态 BP 监测 |
| 测量错误 | 复测多次 + 双臂 + 大袖带 |

> [!warning] **不要因高 BP 数字本身慌张降压** — 关键是有无 AOI 和病因。

## 一线处理

### 通则:降压目标(场景特异性)

> [!danger] 目标错 = 患者死亡
> 不同 AOI 目标完全不同。**先确定病因,再选目标和药物。**

| 情境 | 目标 |
|---|---|
| **常规高血压急症**(无特定 AOI) | **1h 内降 MAP ≤ 25%** → 2-6h 至 SBP ~ 160 / DBP ~ 100-110 → 24-48h 渐至正常 |
| **[[emergencies/主动脉夹层|主动脉夹层]]** | **先 HR < 60 用 β 阻滞剂**,再 SBP < 120(20 min 内) |
| **[[diseases/脑出血|脑出血]]** | **SBP 140**(平稳,24h 内,INTERACT-2 / ATACH-2) |
| **[[emergencies/蛛网膜下腔出血|SAH]]**(未处理) | SBP < 140-160(AHA 2023) |
| **[[emergencies/急性缺血性脑卒中|缺血性卒中]]** 不溶栓 | 许可性高血压(SBP < 220 / DBP < 120 通常不降) |
| 缺血性卒中 + 溶栓前 / 中 | **SBP < 185 / DBP < 110** |
| 溶栓后 24h | **SBP < 180 / DBP < 105** |
| **ACS / 急性 LV 衰竭** | 减负荷,目标 MAP ↓ + 灌注维持 |
| **嗜铬细胞瘤危象** | **先 α 阻滞**(酚妥拉明)再 β,顺序错诱 BP 反弹 |
| **子痫 / 子痫前期** | SBP < 160 / DBP < 110;**[[硫酸镁]]** 抗惊厥 |
| **可卡因 / 安非他明性** | **不用单 β 阻滞剂**(α 占主诱发血管收缩);**苯二氮䓬 + α/β 联合(拉贝洛尔)** |

来源:AHA 2017 / ESC 2018(2024 update);中国高血压防治指南 2024。

### 主要 IV 药物

| 药 | 剂量 | 适应证 / 备注 |
|---|---|---|
| **[[拉贝洛尔]]**(α + β) | 10-20 mg IV q10min,最大 300 mg;持续 1-2 mg/min | **首选**(尤其妊娠 / 卒中 / 夹层后期);**哮喘 / 严重心衰禁用** |
| **[[尼卡地平]]**(二氢吡啶 CCB) | 5 mg/h IV,每 5min 加 2.5 mg/h,最大 15 mg/h | 通用;脑卒中 / 妊娠 / SAH 偏好 |
| **[[艾司洛尔]]**(超短效 β1) | 500 µg/kg 负荷 → 50-300 µg/kg/min | **夹层首选**(快速起效短效) |
| **[[硝普钠]]** | 0.25-10 µg/kg/min | 强效双向扩血管;**警觉氰中毒长期 / 大剂量**;**升 ICP 不用于神经急症** |
| **[[氯维地平]]** | 1-2 mg/h → 最大 32 mg/h | 二氢吡啶超短效;心源性肺水肿优 |
| **[[肼屈嗪]]** | 5-10 mg IV q20-30min(妊娠) | 妊娠常用,但反射性心动过速 |
| **[[酚妥拉明]]**(α 阻滞) | 5-15 mg IV q5-15min | **嗜铬细胞瘤、可卡因相关**;不单用于其他 |
| **[[呋塞米]]** | 20-40 mg IV(若合并急性 HF) | 容量过负荷时 |
| **[[硫酸镁]]** | 4-6 g IV bolus + 1-2 g/h | **子痫 / 子痫前期** 抗惊厥 + 降压辅助 |

具体本院药品规格 [需查证]。

### 经常需避免的组合

> [!warning]
> - **舌下硝苯地平 / 速效硝苯地平** — 已被多指南反对(突然下降 / 反射性心动过速 / 卒中风险);**禁用**
> - 单用 β 阻滞剂治可卡因性高血压
> - 夹层未先 β 阻滞前用扩血管药(反射性心率快加重)
> - 神经急症用硝普钠(升 ICP)
> - 妊娠期 ACEI / ARB / 硝普钠 / 利尿过度

### 特殊情境快查表

- **嗜铬细胞瘤**:酚妥拉明 → 后加 β
- **可卡因**:苯二氮䓬 + 拉贝洛尔(尽管"非二氢吡啶 + α/β"是历史争议;但避免单 β)
- **子痫前期 / 子痫**:硫酸镁 + 拉贝洛尔 / 肼屈嗪 / 尼卡地平
- **戒断综合征**(可乐定 / β 阻滞剂)→ 重启原药 + 必要时拉贝洛尔
- **神经急症**:尼卡地平 / 拉贝洛尔(避免硝普钠)

## 🔺 升级标准

> [!danger]
> - 任何急性 AOI → ICU + 动脉置管 + IV 滴定
> - **夹层** → 心外科 / 血管外科紧急 + 先 β 后扩血管
> - 神经定位 → 卒中 / SAH 通道
> - 急性 LV 衰竭 / 急性 MR → 心内 / 心外
> - 子痫 → 产科 + 硫酸镁
> - 嗜铬细胞瘤危象 → 内分泌 + 顺序 α 后 β
> - 任何"病人在恶化"

## ⚠️ 常见坑

> [!warning]
> 1. **数字降下来 = 治好** → 没找到病因,反复发作
> 2. **不区分急症 vs 亚急症** → 把亚急症当急症 IV 降压,过度降压致缺血
> 3. **降压速度太快** → MAP 1h 内下降 > 25% → 缺血卒中 / 心 / 肾损害
> 4. **舌下硝苯地平** → **禁用,临床仍偶见**
> 5. **夹层未先 β 阻滞** → 反射性心率快 → 夹层进展
> 6. **嗜铬细胞瘤先 β 后 α** → α 不被对抗 → 高血压危象 + 心衰
> 7. **可卡因高血压单用 β** → α 占主血管收缩加重
> 8. **缺血性卒中急性期过度降压** → 半暗带恶化(许可性高血压)
> 9. **妊娠高血压用 ACEI / ARB / 硝普钠** → 胎儿 / 母亲风险
> 10. **没问毒物 / 戒断 / 二次性病因** → 反复发作
> 11. **没做眼底** → 漏诊视网膜 III/IV 级病变 = AOI
> 12. **没查尿微量蛋白** → 漏诊子痫前期早期 + 肾受损
> 13. **MAOI + 富酪胺食物 / 拟交感药相互作用漏问** → 高血压危象
> 14. **降压药戒断**(可乐定 / β 阻滞剂)漏问 → 重启原药即可
> 15. **未识别 RVR 房颤诱发 AHF + 高血压** → 多机制叠加
