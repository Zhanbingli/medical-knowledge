# 急性呼吸衰竭(Acute Respiratory Failure, ARF)

> [!summary] 30 秒摘要
> **红旗**:气道不能保护 / SpO2 下降或需高浓度氧 / 呼吸肌疲劳 / 意识改变 / 休克
> **首查**:ABCDE + SpO2 + 血气 + ECG + CXR/床旁超声 + 找病因
> **一线**:给氧并设目标范围、处理病因、评估 HFNC/NIV/插管；高碳酸风险者避免盲目高氧
> **升级**:低氧或高碳酸进展、意识下降、休克、NIV/HFNC 失败或需气道保护 → 立即叫上级/ICU

## 一句话定义

[[diseases/急性呼吸衰竭|急性呼吸衰竭]] 是呼吸系统不能维持足够氧合和/或二氧化碳排出，导致低氧血症、急性高碳酸血症或两者并存的危急综合征；其本质不是一个单一诊断，而是 [[肺炎]]、[[COPD急性加重]]、[[急性心力衰竭]]、[[emergencies/肺栓塞|肺栓塞]]、[[ARDS]]、[[气胸]]、神经肌肉病等病因的共同终末表现。

## 分型 / 分类

| 分类 | 核心问题 | 常见病因 | 临床提示 |
|---|---|---|---|
| I 型:低氧性呼吸衰竭 | 氧合失败 | [[肺炎]]、[[ARDS]]、[[肺水肿]]、[[emergencies/肺栓塞|肺栓塞]]、[[气胸]] | 低氧明显,CO2 可正常或降低 |
| II 型:高碳酸血症性呼吸衰竭 | 通气失败 | [[COPD急性加重]]、哮喘重症、药物/中毒、神经肌肉病、胸廓疾病 | 嗜睡、头痛、扑翼样震颤、酸中毒 |
| 混合型 | 氧合 + 通气均失败 | 重症肺炎、ARDS 合并疲劳、COPD 合并感染/心衰 | 低氧 + CO2 潴留 |
| 气道保护失败 | 误吸/窒息风险 | 昏迷、癫痫后、脑卒中、创伤、中毒 | GCS 下降、呕吐、分泌物多 |
| 循环-呼吸耦合失败 | 低灌注加重缺氧 | [[diseases/休克|休克]]、[[emergencies/脓毒症|脓毒症]]、[[心源性休克]] | 低血压、乳酸升高、皮肤湿冷 |

## 🚩 红旗(立刻警觉)

- 不能完整说话、明显三凹征、辅助呼吸肌参与、呼吸浅慢或濒死样呼吸。
- 意识改变、躁动转嗜睡、不能保护气道、呕吐或大量分泌物。
- SpO2 持续低或需高浓度氧才能维持，尤其伴发绀、低血压或乳酸升高。
- 急性高碳酸血症伴酸中毒、嗜睡或扑翼样震颤。
- 沉默胸、极度呼吸肌疲劳、胸廓运动差，尤其见于 [[哮喘急性发作]] 或 [[COPD急性加重]]。
- 单侧胸痛 + 单侧呼吸音减弱 + 低血压，警惕 [[emergencies/张力性气胸|张力性气胸]]。
- 气短 + 胸痛/咯血/晕厥/低氧，警惕 [[emergencies/肺栓塞|肺栓塞]]。
- 发热、低血压、少尿、意识改变，警惕 [[emergencies/脓毒症|脓毒症]] 合并呼吸衰竭。

## 首查(到床后 [需查证: 本院要求时限] 分钟内)

- [ ] ABCDE:先判断气道是否通畅、是否能保护气道、是否需要立即叫麻醉/ICU。
- [ ] 生命体征 + SpO2 + 意识状态；使用 [[NEWS2]] 或本院早期预警评分。
- [ ] 动脉/静脉血气:判断低氧、高碳酸、酸中毒、乳酸；高碳酸风险者需复查趋势。
- [ ] 12 导联 [[ECG]]:排除缺血、心律失常、高钾、右心负荷线索。
- [ ] CXR 或床旁肺超声:肺炎、肺水肿、气胸、胸腔积液、ARDS 线索。
- [ ] CBC、CRP/PCT、肾功能、电解质、肝功能、凝血、血糖；按病情加肌钙蛋白、BNP/NT-proBNP、D-dimer。
- [ ] 病原学:疑似感染时按本院流程采集血培养、痰培养、呼吸道病毒/细菌检测，不能因等待检查延误抢救。
- [ ] 床旁超声:肺 B 线、胸水、气胸、右心负荷、容量状态、心包填塞线索。
- [ ] 复核用药和暴露:阿片/镇静药、中毒、支气管扩张剂不足、过敏原、误吸、近期手术/制动。

## 一线处理

### 1. 先稳定 ABCDE

- 坐起或半卧位；清除分泌物、吸引、准备气道设备。
- 氧疗是治疗低氧血症的药物，应设定目标 SpO2 范围并监测；BTS 建议氧疗按目标饱和度处方，而不是把氧气当作单纯缓解气短的措施 (BTS Oxygen Guideline, 2017)。
- 对有高碳酸风险者，如 [[COPD急性加重]]、肥胖低通气、神经肌肉病，避免未监测血气的盲目高浓度氧；目标范围需按本院规范和血气结果设定。

### 2. 同步处理病因

| 病因 | 不能延误的处理 |
|---|---|
| [[哮喘急性发作]] / [[COPD急性加重]] | 雾化支气管扩张剂、糖皮质激素、血气复查；酸中毒/疲劳时评估 NIV 或插管 |
| [[肺炎]] / [[emergencies/脓毒症|脓毒症]] | 采培养后尽早抗感染、液体/血管活性药按休克评估、感染源控制 |
| [[急性心力衰竭]] / [[emergencies/急性肺水肿|急性肺水肿]] | 评估血压和容量，利尿/血管扩张/无创通气按病情与禁忌证 |
| [[emergencies/肺栓塞|肺栓塞]] | 风险分层、抗凝或再灌注策略需上级决策 |
| [[emergencies/张力性气胸|张力性气胸]] | 临床高度怀疑且不稳定时立即减压，不等影像 |
| 中毒/药物抑制呼吸 | 停药、解毒剂按适应证、气道保护和通气支持 |

### 3. 选择呼吸支持

- 常规氧疗:适用于轻中度低氧且呼吸功不重者，需反复评估。
- HFNC:脓毒症相关急性低氧性呼吸衰竭中，SSC 建议 HFNC 优于 NIV (Surviving Sepsis Campaign, 2021)。
- NIV:ERS/ATS 2017 对 COPD 急性加重合并急性呼吸性酸中毒、心源性肺水肿等场景支持 NIV；但该指南超过 5 年，需结合本院 ICU/呼吸科规范。
- 有创机械通气:不能保护气道、呼吸肌疲劳、意识下降、休克、严重低氧/酸中毒进展或 NIV/HFNC 失败时，应尽早叫 ICU/麻醉评估。
- ARDS:ESICM 2023 和 SSC 2021 支持肺保护通气策略；SSC 2021 对脓毒症相关 ARDS 推荐低潮气量 6 mL/kg 预测体重和平台压上限 30 cmH2O (Surviving Sepsis Campaign, 2021)。

> [!warning] ESC/ATS/NIV 与 ARDS 指南适用人群不同
> [[diseases/急性呼吸衰竭|急性呼吸衰竭]] 的支持方式取决于病因、氧合/通气障碍类型、意识、血流动力学和本院资源；不要把“能上 NIV”当成不叫 ICU 的理由。

## 🔺 升级标准(立刻叫上级)

- 不能保护气道、意识下降、误吸风险或分泌物无法清除。
- SpO2 低或需不断提高氧浓度，且病因未被快速控制。
- 高碳酸血症进展、酸中毒加重、嗜睡或呼吸浅慢。
- 呼吸肌疲劳:三凹征、辅助呼吸肌、出汗、不能说完整句、呼吸节律异常。
- 血流动力学不稳、休克、严重心律失常或胸痛。
- [[emergencies/张力性气胸|张力性气胸]]、大咯血、上气道梗阻、过敏反应等需要立即操作的病因。
- HFNC/NIV 后仍恶化或无法耐受，需评估插管和 ICU。
- 儿童、妊娠、神经肌肉病、肥胖低通气、免疫抑制或多器官衰竭患者。

## ⚠️ 常见坑

- 只看 SpO2，不看呼吸功、意识和血气，漏掉高碳酸或疲劳。
- 高碳酸风险患者盲目高流量给氧，不复查血气。
- 把 NIV 当成“安全观察”，没有设定失败标准和复评时间。
- 张力性气胸等不稳定病因等待影像，延误减压。
- 肺水肿患者盲目大量补液，导致低氧恶化。
- 低氧改善后忘记找病因，漏掉 PE、ACS、脓毒症或误吸。
- 没有记录氧疗设备、流量/FiO2、SpO2、血气和复评时间。
- 急性呼吸衰竭患者平卧转运或检查，导致误吸或低氧加重。

## 关键评分

- **[[NEWS2]]**:急性病情恶化预警，结合 SpO2、氧疗、呼吸频率、血压、心率、意识等。
- **[[GCS]]**:判断意识和气道保护能力。
- **[[scores/qSOFA|qSOFA]] / [[SOFA]]**:感染/脓毒症相关器官功能评估。
- **[[ROX指数]]**:HFNC 过程中评估低氧性呼吸衰竭进展风险；具体阈值需查证专题。
- **[[scores/Wells-PE|Wells-PE]]**:疑似 [[emergencies/肺栓塞|肺栓塞]] 的临床概率评估。
