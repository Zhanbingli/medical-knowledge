# 急性阑尾炎(Acute Appendicitis)

> [!summary] 30 秒摘要
> **红旗**:转移性右下腹痛 + 发热 + 反跳痛 + 白细胞 ↑ + CRP ↑
> **首查**:Alvarado / AIR 评分 + 妊娠试验(育龄女性)+ **首选 CT**(儿童 / 妊娠优先 US)
> **一线**:**手术**(腹腔镜首选);非复杂性可考虑抗生素治疗(选择性,需谨慎)
> **升级**:穿孔 / 弥漫性腹膜炎 / 阑尾脓肿 / 妊娠期 → 立即上级 + 外科

---

## 一句话定义

阑尾管腔阻塞(粪石、淋巴增生、肿瘤)→ 腔内压升 → 静脉回流受阻 → 缺血 → 细菌增殖 → 炎症进展 → 坏疽 / 穿孔。

**最常见外科急腹症**;终生发病率 7-8%(欧美数据,中国略低)。

来源:WSES Jerusalem Guidelines for Acute Appendicitis 2020;中国成人急性阑尾炎诊治指南 2014(中华外科杂志 / 中华消化外科杂志);UpToDate Acute Appendicitis 2024。

---

## 病理分型(决定治疗激进度)

| 类型 | 特征 |
|---|---|
| **单纯性** | 阑尾炎症,管壁完整 |
| **化脓性** | 管壁 / 周围渗出 |
| **坏疽性** | 阑尾壁坏死 |
| **穿孔性** | 局限性脓肿 / 弥漫性腹膜炎 |
| **特殊位置** | 后位(retrocecal)、盆位、肝下位、左位(左旋异位)→ 体征不典型 |

---

## 🚩 红旗(立刻警觉)

- 经典**转移性疼痛**:脐周 / 上腹模糊隐痛 → **6-24h 后转移至右下腹**(McBurney 点)
- 持续锐痛 + 反跳痛 + 板状腹 → **穿孔 / 腹膜炎**
- 高热(> 39℃)+ 心动过速 + 全身中毒征
- **"发病 24-48h 内疼痛突然缓解"**=穿孔减压(假愈期)→ 几小时后弥漫性腹膜炎
- 老年 + 阑尾炎 → 穿孔率高(查体不典型)
- 妊娠 + 右上 / 右中腹痛(妊娠期阑尾位置上移)
- 儿童 < 5 岁 + 不典型 → 穿孔率 > 50%

## 首查

### 必查体征
- 生命体征 + 一眼判断
- **McBurney 点压痛 + 反跳痛**
- **Rovsing 征**(左下腹按压触发右下腹痛)
- **Psoas 征**(屈膝伸髋诱发疼痛 — 后位 / 盆位阑尾,腰大肌刺激)
- **Obturator 征**(屈髋内旋诱发 — 盆位阑尾)
- **Markle 征 / 跳跃试验**(踮脚跳落诱痛 — 腹膜刺激敏感)
- 直肠指检(盆位阑尾压痛)
- 妇科双合诊(育龄女性)
- 肺底听诊(下叶肺炎可拟阑尾炎)

### 实验室
- [ ] **CBC + 分类**(白细胞 ≥ 10 + 中性 ≥ 75% 支持)
- [ ] **CRP**(逐步升高更支持)
- [ ] **HCG**(育龄女性强制)
- [ ] 尿常规(排泌尿感染 + 邻近膀胱反应可见少量白细胞)
- [ ] 肝肾功 + 电解质 + 血型(术前)

### 影像

| 人群 | 首选 |
|---|---|
| **成人**(非妊娠) | **腹盆腔 CT 增强** —— 敏感性 / 特异性 ≥ 95%(WSES 2020 推荐)|
| **妊娠** | **超声**首选;阴性 + 持续怀疑 → **MRI**(避免 CT 辐射) |
| **儿童** | **超声**首选(无辐射);阴性 / 阳性不明确 → MRI / 选择性 CT |

CT 阳性征象:阑尾直径 > 6 mm、壁增厚、周围渗出、脂肪条索、阑尾结石。

> 来源:WSES 2020;中国成人急性阑尾炎诊治指南 2014;ACR 阑尾炎影像适宜性标准 2018。

## 一线处理

### 1. 围术期共同处理

- 禁食水
- 静脉补液(乳酸林格)
- 镇痛(吗啡 / 哌替啶 — **现代证据支持术前镇痛不掩盖诊断**)
- 止吐(昂丹司琼 / 甲氧氯普胺)
- VTE 预防

### 2. 术前抗生素

> [!warning] 术前 1 小时内单次给药,降低伤口感染

| 情境 | 经验首选 |
|---|---|
| **非穿孔(阑尾切除围术期)** | 头孢曲松 + 甲硝唑(单次)/ 哌拉西林他唑巴坦(单次) |
| **穿孔 / 腹膜炎 / 脓肿** | 哌拉西林他唑巴坦 / 厄他培南 / 头孢 + 甲硝唑;延长 4-7 天 |
| **过敏 β-内酰胺** | 氟喹诺酮 + 甲硝唑 / 克林霉素 + 庆大霉素 |

剂量按体重 + 肾功能调整 [需查证: 本院抗菌药物分级管理]。

来源:WSES 2020;IDSA Complicated Intra-abdominal Infections 2010(2017 update);中国急性腹腔感染诊治指南 <!-- 待具体年份 -->。

### 3. 手术(标准治疗)

| 方式 | 适应证 |
|---|---|
| **腹腔镜阑尾切除**(LA) | **首选**(成人 / 妊娠 / 儿童均可)— WSES 2020 强推 |
| 开腹阑尾切除 | LA 禁忌(严重粘连 / 设备不可) |
| 介入引流(经皮 / 超声引导) | 阑尾脓肿(直径 > 4 cm + 外科条件不佳)→ 8-12 周择期 LA |
| 选择性单纯抗生素治疗(NOTA / NOA) | **非复杂性**阑尾炎,患者拒绝 / 不适手术,**复发率 ~ 30% 5 年内**(APPAC, CODA trials)→ **不建议作为常规** |

### 4. 术后

- 早期下床、肠鸣音恢复后流食
- 单纯性 LA 术后 24-48h 出院
- 穿孔 / 脓肿 / 腹膜炎延长抗生素 + 监测发热 / 切口
- 阑尾标本病理(警觉阑尾肿瘤,> 40 岁约 1-2% 发现)

## 🔺 升级标准

> [!danger]
> - 弥漫性腹膜炎 / 板状腹 → 立即外科会诊 + 备血 + 准备手术
> - 阑尾脓肿(CT 局限性积液 + 包块)→ 介入引流可能 + 外科会诊
> - 高热 + 持续心动过速 + 乳酸 ↑ → 警觉脓毒症 + 1h bundle
> - 妊娠期阑尾炎 → **产科 + 普外联合**(阑尾位置上移 + 母胎风险)
> - 儿童 < 5 岁 → 穿孔率高 + 临床表现不典型,低阈值影像 + 早期外科
> - 老年 → 穿孔 / 肿瘤伪装 / 合并症,低阈值留观 + 影像
> - 免疫抑制 / 化疗后 → 可表现不典型 + 真菌感染叠加

## ⚠️ 常见坑

> [!warning]
> 1. **典型转移痛 ≠ 必有,无典型转移痛 ≠ 不是阑尾炎** — 仅 50-60% 病例有经典进展
> 2. **后位(retrocecal)阑尾**腹部体征轻 → **Psoas 征 + 腰部触诊**才能发现
> 3. **盆位阑尾**腹部体征轻 → **直肠指检 / Obturator 征**
> 4. **早期 24h 内 CRP 正常**,不能排除;白细胞 / CRP 阴性 + 高度怀疑 = 影像
> 5. **老年穿孔率高**,但发热 / WBC / 体征常不典型 → 低阈值 CT
> 6. **妊娠期阑尾位置上移**(右上甚至右肋下) → 漏诊后果严重(母胎双风险)
> 7. **儿童 < 5 岁穿孔率 > 50%** → 别"先观察",影像 / 早期外科评估
> 8. **DKA 表现腹痛** → 测血糖 + 酮 + ABG;别开错刀
> 9. **阑尾炎"突然不痛了"** → 警觉穿孔后假愈期,几小时内弥漫性腹膜炎
> 10. **老年 / 慢性病程 / > 40 岁 + 阑尾肿瘤** → 病理必送;CT 包块 / 增厚警觉
> 11. **左旋异位 / 内脏反位** 罕见但存在,左下腹"阑尾炎"不要笑
> 12. **儿童肠套叠假装阑尾炎** → 阵发哭闹 + 果酱便 = 肠套叠优先
> 13. **阴性阑尾切除率(NAR)~ 5-15%** 是合理代价,但 CT 普及后应控制 < 5%
> 14. **抗生素仅治非复杂阑尾炎** 复发率约 30%(5 年),不应作为常规替代手术
> 15. **术后腹腔脓肿** 穿孔病例 5-10% → 影像随访 + 必要引流
