# 休克(Shock)

> [!summary] 30 秒摘要
> **红旗**:低血压或相对低血压 + 意识改变 / 少尿 / 皮肤湿冷或花斑 / 乳酸升高
> **首查**:ABCDE + 监护 + 乳酸/血气 + ECG + 床旁超声 + 找出血/感染/心脏/梗阻
> **一线**:边复苏边找病因；低容量先止血/补容量，感染早抗菌，心源性慎补液，过敏性先肾上腺素
> **升级**:任何疑似休克都应立即叫上级；需血管活性药、输血、气道或源控制时同步叫 ICU/相关专科

## 一句话定义

休克是急性循环衰竭导致组织氧供不足和细胞代谢障碍的危急综合征；血压正常不能排除早期休克，判断核心是组织低灌注。

## 🚩 红旗(立刻警觉)

- 低血压或较平时血压明显下降，伴意识改变、少尿、皮肤湿冷/花斑、乳酸升高。
- 晕厥、胸痛、气短、发绀、SpO2 下降或呼吸功增加。
- 活动性出血、呕血黑便、外伤、腹痛背痛、妊娠相关腹痛/阴道流血。
- 发热/低体温、寒战、感染灶、免疫抑制或近期手术/导管。
- 颈静脉怒张 + 低血压，警惕 [[梗阻性休克]] 或 [[心源性休克]]。
- 皮疹/喘鸣/喉头水肿/暴露后急发，警惕 [[过敏性休克]]。
- “病人看起来很差”:焦虑躁动、濒死感、出汗、不能平卧、说话困难。

## 首查(到床后 [需查证: 本院要求时限] 分钟内)

- [ ] ABCDE + 立即叫上级；启动监护、除颤/抢救车到床旁。
- [ ] 生命体征趋势:血压、心率、呼吸频率、SpO2、体温、意识、尿量。
- [ ] 建立可靠静脉/骨髓/中心通路；抽血同时准备复苏。
- [ ] 血气 + 乳酸；SSC 2021 建议疑似脓毒症测乳酸，并在乳酸升高时将降低乳酸作为复苏目标之一 (Surviving Sepsis Campaign, 2021)。
- [ ] CBC、肾功能、电解质、肝功能、凝血、血糖、肌钙蛋白、BNP/NT-proBNP、血型和交叉配血。
- [ ] 12 导联 ECG:ACS、心律失常、高钾、右心负荷。
- [ ] 床旁超声:心脏收缩、右心负荷、心包积液、IVC、肺 B 线/气胸、腹腔游离液。
- [ ] CXR/CT/CTA:只在患者可安全转运且不会延误抢救时进行。
- [ ] 感染怀疑时采培养并启动抗菌药；出血怀疑时同步找出血源、联系外科/介入/输血。
- [ ] 留置尿管评估尿量；疑似尿道损伤需先叫上级/泌尿外科。

## 一线处理

### 1. 通用原则

- 休克是抢救状态:评估、复苏、病因控制必须并行。
- 氧疗/气道:低氧、意识差、误吸风险或呼吸衰竭时，尽早叫麻醉/ICU 评估气道。
- 液体:不要机械“见休克就大量补液”；根据类型、床旁超声、被动抬腿/小量试验和灌注反应动态评估。SSC 2021 建议脓毒症/脓毒性休克使用动态指标指导补液优于单靠体征或静态指标 (Surviving Sepsis Campaign, 2021)。
- 血管活性药:感染性休克需血管活性药时，SSC 2021 推荐 [[drugs/去甲肾上腺素|去甲肾上腺素]] 为一线，并建议初始 MAP 目标 65 mmHg (Surviving Sepsis Campaign, 2021)。
- 复评:每次干预后看血压、乳酸、尿量、意识、皮肤灌注、呼吸和肺部负荷。

### 2. 按类型处理

| 类型 | 一线方向 | 不能延误 |
|---|---|---|
| [[低血容量性休克]] | 控制出血/丢失 + 补容量/血制品 | 创伤出血需要止血、输血策略和损伤控制；中国创伤失血性休克专家共识 2023 强调快速识别、控制出血和纠正凝血障碍 |
| [[感染性休克]] | 抗感染 + 液体复苏 + 血管活性药 + 源控制 | SSC 2021 推荐治疗和复苏立即开始；高可能脓毒性休克抗菌药理想在识别后 1 小时内给药 |
| [[心源性休克]] | 评估 ACS/心律失常/机械并发症 + 血管活性/正性肌力 + 再灌注/机械支持 | 盲目补液可加重肺水肿；尽早心内/ICU |
| [[梗阻性休克]] | 解除梗阻 | 张力性气胸减压、心包填塞引流、肺栓塞再灌注策略 |
| [[过敏性休克]] | [[肾上腺素]] + 气道 + 液体 + 去除过敏原 | 肾上腺素是首要治疗；具体剂量按本院过敏反应专题和急救流程 |
| 神经源性休克 | 脊柱保护 + 血流动力学支持 | 排除出血性休克后再下结论 |

### 3. 源控制和团队沟通

- 出血源:外科、创伤、介入、妇产、消化内镜根据病因同步联系。
- 感染源:胆道、腹腔、尿路、肺部、导管、软组织坏死感染都需要源控制思维。
- 心源性:ACS、心律失常、瓣膜/机械并发症需心内科和 ICU。
- 梗阻性:张力性气胸、心包填塞、大面积 PE 是“解除梗阻”优先。

## 🔺 升级标准(立刻叫上级)

- 疑似任何类型休克。
- 低血压、乳酸升高、少尿、意识改变、皮肤花斑/湿冷或灌注差。
- 需要血管活性药、输血、气管插管、中心静脉/动脉置管或 ICU。
- 活动性出血、创伤、消化道出血、产科出血或抗凝相关出血。
- ACS、恶性心律失常、急性肺水肿或疑似心源性休克。
- 张力性气胸、心包填塞、肺栓塞等梗阻性病因。
- 过敏反应伴气道、呼吸或循环异常。
- 儿童、妊娠、免疫抑制、透析、单肾、严重基础心肺肾病。

## ⚠️ 常见坑

- 血压还没低就否认休克，漏掉代偿期组织低灌注。
- 所有休克都大量补液，导致心源性休克或肺水肿恶化。
- 感染性休克只补液不及时给抗菌药和做源控制。
- 创伤出血性休克只补晶体，不启动输血/止血/创伤团队。
- 休克患者离开监护去做 CT，途中恶化。
- 用血管活性药前后不复评灌注、乳酸、尿量和肺水肿。
- 忘记低血糖、肾上腺危象、过敏性休克等可逆病因。
- 没有明确记录“休克类型假设、已做处理、复评结果、已通知谁”。

## 关键评分

- **[[NEWS2]]**:早期识别急性恶化和触发升级。
- **[[scores/qSOFA|qSOFA]] / [[SOFA]]**:感染相关器官功能评估；SSC 2021 不推荐单独用 qSOFA 作为脓毒症/脓毒性休克筛查工具 (Surviving Sepsis Campaign, 2021)。
- **[[GCS]]**:意识和气道保护能力。
- **[[Shock Index]]**:可作为休克风险提示；具体阈值需查证后写入评分笔记。
- **[[scores/Wells-PE|Wells-PE]]**:疑似 [[emergencies/肺栓塞|肺栓塞]] 风险分层。
