# 急性胆囊炎(Acute Cholecystitis)

> [!summary] 30 秒摘要
> **红旗**:右上腹痛 + 发热 + Murphy 征阳性 + 超声胆囊壁厚 / 周围积液
> **首查**:CBC + CRP + 肝功(**多正常或轻升**;显著升 → 警觉胆管炎)+ **腹部超声**(首选)
> **一线**:**抗生素 + 早期腹腔镜胆囊切除(72h - 7 天内)**;高危 → 经皮胆囊造瘘
> **升级**:坏疽 / 穿孔 / 重度 + 老年 / 免疫抑制 → 普外 + ICU 评估

---

## 一句话定义

胆囊管阻塞(**95% 结石**)+ 胆囊壁炎症,可继发感染。

- **结石性**(90-95%):胆囊管或 Hartmann 袋结石阻塞
- **非结石性**(5-10%):重症患者(创伤、烧伤、ICU、心衰、糖尿病、HIV、TPN)→ 胆囊缺血 / 胆汁淤积;**死亡率高 30%**

> 关键鉴别:**急性胆囊炎 ≠ [[diseases/急性胆管炎|急性胆管炎]]**
> - 胆囊炎:胆囊本身炎症,胆红素 / ALP 多正常或轻升
> - 胆管炎:胆道**梗阻 + 感染**,Charcot 三联(发热 + 黄疸 + 右上腹痛)
> - **两者可并存**

来源:Tokyo Guidelines TG18 (Acute Cholecystitis);WSES Acute Calculous Cholecystitis 2020;中国急性胆道系统感染诊治指南 2021(中华急诊医学杂志)。

## TG18 严重度分级(决定手术 / 引流时机)

| 级 | 标准 | 处理 |
|---|---|---|
| **Grade I(轻度)** | 局部炎症,无器官功能障碍 | 早期 LC(72h - 7 天) |
| **Grade II(中度)** | 局部进展(白细胞 > 18,可触及肿块、症状 > 72h、明显局部炎症) | LC 仍可早期;高手术风险 → 抗生素 + PTGBD 桥接 |
| **Grade III(重度)** | 任一器官衰竭(循环 / 神经 / 呼吸 / 肝 / 肾 / 血液) | **抗生素 + PTGBD 立即** + ICU + 后期 LC |

来源:Tokyo Guidelines TG18(J Hepatobiliary Pancreat Sci 2018);中国 2021。

## 病理进展(漏诊的危险路径)

未治急性胆囊炎 → **气肿性胆囊炎(emphysematous,糖尿病多见)**→ **坏疽性胆囊炎**→ **胆囊穿孔**:
- 局限性穿孔(肝下脓肿)
- 自由性穿孔(弥漫性腹膜炎,死亡率 ~ 30%)
- 内瘘(胆囊-十二指肠 / 结肠;**胆石性肠梗阻**)

## 🚩 红旗(立刻警觉)

- **右上腹持续痛 > 6h**,放射至右肩 / 右肩胛
- **Murphy 征阳性**(深吸气时按压右上腹诱痛 + 屏吸)
- **超声 Murphy 阳性**(B 超探头按压痛部最痛点;敏感性更高)
- 发热 / 寒战
- 恶心 / 呕吐
- 老年 / 糖尿病 / 免疫抑制者**症状不典型**(可仅低热 + 不适)
- **重度征象**:
  - 坏疽 / 穿孔(腹膜征 / 包块 / 弥漫性腹膜炎)
  - 气肿性胆囊炎(腹片 / CT 见胆囊壁内气体)
  - 黄疸 / 转氨酶显著升 → 警觉合并 [[diseases/急性胆管炎|胆管炎]]
  - 脓毒症征象:低血压、心动过速、灌注差

## 首查

### 立即(到院 1h 内)

- [ ] 生命体征 + qSOFA / NEWS2
- [ ] 静脉通路 + 心电监护
- [ ] **CBC + CRP + PCT + 电解质 + 肌酐 + 肝功(ALT/AST/胆红素/ALP/γ-GT) + 凝血 + 血型**
- [ ] 淀粉酶 / 脂肪酶(排除胰腺炎合并)
- [ ] 乳酸(脓毒症怀疑)
- [ ] **HCG**(育龄女性)
- [ ] **腹部超声(US)** — **首选影像**(便宜、床旁、敏感性 > 90%)
  - 胆囊壁厚 ≥ 4 mm
  - 胆囊周围积液
  - 胆囊增大
  - 结石嵌顿胆囊管 / Hartmann 袋
  - **超声 Murphy 征**
- [ ] **HIDA / 胆管闪烁显像**(US 不明确时):胆囊不显影 = 胆囊管阻塞;敏感性、特异性最高,但耗时
- [ ] **腹部 CT 增强**:并发症评估(穿孔、脓肿、气肿性、肠梗阻)/ 老年 / 不明确诊断
- [ ] MRCP(怀疑合并胆总管结石 + 胆管炎)
- [ ] 血培养 ×2 套(中重度 / 脓毒症)

## TG18 诊断标准

诊断 = **A + B + C 各至少 1 项**:
- **A 局部炎症体征**:Murphy 征 / RUQ 包块 / 痛 / 压痛
- **B 全身炎症体征**:发热 / 白细胞或 CRP 升高
- **C 影像证据**:超声 / CT / MRI 急性胆囊炎征象

来源:Tokyo Guidelines TG18 2018。

## 一线处理

### 1. 共同处理

- 禁食 + IV 补液(乳酸林格)
- 镇痛(NSAID / 阿片;别因"怕掩盖诊断"不给镇痛 — 现代证据支持)
- 止吐(昂丹司琼)
- 监测尿量、生命体征

### 2. 经验性抗生素(< 1h 启动)

| 严重度 / 情境 | 经验首选 |
|---|---|
| **轻度(Grade I)社区** | **头孢曲松 + 甲硝唑** 或单用头孢哌酮 / 哌拉西林他唑巴坦 |
| **中度(Grade II)** | **哌拉西林他唑巴坦** 或 厄他培南 |
| **重度(Grade III)/ 院内 / 反复** | **美罗培南** / 哌拉他唑(覆盖 ESBL)± 万古(若 MRSA 风险) |
| 过敏 β-内酰胺 | 喹诺酮 + 甲硝唑;具体方案 [需查证: 本院抗菌药物分级] |

**疗程**:
- 早期 LC + 临床改善 → 24-72h(炎症局限)
- LC 后无并发症 → 不需长期抗生素
- 穿孔 / 脓肿 / 菌血症 → 4-7+ 天

来源:Tokyo Guidelines TG18;WSES 2020;IDSA 复杂腹腔感染 2017 update;中国 2021。

### 3. 决定性治疗(关键!)

#### 早期腹腔镜胆囊切除(LC)— 首选

> [!danger] **TG18 / WSES / 中国 2021 强推荐:症状起病 < 7 天(理想 < 72h)内 LC**
> RCT(Loozen 2018, ACDC trial)显示早期 LC 优于"先抗生素 + 延迟手术":住院短、并发症少、复发少。

| 情境 | 处理 |
|---|---|
| Grade I + 手术耐受 | LC 早期(< 7 天) |
| Grade II + 手术耐受 | LC 早期 |
| Grade II + 高手术风险 | 抗生素 + 评估 PTGBD 桥接 |
| Grade III(器官衰竭) | **PTGBD 立即** + ICU + 稳定后延迟 LC(6-12 周) |
| 妊娠期 | 非急诊期(妊娠中期最安全)— LC 仍首选 |
| 老年 / 高合并症不适合 LC | PTGBD ± 期待 |

#### 经皮经肝胆囊穿刺引流(PTGBD)

- **适应证**:Grade III / 高手术风险患者 / 抗生素 24h 不响应 / 桥接到 LC
- 床旁超声 / CT 引导;成功率 > 90%
- 完成桥接后 6-12 周择期 LC(否则复发率 > 30%)

#### 内镜下经乳头胆囊引流(ETGBD)/ 内镜超声引导(EUS-GBD)

- 高手术风险 + PTGBD 不可行 → 经验中心选择

### 4. 病理 / 标本

- LC 标本送病理(警觉胆囊癌,**1-2% > 65 岁**)

## 🔺 升级标准

> [!danger]
> - 弥漫性腹膜炎 / 板状腹 → 立即外科 + 备血 + 备手术
> - 气肿性胆囊炎(影像见胆囊壁气体)→ 急诊手术
> - 坏疽性胆囊炎(超声胆囊壁不规则、不连续)
> - 脓毒性休克 → ICU + 1h bundle + 立即引流
> - 黄疸 + 高热 + 右上腹痛(Charcot)→ 警觉合并胆管炎,**ERCP 路径**
> - 老年 / 糖尿病 / 免疫抑制 + 任何不稳定 → 低阈值升级
> - 抗生素 24-48h 不响应 → 重新影像 + 引流评估

## ⚠️ 常见坑

> [!warning]
> 1. **没做超声 Murphy** → 体表 Murphy 不如超声敏感
> 2. **肝功正常就排除胆道病** → 急性胆囊炎肝功多正常或轻升;**显著升 = 警觉胆管炎合并**
> 3. **老年 / 糖尿病不典型** → 仅低热 + 食欲差,易漏 → 低阈值影像
> 4. **认为"先观察 6 周再做手术"是标准** → 早期 LC < 7 天才是现代标准
> 5. **Grade III 直接 LC** → 死亡率高;先 PTGBD 桥接
> 6. **气肿性胆囊炎漏诊** → 糖尿病 + 胆囊壁气体 → 急诊外科,死亡率 15-25%
> 7. **非结石性胆囊炎漏诊** → ICU / 重症 / TPN + 不明原因发热 → 影像
> 8. **抗生素当主要治疗** → 抗生素 + 引流 / 手术才是核心
> 9. **PTGBD 后忘了延迟 LC** → 复发率 > 30%
> 10. **胆石性肠梗阻漏诊** → 老年 + 肠梗阻 + 胆道气体(Rigler 三联)= 内瘘 + 结石移位
> 11. **胆囊癌漏检** → > 65 岁 + 胆囊壁不均匀增厚 / 包块 → 病理必送
> 12. **DKA 表现腹痛误为胆囊炎** → 糖尿病人新发腹痛 → 测血糖 + 酮 + ABG
> 13. **下壁 MI 误为胆囊炎** → 老年上腹 / 右上腹痛 → ECG 永远先做
> 14. **妊娠期保守治疗结果反复** → 妊娠中期 LC 是优选
