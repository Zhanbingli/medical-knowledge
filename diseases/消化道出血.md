# 消化道出血(GI Bleeding, GIB)

> [!summary] 30 秒摘要
> **红旗**:呕血 / 黑便 / 便血 + 心动过速 / 低血压 / 苍白 / 意识改变
> **首查**:**Glasgow-Blatchford 评分** + 血型 + 交叉配血 + 凝血 + Hb;不稳定 → 立即复苏 + 内镜
> **一线**:**复苏(晶体 + 限制性输血 Hb 目标 7)** + **PPI 80mg bolus + 8mg/h**(UGIB)+ 内镜 < 24h;静脉曲张加 [[奥曲肽 / 特利加压素]] + [[头孢曲松]]
> **升级**:不稳定 / 呕血量 / 持续黑便 → ICU + 内镜中心 24h 通道

---

## 一句话定义

**Treitz 韧带**为分界:

- **上消化道出血(UGIB)**:Treitz 上方 → 表现呕血 / 咖啡渣样吐出 / 黑便(柏油样);严重也可出现便血
- **下消化道出血(LGIB)**:Treitz 下方 → 表现便血 / 鲜血 / 暗红血;少数严重 UGIB 表现便血
- **隐性出血(occult bleeding)**:化验阳性,无明显临床表现
- **不明原因出血(obscure bleeding)**:常规内镜未找到原因 → 小肠源(胶囊内镜 / 双气囊)

来源:ACG Upper Gastrointestinal Bleeding 2021;ESGE Non-variceal UGIB 2021;BSG Acute Lower GIB 2019;中国急性非静脉曲张性上消化道出血诊治指南 2018(中华内科杂志);中国肝硬化门静脉高压食管胃静脉曲张出血诊治指南 2022。

## 病因

### UGIB(常见 → 罕见)

| 病因 | 占比 | 备注 |
|---|---|---|
| **消化性溃疡(PUD)** | 30-50% | NSAIDs / Hp / 应激;Forrest 分型 |
| **食管胃底静脉曲张** | 5-15% | 肝硬化门脉高压;**死亡率最高** |
| 食管炎 / Mallory-Weiss 撕裂 | 5-15% | 反流;剧烈呕吐后 |
| 胃糜烂 / 出血性胃炎 | 5-15% | NSAIDs / 应激 / 酒精 |
| 上消化道恶性肿瘤 | 5% | 老年慢性 + 体重下降 |
| **Dieulafoy 损害** | < 5% | 隐匿动脉破裂,**反复大出血** |
| 主动脉肠瘘 | 罕见 | 主动脉术后 + 上消化道大出血 |

### LGIB(常见 → 罕见)

| 病因 | 占比 | 备注 |
|---|---|---|
| **憩室出血** | 30-50% | 老年最多;多自限 |
| **血管发育不良 / 血管畸形** | 5-20% | 老年、慢性肾病 |
| 痔疮 / 肛裂 | 5-15% | 鲜血但量少 |
| **缺血性结肠炎** | 5-10% | 老年 + 心血管病 |
| **炎症性肠病(IBD)** | 5-10% | 年轻 + 慢性腹泻 |
| 结直肠肿瘤 / 息肉 | 5-15% | 老年 + 慢性 |
| 息肉切除术后出血 | < 5% | 内镜术后数日 |
| 放射性结肠炎 | 罕见 | 盆腔放疗史 |

## 🚩 红旗(立刻警觉)

- 呕血 / 咖啡渣样物 / 黑便(柏油 — 上)/ 便血(下,但 UGIB 量大也可)
- HR > 100 / SBP < 100 / 体位性血压下降 ≥ 20 / 心率上升 ≥ 20(隐匿失血)
- 苍白 / 出汗 / 焦虑 / 意识改变
- BUN / 肌酐比值 > 30(支持 UGIB,血红蛋白消化吸收致 BUN 升)
- Hb 急剧下降(注意急性出血初期 Hb 可正常,因比例稀释滞后)
- **静脉曲张高危**:已知肝硬化、腹水、黄疸、脾大、蜘蛛痣
- 持续呕血 + 不能稳定 = ICU + 内镜 + IR / 外科备
- 老年 + 抗凝 / 抗血小板 + 大量便血 → 血管畸形 / 缺血性结肠炎可能

## 鉴别诊断(看似消化道出血但不是)

| 情况 | 区分点 |
|---|---|
| 鼻 / 咽出血 → 吞下后呕出 | 鼻咽检查 |
| 口腔出血 | 口检 |
| 咯血(血管 / 肺源) | 鲜红 + 泡沫 + 咳嗽前驱 |
| 食用红色食物 / 药物(铋剂、铁剂、活性炭、甜菜)模拟黑便 / 便血 | 病史 + 大便潜血 |
| 月经 / 阴道出血污染 | 病史 |
| 大量血液下消化道经直肠排出 vs 上消化道大量快速排出 | 不能仅凭外观区分 → **NG 灌洗或快速内镜** |

## 首查

### 立即(到院 30 分钟内)

- [ ] 生命体征 + qSOFA / NEWS2 + 一眼判断
- [ ] **2 条粗静脉通路(16-18G)** + 心电监护 + SpO2
- [ ] 血型 + 交叉配血(立即备 4U RBC,不稳定 6U)
- [ ] **CBC** + 电解质 + 肌酐 + **BUN**(BUN/Cr > 30 提示 UGIB)
- [ ] 肝功 + 凝血(PT / INR / APTT)+ 血气 + 乳酸
- [ ] **ECG**(心血管病人 / 老年)+ 肌钙蛋白(休克可致 II 型 MI)
- [ ] 妊娠试验(育龄女性)
- [ ] 抗凝 / 抗血小板用药史(华法林 / DOAC / 双抗)
- [ ] 既往内镜 / 肝硬化 / 主动脉手术史

### 评分(分诊用)

- [ ] **Glasgow-Blatchford(GBS)**(0-23):入院初评,**= 0 可考虑门诊处理**(BSG / NICE 推荐);> 0 需评估
- [ ] **Rockall** 完整(内镜后)/ 简化(内镜前)
- [ ] **AIMS65**:5 项简单
- [ ] **Oakland**(LGIB):≤ 8 可考虑门诊处理

### 影像 / 内镜决策

- [ ] **NG 灌洗 / 抽吸**:历史用,现代弱化(可阴性 + 不必要痛苦);仅特定情境
- [ ] **上消化道内镜(EGD)**:UGIB 诊断 + 治疗,**< 24h(稳定)/ < 12h(高危如静脉曲张)**;太早(< 6h)无获益反风险
- [ ] **结肠镜(LGIB)**:24-48h 内,需肠道准备
- [ ] CTA:大量便血不稳定 / 内镜不能完成 → 介入血管造影 + 栓塞
- [ ] 标记红细胞核素扫描:出血率低但持续(0.1-0.5 mL/min)
- [ ] 胶囊内镜:不明原因小肠出血

来源:ACG UGIB 2021;BSG Acute LGIB 2019;ESGE 2021;中国 UGIB 2018。

## 一线处理

### 0. 复苏(优先于任何检查)

- **晶体液**(平衡盐 / 乳酸林格);避免大量生理盐水(高氯酸中毒)
- **限制性输血策略**:
  - 目标 **Hb 7 g/dL**(70 g/L)— TRICC、Villanueva 2013(UGIB);心血管病者 Hb 8 g/dL
  - **过度输血(> 9)反增静脉曲张再出血**(Villanueva NEJM 2013)
- **大量出血协议(Massive Transfusion Protocol)**:RBC : FFP : Plt = 1:1:1(创伤 PROPPR 类比)
- 凝血校正:INR > 1.5 → FFP 或 PCC;血小板 < 50 → 输血小板;纤维蛋白原 < 1.5 → 冷沉淀

### 1. 抗酸(UGIB)

| 药 | 用法 | 适应证 |
|---|---|---|
| **PPI**(埃索美拉唑 / 奥美拉唑) | **80 mg IV bolus + 8 mg/h × 72h** | 内镜前经验性 + 高危 PUD 内镜后(Forrest Ia/Ib/IIa/IIb) |
| 内镜后低危 PUD(Forrest IIc/III) | 口服 PPI bid | 短疗程 |

来源:ACG 2021;中国 2018。

### 2. 静脉曲张特殊处理

> [!danger] 静脉曲张大出血死亡率 15-20%,需联合多线治疗

- **血管活性药物**(立即给,内镜前)
  - **奥曲肽 50 µg IV bolus + 50 µg/h** 持续 3-5 天
  - 或 **特利加压素 2 mg IV q4h**(部分国家;心血管病人慎用)
- **预防性抗生素**(降低感染 + 再出血 + 死亡率):**头孢曲松 1 g IV qd × 5-7 天**
- **内镜下治疗**:静脉套扎(EVL,首选)/ 硬化(EIS)
- 不能控制 → **TIPS** / Sengstaken-Blakemore 管(暂时桥接)
- 后续二级预防:非选择性 β 阻滞剂(普萘洛尔 / 卡维地洛)+ EVL

来源:Baveno VII 2022;ACG Variceal Bleeding;中国肝硬化门静脉高压 2022。

### 3. 反转抗凝 / 抗血小板(关键!)

| 药 | 处置 |
|---|---|
| **华法林** | 维生素 K 5-10 mg IV + **4F-PCC**(凝血酶原复合物)优于 FFP |
| **达比加群** | **idarucizumab 5 g IV** |
| **Xa 抑制剂**(利伐沙班 / 阿哌沙班 / 艾多沙班)| **andexanet alfa**(若可得)/ **4F-PCC 50 U/kg** |
| **肝素** | 鱼精蛋白 1 mg / 100 U UFH |
| **LMWH** | 鱼精蛋白部分中和(~ 60%) |
| **抗血小板**(阿司匹林 / P2Y12) | **不能逆转**;严重出血输血小板 |

> 来源:ACG / ASH / 中国心血管病急救;具体本院药品可用性 [需查证]。

### 4. 内镜治疗

| 损伤 | 治疗 |
|---|---|
| **PUD 高危(Forrest Ia/Ib/IIa)** | 注射 + 热凝 / 止血夹;部分 IIb 视情况 |
| **PUD 低危(Forrest IIc/III)** | 不需内镜治疗,口服 PPI |
| 静脉曲张 | 套扎 / 硬化 |
| Mallory-Weiss | 多自限,出血点止血夹 |
| 血管畸形 / Dieulafoy | 热凝 / 夹闭 |
| 憩室出血 | 夹闭 / 注射 |

### 5. Hp 检测 / 治疗

- **任何 PUD 出血 → Hp 检测 + 阳性根除**(降低复发)
- 急性出血期假阴性高 → **出院后 4 周复查**

### 6. 介入栓塞 / 外科

- 内镜失败 + 不稳定 → **介入血管造影 + 栓塞**
- 不可栓塞 / 反复失败 → 外科

## 🔺 升级标准

> [!danger]
> - 不稳定(SBP < 90 / HR > 130)→ ICU + 大量输血协议
> - 持续呕血 / 大量便血不能控
> - GBS ≥ 12 / Rockall ≥ 6 / AIMS65 ≥ 2
> - 已知 / 怀疑静脉曲张 → ICU + 内镜中心 + IR 备
> - 反复出血 / 内镜失败 → IR / 外科会诊
> - 抗凝 / 出血倾向 + 大出血 → 紧急逆转
> - 任何"病人在恶化"

## ⚠️ 常见坑

> [!warning]
> 1. **复苏不足就送内镜** → 内镜下血压崩,镇静后心搏骤停
> 2. **过度输血(目标 Hb > 9)** → 静脉曲张再出血风险升 + 死亡率升(Villanueva 2013)
> 3. **大量便血不查 UGIB** → 上消化道大量快速也可表现便血;**NG 灌洗 / 紧急 EGD 不能省**
> 4. **静脉曲张未给抗生素预防** → 感染 / 再出血 / 死亡率均升
> 5. **静脉曲张未给奥曲肽 / 特利就送内镜** → 增加内镜难度 + 失败率
> 6. **PPI 仅 IV 没持续输注** → 高危 PUD 需要 80 mg + 8 mg/h × 72h
> 7. **抗凝未逆转** → 持续出血,任何时间都可
> 8. **内镜过早(< 6h)或过晚(> 24h)** → 早无获益反风险,晚增加死亡
> 9. **PUD 出血没查 Hp / 没根除** → 1 年复发率 ≥ 25%
> 10. **内镜结束就停 PPI** → 低危可口服 bid,高危需 IV 72h 后改口服 8 周
> 11. **NG 阴性当排除 UGIB** → 阴性预测值 ~ 60-70%,不可靠
> 12. **抗血小板单药 ACS 病人 PUD 出血"停一切"** → 二级预防有适应证,**急性期短暂停 + 早期重启**(评估出血风险与血栓权衡)
> 13. **下肢制动 + GIB 不预防 VTE** → 出血控制后机械预防,稳定后启动药物
> 14. **慢性 LGIB 当痔疮**(老年) → 漏诊结肠癌
> 15. **铁剂 / 铋剂粪便变黑误为黑便** → 病史 + 潜血确认
> 16. **主动脉肠瘘漏诊** → 主动脉术后 + 上消化道大出血 → 高度怀疑 + CTA

## 关键评分

- **[[Glasgow-Blatchford]](GBS,0-23)**:**入院前评估**,= 0 可考虑门诊;> 0 需 EGD

- **Forrest 分型**(PUD 内镜):Ia 喷射 / Ib 渗 / IIa 可见血管 / IIb 凝块附着 / IIc 黑斑 / III 干净
