# 细菌性脑膜炎(Bacterial Meningitis)

> [!summary] 30 秒摘要
> **红旗**:发热 + 头痛 + 颈强直 + 意识 / 认知改变(经典四联征,任意 2 项即筛查)+ 不褪色瘀点(脑膜炎球菌)
> **首查**:**< 1 小时**经验性抗生素 + 地塞米松;血培养 → CT(若指征)→ LP
> **一线**:**头孢曲松 2 g IV q12h + 万古霉素**(地区耐药),老 / 免疫抑制加 **阿莫西林**(覆盖 Listeria)+ **地塞米松**(首剂前 / 同时)
> **升级**:任何怀疑 → 立即上级 + 1 小时内启动经验治疗

---

## 一句话定义

细菌侵入软脑膜与蛛网膜下腔引起的急性化脓性炎症,死亡率 10-30%,后遗症 30-50%。**每延误 1 小时抗生素 → 死亡率上升**。

来源:NICE Bacterial Meningitis NG240, 2024;IDSA Practice Guidelines for Bacterial Meningitis 2017;中国成人社区获得性细菌性脑膜炎诊治专家共识 2015(中华神经科杂志)。

## 病原(决定经验性抗生素)

### 成人(社区获得性)

| 病原 | 占比 | 重点人群 |
|---|---|---|
| **肺炎链球菌**(S. pneumoniae) | 50-70% | 所有年龄,最常见 |
| **脑膜炎奈瑟菌**(N. meningitidis) | 10-30% | 青少年 / 年轻成人;**瘀点紫癜暴发性** |
| **Listeria monocytogenes** | 5% | **> 50 岁、免疫抑制、孕妇、酗酒** |
| 流感嗜血杆菌 | < 5%(疫苗后下降) | 儿童 / 老年 |
| 革兰阴性杆菌 | < 5% | 老年、神外术后、HIV |
| 金黄色葡萄球菌 | 罕见 | 神外操作 / 引流管 / IE 栓子 |

### 院内 / 术后 / 有引流管

铜绿假单胞菌、葡萄球菌、革兰阴性 → **覆盖范围更广**

## 🚩 红旗(立刻警觉)

> [!danger] 经典四联征(全部具备 < 50%,但任意 2 项即应启动评估)
> - **发热**
> - **头痛**(剧烈、持续、对镇痛反应差)
> - **颈强直 / 脑膜刺激征**
> - **意识 / 认知改变**

**其他高警觉信号**:
- **不褪色瘀点 / 紫癜**(玻片按压不退色)→ **[[脑膜炎球菌病]] / 暴发性紫癜**——数小时内致死
- 抽搐(初次发病可见)
- 局灶神经定位(脑炎 / 脑脓肿叠加)
- 脓毒症征象:低血压、心动过速、毛细血管再充盈差
- 老年 / 免疫抑制 / 婴儿:可**无典型脑膜征**——表现为单纯意识改变 / 发热 / 反应差
- 颅高压征象:库欣三联(高血压 + 心动过缓 + 不规则呼吸)、瞳孔不等大

## 鉴别诊断

| 疾病 | 区分点 | 关键检查 |
|------|--------|---------|
| **病毒性脑膜炎** | 发热 + 头痛但意识较少改变,CSF 淋巴细胞为主、葡萄糖正常、培养阴性 | LP + PCR |
| **病毒性脑炎**(HSV / 自免) | 发热 + 意识 + **抽搐 / 行为异常 / 局灶**,颞叶受累 | MRI + LP + HSV PCR |
| [[emergencies/蛛网膜下腔出血|蛛网膜下腔出血]] | **雷击样头痛** + 颈强 + CT 高密度 | CT + LP(黄变) |
| 真菌性 / 结核性脑膜炎 | 亚急性 + 免疫抑制 / TB 接触 | CSF 染色 / 培养 + ADA |
| 脑脓肿 | 发热 + 局灶 + 影像环形强化 | MRI + 抽吸培养 |
| 自身免疫性脑炎(NMDA 等) | 年轻 + 精神症状 + 抽搐 + 自主神经紊乱 | 抗体 |
| 偏头痛 + 颈痉挛 | 既往史 + 阴性查体 + LP 阴性 | 排除性 |
| 颈源性头痛 | 慢性 + 颈活动相关 | 病史 |
| 急性鼻窦炎 + 全身症状 | 局部压痛 + 影像 | 临床 |

## 首查

### 立即(< 1 小时,病人到院起算)

> [!warning] 时间目标:**Door-to-antibiotic ≤ 1h**
> 不要因等检查延迟抗生素。

- [ ] 生命体征 + 意识(GCS)+ qSOFA / NEWS2
- [ ] 静脉通路 ×2 + 心电监护
- [ ] **血培养 ×2 套**(抗生素前;但**不应延迟抗生素 > 30min**)
- [ ] CBC、电解质、肌酐、肝功、凝血、CRP、PCT、乳酸、血糖
- [ ] 妊娠试验(育龄女性)
- [ ] **是否需要 CT 头**(下方)— 是 → 抗生素 + 地塞米松立刻给后再 CT,不要等
- [ ] **腰穿(LP)+ CSF 分析**

### CT 头先于 LP 的指征(IDSA / NICE 推荐)

- [ ] 免疫抑制 / HIV
- [ ] 既往中枢神经系统疾病
- [ ] 新发癫痫(< 1 周)
- [ ] **意识改变 / GCS < 12**
- [ ] **局灶神经定位**
- [ ] 视乳头水肿
- [ ] **凝血障碍 / 抗凝中**(LP 出血风险)

> 没有这些指征 → **直接 LP**;有任一 → CT 排除占位 / 移位 / 后颅窝病变后再 LP。
> **任何 CT 等待都不应延迟抗生素和地塞米松。**

### CSF 分析(典型表现)

| 指标 | 细菌性 | 病毒性 | 结核性 | 真菌性 |
|---|---|---|---|---|
| 外观 | 浑浊 / 脓性 | 清亮 | 黄色 / 毛玻璃 | 清亮 / 雾 |
| 压力(cmH2O) | ↑↑ (> 25) | 正常 / ↑ | ↑ | ↑↑ |
| 白细胞 / mm³ | > 1000(中性为主)| 50-1000(淋巴)| 100-500(混合)| 50-500 |
| 蛋白 g/L | ↑ > 1 | 轻度 ↑ | ↑↑ | ↑ |
| 葡萄糖 | **↓↓**(< CSF/血 0.4)| 正常 | ↓ | ↓ |
| 革兰染色 | + 60-90% | — | — | — |
| 培养 | + | PCR 病原 | 培养慢、ADA、PCR | 墨汁 / 抗原 |

来源:IDSA 2017;NICE NG240 2024;中国细菌性脑膜炎共识 2015。

## 一线处理(经验性,**1 小时内**)

### 1. 经验性抗生素(按年龄 + 高危)

| 人群 | 经验方案 |
|---|---|
| **2 月 - 50 岁** 免疫正常 | **头孢曲松 2 g IV q12h** + **万古霉素 15-20 mg/kg IV q8-12h**(地区肺炎链球菌耐药 ≥ 5%) |
| **> 50 岁 / 免疫抑制 / 孕妇 / 酗酒** | 上述 + **阿莫西林 / 氨苄西林 2 g IV q4h**(覆盖 Listeria) |
| **β-内酰胺过敏** | 万古 + 莫西沙星 / 美罗培南(过敏史评估) |
| **神外术后 / 引流管 / 院内** | 万古 + 头孢吡肟 / 美罗培南(覆盖铜绿、葡萄球菌) |
| 儿童(请儿科会诊) | 头孢曲松 + 万古;< 3 月加氨苄(GBS、Listeria、E.coli) |

> 来源:IDSA 2017;NICE NG240 2024;中国细菌性脑膜炎共识 2015;具体本院抗菌药物分级与剂型 [需查证]。

### 2. 辅助类固醇

> [!danger] **地塞米松 0.15 mg/kg IV q6h × 4 天**
> - **必须首剂抗生素前 10-20 分钟,或与首剂同时给**
> - 之后才给若 CSF 染色为肺炎链球菌
> - 来源:de Gans NEJM 2002;IDSA 2017;NICE NG240 2024
> - 减少听力损失与神经后遗症,主要在肺炎链球菌

### 3. 颅高压管理

- 抬头 30°
- 镇静 + 充分镇痛
- 严重颅高压 / 即将脑疝:**[[甘露醇]] 0.25-1 g/kg IV** 或 **3% 高渗盐水**
- 避免过度补液
- 神外评估(EVD 引流)

### 4. 抗癫痫

- 出现抽搐 → 苯二氮䓬 + 长期(左乙拉西坦)
- 不再常规预防性

### 5. 病原确认后调整

| 病原 | 调整 |
|---|---|
| 肺炎链球菌(敏感) | 头孢曲松或青霉素继续,共 10-14 天 |
| 肺炎链球菌(耐药) | 加万古或利福平 |
| 脑膜炎奈瑟菌 | 头孢曲松 7 天;**密切接触者预防**(利福平 / 环丙 / 头孢曲松) |
| Listeria | 阿莫西林 + 庆大霉素 21 天 |
| 流感嗜血杆菌 | 头孢曲松 7-10 天 |
| 革兰阴性 | 头孢吡肟 / 美罗培南 21 天 |

### 6. 公共卫生(脑膜炎球菌)

- 立即上报传染病
- 密切接触者(家庭、托儿、医护暴露)化学预防
- 隔离飞沫至有效抗生素 24h 后

## 🔺 升级标准

> [!danger]
> - 任何怀疑 → 立刻打电话 + 1 小时内启动抗生素 + 地塞米松
> - GCS < 12 / 意识进行性下降 / 抽搐持续 / 局灶定位 → ICU + 神外评估
> - 暴发性紫癜 / 脓毒性休克 → ICU + 升压 + 脓毒症 1h bundle
> - 颅高压征象(瞳孔异常、库欣三联) → 立即降颅压 + 神外评估 EVD
> - 不响应抗生素 48-72h 不退热 → 重新评估病原 / 并发症(脓肿、积液、脑积水)
> - 任何"病人在恶化"

## ⚠️ 常见坑

> [!warning]
> 1. **等 CT / LP 才给抗生素** → 1h 即至关重要,先抽培养 + 给抗生素 + 地塞米松,再做检查
> 2. **抗生素前未抽血培养** → 病原诊断率显著下降,但**急救 > 培养**
> 3. **地塞米松没在抗生素前 / 同时给** → 失去神经保护意义(炎症已激活)
> 4. **老年 / 免疫抑制者临床表现不典型** → 单纯意识改变 / 发热,无颈强,易漏
> 5. **婴儿表现非典型** → 喂养差 / 烦躁 / 囟门膨隆
> 6. **没问"是否 > 50 岁 / 孕妇 / 酗酒 / 免疫抑制"** → 漏覆盖 Listeria
> 7. **β-内酰胺过敏没正确替代方案** → 万古 + 莫西沙星 / 美罗培南
> 8. **LP 后颅压问题** → 颅高压未充分评估前 LP 可致脑疝
> 9. **暴发性紫癜被当一般皮疹** → 玻片按压不退色 = 红色警报
> 10. **密接预防遗漏** → 脑膜炎球菌家属暴露未化学预防 → 群体爆发风险
> 11. **不报告疑似传染病** → 法定报告漏掉
> 12. **抗生素疗程不够 / 提前停** → 复发风险
> 13. **CSF 葡萄糖正常就排除细菌性** → 早期可正常,需结合白细胞计数和分类
> 14. **LP 前不查凝血** → 抗凝 / 凝血障碍致硬膜下血肿 / 椎管内出血
> 15. **复查 LP 时机错** → 通常治疗 2-3 天后症状不缓解才复查
