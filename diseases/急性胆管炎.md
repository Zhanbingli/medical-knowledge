# 急性胆管炎(Acute Cholangitis)

> [!summary] 30 秒摘要
> **红旗**:**Charcot 三联**(发热 + 黄疸 + 右上腹痛);**Reynolds 五联**(+ 休克 + 意识)= 重症
> **首查**:CBC + 肝功 + 胆红素 + CRP + PCT + 血培养 + **腹部超声**(胆总管扩张 / 结石)
> **一线**:**广谱抗生素 < 1h** + **胆道引流**(ERCP < 24-48h);重症立即引流
> **升级**:Reynolds 五联 / 脓毒症 / 引流后无改善 → ICU + 紧急 ERCP

---

## 一句话定义

胆道系统(肝外胆管 / 肝内胆管)**梗阻 + 感染**导致的化脓性炎症。

> 关键概念:**梗阻 + 感染并存才是胆管炎**。单纯结石无感染 = 胆绞痛 / 胆石症,不算胆管炎。

来源:Tokyo Guidelines TG18(2018);中国急性胆道系统感染诊治指南 2021(中华急诊医学杂志 / 中国医师协会急诊医师分会);ESCP / ASGE 共识。

## 🚩 红旗(立刻警觉)

### Charcot 三联(诊断阳性,但发生率仅 50-70%)

- **发热**(寒战常见)
- **黄疸**
- **右上腹痛**

### Reynolds 五联(三联 + 重症)

- + **休克 / 低血压**
- + **意识改变**

→ **重症胆管炎 / 急性梗阻性化脓性胆管炎(AOSC)**,死亡率 > 30%(无引流)

### 其他高警觉

- 老年 / 免疫抑制可表现不典型(无典型三联,仅意识改变 + 发热)
- 既往胆道手术 / 支架 / ERCP
- 已知胆石症 / 肝吸虫流行区

- [ ] **MRCP**(超声不明确 + 稳定时)
- [ ] **腹部 CT 增强**(快速诊断 + 评估并发症)
- [ ] 妊娠试验

### Tokyo TG18 诊断标准

诊断 = **A + B + C 各至少 1 项**:
- **A 全身炎症**:发热 / 寒战;白细胞 / CRP 异常
- **B 胆汁淤积**:黄疸 / 异常肝功
- **C 影像证据**:胆道扩张 / 病因(结石、狭窄、支架等)

### TG18 严重度分级

| 级 | 标准 |
|---|---|
| **Grade III(重度)** | 任一器官功能障碍:循环 / 神经 / 呼吸 / 肝 / 肾 / 血液 |
| **Grade II(中度)** | 任意 2 项:WBC > 12 或 < 4;高热 ≥ 39℃;年龄 ≥ 75;胆红素 ≥ 5 mg/dL;白蛋白 < 0.7× LLN |
| **Grade I(轻度)** | 不符合 II / III |

来源:Tokyo Guidelines TG18(Hepatobiliary Pancreat Sci 2018)。

## 一线处理

### 1. 复苏(重症)

- 晶体液(乳酸林格,首选)
- 升压药维持 MAP ≥ 65(脓毒性休克 → [[drugs/去甲肾上腺素|去甲肾上腺素]] 0.05-0.1 µg/kg/min 起)
- 监测尿量 / 乳酸 / 末梢灌注
- 不稳定 → 走 [[emergencies/脓毒症|脓毒症]] 1h bundle

### 2. 抗生素(< 1h)

> [!danger] 不应因等待引流延误抗生素

| 情境 | 经验首选 |
|---|---|
| **社区获得性 + 轻 / 中度** | **头孢曲松 2 g IV qd + 甲硝唑 500 mg IV q8h** |
| **社区获得性 + 重度** | **哌拉西林他唑巴坦 4.5 g IV q6h** 或 **厄他培南 1 g IV qd** |
| **院内 / ERCP 后 / 反复治疗** | **美罗培南 1 g IV q8h ± 万古**(覆盖 ESBL / 铜绿) |
| 过敏 β-内酰胺 | 喹诺酮 + 甲硝唑;具体方案 [需查证: 本院抗菌药物分级] |

**疗程**:
- 引流成功 + 临床改善 → **总 4-7 天**(TG18 / 中国 2021)
- 菌血症持续 / 控制差 / 肝脓肿 → 14 天 +
- 培养结果出来后**降阶梯**

来源:Tokyo Guidelines TG18;中国急性胆道系统感染诊治指南 2021;IDSA 复杂腹腔感染 2017 update。

### 3. 胆道引流(决定性治疗,核心!)

> [!danger] **抗生素 + 引流缺一不可**
> 引流时机:
> - **重度(Grade III)** → **紧急 ERCP < 24h**(TG18 强推荐)
> - **中度(Grade II)** → 早期 ERCP 24-48h 内
> - **轻度(Grade I)** → 抗生素后初始改善;72h 内 ERCP 或择期(取决于结石 / 狭窄)

| 引流方式 | 适应证 |
|---|---|
| **ERCP + 取石 + 鼻胆引流(ENBD)/ 支架** | **首选**;成功率 > 90% |
| **PTCD**(经皮经肝胆道引流) | ERCP 失败 / 不可达(肝门部肿瘤 / 解剖异常 / Roux-en-Y 术后) |
| **EUS 引导下胆道引流** | ERCP / PTCD 失败;经验中心 |
| 外科 T 管引流 | 介入失败 / 不可得;现少用 |

### 4. 胆囊切除(后续)

- 胆源性 + 病情稳定后 → **2-6 周内胆囊切除**(预防复发);老年高危可保守
- 急性期不进行(增加风险)

### 5. 监测 + 调整

- 24h 内热退 / 改善 → 抗生素继续按疗程
- 不响应 48-72h → 重新评估:引流是否充分?并发症(肝脓肿 / 多重耐药)?
- 肝脓肿 → 影像随访 + 必要经皮引流 + 延长抗生素

## 🔺 升级标准

> [!danger]
> - Reynolds 五联(三联 + 休克 + 意识)→ ICU + 紧急 ERCP
> - TG18 Grade III(任一器官衰竭)→ ICU + 紧急 ERCP
> - 抗生素 + 引流后 24-48h 不退热 → 重新评估
> - 不能 ERCP / 失败 → PTCD / EUS / 外科会诊
> - 持续低血压 / 乳酸 > 4 → 1h bundle + ICU
> - 肝脓肿 / 反复菌血症 → 影像 + 介入引流
> - 任何"病人在恶化"

## ⚠️ 常见坑

> [!warning]
> 1. **Charcot 三联缺一就排除** → 仅 50-70% 病例三联齐全;老年 / 免疫抑制可仅意识改变 + 发热
> 2. **抗生素给了就放心** → **抗生素不能替代引流**,梗阻不解除 → 进展死亡
> 3. **引流延迟** → Grade III 应 < 24h,延迟死亡率显著上升
> 4. **超声胆道不扩张就排除** → 早期可不扩张,持续临床怀疑 → MRCP / CT
> 5. **没问 ERCP / 支架病史** → 院内菌可能耐药,经验性需调整
> 6. **不查血培养** → 病原识别后降阶梯失败
> 7. **过早改口服 / 停药** → 4-7 天最低;肝脓肿延长
> 8. **轻度 + 抗生素响应就不引流** → 仍可复发;胆石症需后续胆囊切除
> 9. **重症没准备 ERCP / 内镜不可用就只用抗生素** → 必须 PTCD / 外科备选,不能纯保守
> 10. **病史漏问肝吸虫(中国某地区)** → 流行地 + 反复发作 / 不典型梗阻
> 11. **胆管癌 / 胰头癌伪装"反复胆管炎"** → 老年慢性 + 体重下降 → 影像评估梗阻原因
> 12. **ERCP 后胰腺炎(PEP)漏诊** → 操作后腹痛 + 酶升 → 即时识别处理
> 13. **认为"老年 / 高危胆囊不切就完了"** → 仍可保守;权衡风险收益
> 14. **凝血障碍 / 抗凝下做 ERCP / PTCD** → 出血;需评估并矫正
> 15. **疾病早期肝功完全正常** → 仍可不排除;影像 + 临床进展性
