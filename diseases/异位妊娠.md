# 异位妊娠(Ectopic Pregnancy)

> [!summary] 30 秒摘要
> **红旗**:育龄女性 + 腹痛 + 阴道流血;**HCG 阳性 + 宫内未见妊娠 + 腹痛 = 异位直到证实排除**
> **首查**:**HCG 定量** + 经阴道超声 + CBC + 血型;不稳定 → 床旁超声(腹腔游离液 = 破裂)
> **一线**:稳定 → 甲氨蝶呤 / 输卵管手术(腹腔镜首选);**不稳定 / 破裂 → 立即手术**
> **升级**:任何不稳定 + HCG 阳性 → 妇产科立即会诊 + 手术室准备 + 备血

---

## 一句话定义

受精卵种植于子宫腔以外。

- **97% 输卵管**(壶腹部最多);其他:卵巢、宫颈、子宫角间质部、腹腔、剖宫产瘢痕
- **未破裂 vs 破裂**:破裂死亡率仍是孕期死亡 6-13%(早孕死亡首因)
- **正常妊娠 vs 异位**:**HCG 翻倍延迟 + 经阴超声宫内空** = 警觉

来源:ACOG Practice Bulletin Tubal Ectopic Pregnancy 2018;NICE NG126 Ectopic Pregnancy and Miscarriage 2019;中华医学会妇产科学分会异位妊娠诊治专家共识 2019(中华妇产科杂志)。

## 危险因素(增加警觉度,不能用于排除)

| 类别 | 具体 |
|---|---|
| **盆腔感染史 / PID** | 输卵管损伤 |
| **既往异位妊娠** | 复发率 10-25% |
| 输卵管手术 / 结扎 / 复通 | — |
| **辅助生殖(IVF)** | 同时宫内 + 异位(heterotopic)发生率 1/100 |
| 宫内节育器(IUD)失败妊娠 | IUD 下怀孕 → 异位概率显著升 |
| **吸烟** | 输卵管纤毛障碍 |
| 高龄(> 35) | — |
| 子宫内膜异位 | — |
| 既往剖宫产(瘢痕妊娠) | — |

> **20-50% 异位妊娠无任何高危因素** — 不能用"她没风险"来排除。

---

## 🚩 红旗(立刻警觉)

> [!danger] **任何育龄女性腹痛都要做 HCG**(说三遍)

- **HCG 阳性 + 单 / 双侧下腹痛**(锐 / 钝痛 / 撕裂)
- 阴道流血(常少量、暗红、间断)
- 晕厥 / 接近晕厥 + 育龄女性 → **警觉破裂**(腹腔出血)
- 肩痛(膈肌刺激,腹腔出血上行)
- 低血压 + HCG 阳性 → 立即外科 / 妇产科
- 已知有 IUD + 怀孕症状 + 腹痛
- **辅助生殖术后**孕妇腹痛(heterotopic 风险)
- 腹腔游离液 + HCG 阳性 → 破裂直到证实排除

## 鉴别诊断

| 疾病 | 区分点 | 关键检查 |
|------|--------|---------|
| **先兆 / 不全流产**(宫内) | HCG 阳性 + 阴道流血 + **超声宫内可见妊娠** | TVUS |
| 黄体破裂 / 卵巢囊肿破裂 | HCG 阴性 + 突发剧痛 | HCG + 超声 |
| **[[卵巢扭转]]** | 突发剧痛 + 卵巢肿大 + 多普勒血流消失 | 超声 |
| [[盆腔炎]](PID) | 双侧 + 发热 + 分泌物 + 宫颈摇摆痛 | 临床 + 实验 |
| **[[diseases/急性阑尾炎|急性阑尾炎]]** | 转移性右下腹 + 反跳痛 + WBC + CRP 升 | 临床 + 影像 |
| [[肾绞痛]] | 腰部 → 腹股沟、坐立不安 | 尿检 + US |
| 宫颈病变出血 | 仅出血无腹痛或轻痛 | 妇科检查 |
| **滋养细胞疾病(GTD)** | HCG 异常高 + 葡萄胎影像 | 超声 + HCG |
| 胃肠病(炎症性 / 缺血) | 胃肠症状为主 | 病史 + 影像 |

## 首查

### 立即(到院 30 min 内)

- [ ] 生命体征 + qSOFA / NEWS2
- [ ] 静脉通路 ×2(粗;不稳定备血)+ 心电监护
- [ ] **HCG 定量**(尿 HCG 阴性也不能排除——稀尿可假阴 / 极早孕;用血定量)
- [ ] CBC + 血型 + 交叉配血(立即备 4U,不稳定 6U)
- [ ] **经阴道超声(TVUS)**:
  - 子宫腔妊娠囊?(宫内 = 排除单纯异位但 IVF 后注意 heterotopic)
  - 附件包块?胎心?胚芽?
  - **直肠子宫陷凹游离液**(异位破裂典型)
- [ ] 经腹超声(无 TVUS / 不稳定 → 床旁 FAST)
- [ ] 凝血、肝肾功(MTX 决策需要)
- [ ] **抗 D 状态**(Rh- 妇女)— 决定抗 D 免疫球蛋白

### HCG + 超声决策(辨别阈值)

| 情境 | 处置 |
|---|---|
| **HCG > 1500-2000 mIU/mL + TVUS 宫内空** | 强烈提示异位(辨别阈值);进一步评估 |
| HCG < 1500 + 宫内空 + 稳定 | "未明确部位妊娠(PUL)"— 48h 复查 HCG;**正常宫内 48h ↑ ≥ 53%(NICE)** |
| **HCG 不当增长 / 平台 / 下降** | 异位或非存活妊娠 |
| 任何不稳定 + HCG 阳性 | **不等 HCG 复查,直接外科 / 介入** |

> 来源:ACOG 2018;NICE NG126 2019;具体阈值因实验室异;具体本院辨别阈值 [需查证]。

## 一线处理

### 0. 不稳定 / 破裂

> [!danger] **立即外科**

- 复苏:晶体 + 输血(限制性 Hb 7 / 大出血 MTP 1:1:1)
- 升压药维持 MAP(脓毒性休克思路)
- **腹腔镜手术**(首选)/ 开腹(不稳定不能延迟)
- 备血 4-6U
- 避免延迟,**等 HCG 报告无意义**

### 1. 期待治疗(选择性,极小心)

适应证:
- 稳定 + 无症状
- HCG **逐步下降** + < 200 mIU/mL
- 超声无显著包块
- 患者依从性好,能反复随访

**多数指南不推荐常规期待**;选择性病例妇产科决策。

### 2. 药物治疗 — 甲氨蝶呤(MTX)

| 适应证 | 标准 |
|---|---|
| **生命体征稳定** | ✓ 必须 |
| 无破裂征象(无腹腔游离液 / 无显著腹痛) | ✓ |
| HCG < 5000 mIU/mL(单剂方案,某些指南允许至 5000-10000) | 有些方案延伸 |
| 异位包块 < 3.5-4 cm,无胎心 | ✓ |
| 肝肾功能 / 血常规正常 | ✓ |
| 患者依从性好,可随访 | ✓ |
| 无 MTX 禁忌(免疫抑制、活动性肺疾病、肝肾不全、酗酒、活动性溃疡、母乳) | ✓ |

**剂量(单剂方案)**:**MTX 50 mg/m² IM 单次**
- D4 + D7 复测 HCG;**D4-D7 下降 ≥ 15% = 成功**;否则给第二剂
- 多剂方案:MTX 1 mg/kg D1, 3, 5, 7 + 亚叶酸 D2, 4, 6, 8(更高初始 HCG)

**MTX 后随访**:每周 HCG 至 < 5 mIU/mL(可需数周)。

> 来源:ACOG 2018;NICE NG126 2019;中国 2019 共识;具体本院 MTX 方案 [需查证]。

### 3. 手术治疗

| 方式 | 适应证 |
|---|---|
| **腹腔镜输卵管切除(salpingectomy)** | 输卵管损伤严重 / 反复异位 / 不打算保留生育 / 出血严重 |
| **腹腔镜输卵管造口(salpingostomy)** | 对侧输卵管异常 / 想保留生育 / 输卵管损伤可逆 |
| 开腹手术 | 不稳定 + 不能腹腔镜 / 解剖复杂 |
| 残留滋养组织 → 术后 MTX | 监测 HCG;持续不降需补充 |

### 4. 抗 D 免疫球蛋白(Rh- 妇女)

- 任何 Rh- 妇女 + 异位妊娠破裂 / 流产 / 妇产科介入 → **抗-D 免疫球蛋白 50-300 µg IM**
- 保护未来妊娠

来源:ACOG / NICE / 中国相关共识。

### 5. 后续随访

- 异位妊娠后**复发率 10-25%**;下次妊娠早期 HCG + 超声追踪
- 心理支持(流产 + 手术双重创伤)
- 不孕评估(若反复)

## 🔺 升级标准

> [!danger]
> - 任何不稳定 + HCG 阳性 → 妇产科立即会诊 + 手术准备 + 备血
> - 腹腔游离液 + HCG 阳性 → 破裂处理路径
> - 异位 + 大量阴道流血 → 警觉宫颈 / 子宫角破裂(出血迅猛)
> - 子宫角间质部异位 → 破裂出血严重,手术挑战
> - **剖宫产瘢痕妊娠**(CSP)→ 大出血风险,介入 + 妇产科多学科
> - 心理:任何自杀念头 / 家暴疑虑

## ⚠️ 常见坑

> [!warning]
> 1. **没问末次月经 / 没做 HCG** → 育龄女性腹痛漏诊 No.1,**说三遍**
> 2. **尿 HCG 阴性就排除** → 稀尿 / 极早孕假阴;用**血 HCG 定量**
> 3. **不稳定还在等 HCG 报告** → 临床高度怀疑直接手术
> 4. **TVUS 看到宫内胎囊就排除异位**——**辅助生殖患者 heterotopic 1/100**;有腹痛 + 腹腔积液仍要查
> 5. **HCG 单值正常 + 有症状不复查** → 异位 HCG 可低,需 48h 趋势
> 6. **MTX 给了就放心** → 必须随访 HCG 到 < 5;部分病人需手术
> 7. **MTX 用药期间未停 NSAID / 叶酸 / 酒精** → 增加毒性
> 8. **Rh- 妇女漏给抗-D** → 影响未来妊娠
> 9. **不查血型 + 交叉** → 大出血时延误
> 10. **异位破裂误为"卵巢囊肿破裂 / 黄体破裂"** → HCG 必查
> 11. **CSP / 间质部异位**视觉与"宫内妊娠"难分 → 有 CS 史 + 怀孕 → 警觉 CSP
> 12. **PID 误为异位 / 反向** → 体征 + HCG + 超声综合
> 13. **GTD 漏诊** → HCG 异常高 + "葡萄"样影像
> 14. **心理评估缺失** → 流产 + 紧急手术心理冲击大

## 关键评分(无标准评分,临床决策为主)

- HCG 趋势(48h ↑ ≥ 53% 提示宫内存活;< 53% 不能区分异位 / 早期失败妊娠)
- TVUS 妊娠囊辨别阈值(HCG ~ 1500-2000)
- 临床稳定性(生命体征、查体)
