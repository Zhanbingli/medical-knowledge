# 急性胰腺炎(Acute Pancreatitis)

> [!summary] 30 秒摘要
> **红旗**:上腹剧痛放射至背部 + 持续不缓解 + 恶心呕吐
> **首查**:**淀粉酶 / 脂肪酶**(≥ 3× 正常 = 诊断标准之一)+ 影像 + BISAP 评分
> **一线**:**积极液体复苏(乳酸林格 5-10 mL/kg/h)** + 镇痛 + 早期肠内营养;**早期不预防性抗生素**
> **升级**:器官衰竭 ≥ 48h(SAP)/ 局部并发症 / BISAP ≥ 3 → ICU + 多学科

---

## 一句话定义

胰酶在胰腺内自身激活 → 自我消化 → 局部 / 全身炎症反应。

**Atlanta 2012 修订诊断**(三选二):
1. 临床:典型上腹剧痛(常放射至背)
2. 实验室:**淀粉酶 / 脂肪酶 ≥ 3 × 正常上限**(脂肪酶更敏感、更长存)
3. 影像:CT / MRI / US 显示胰腺炎症征象

## 严重度分级(Atlanta 2012)

| 等级 | 标准 | 死亡率 |
|---|---|---|
| **轻度(Mild)** | 无器官衰竭 + 无局部 / 全身并发症 | < 1% |
| **中重(Moderately Severe)** | 一过性器官衰竭(< 48h)和 / 或局部并发症 | < 8% |
| **重度(Severe, SAP)** | **持续性器官衰竭 ≥ 48h**(呼吸 / 循环 / 肾) | 30-50% |

局部并发症:急性液体积聚、急性坏死性积聚、胰腺假性囊肿、包裹性坏死(WON)。

## 🚩 红旗(立刻警觉)

- 持续上腹剧痛(数小时-数天),**前倾位减轻、平卧加重**,常放射至背
- 恶心 / 呕吐(进食后加重)
- 腹胀、肠鸣音减弱(麻痹性肠梗阻)
- **Cullen 征**(脐周瘀斑)/ **Grey-Turner 征**(腰胁瘀斑)→ **重症胰腺炎 / 腹膜后出血**
- 黄疸(胆源性 / 头部肿瘤压迫)
- 心动过速、低血压(SIRS / 第三间隙液体丢失)
- 呼吸窘迫 / 低氧(ARDS)
- 少尿(AKI)
- 意识改变(中毒性 / 脓毒性)
- 高热(感染性坏死)

## 首查

### 立即(到院 1h 内)

- [ ] 生命体征 + qSOFA / NEWS2 + SpO2
- [ ] 静脉通路 ×2 + 心电监护 + 留置尿管(SAP 监尿量)
- [ ] **淀粉酶 + 脂肪酶**(脂肪酶更敏感更特异)
- [ ] CBC + 电解质 + 肌酐 + 钙 + 镁 + 白蛋白 + 凝血 + 乳酸
- [ ] **甘油三酯**(寻找病因 + 病情严重度;TG > 11.3 mmol/L 高 TG 性)
- [ ] **肝功**(ALT > 3× → 强烈提示胆源性)
- [ ] **CRP**(48h 后 > 150 提示 SAP)
- [ ] **血气**(pH / BE / 乳酸)
- [ ] **血糖**(应激性高血糖,警觉 DKA 鉴别)
- [ ] CXR(胸腔积液 - 多左侧、ARDS 早期)
- [ ] ECG + 肌钙蛋白(鉴别下壁 MI)
- [ ] 妊娠试验(育龄女性)
- [ ] **腹部超声**(胆囊结石 / 胆总管扩张 → 胆源性诊断)
- [ ] **腹部增强 CT(48-72h)** — 不在初诊立刻做(早期可低估)

### 评分(早期分层)

- [ ] **BISAP**(0-5,入院 24h 内,简单):BUN > 25, 意识改变, SIRS, 年龄 > 60, 胸腔积液;≥ 3 = 高危
- [ ] **Ranson**(入院 + 48h,11 项)
- [ ] **APACHE II**(ICU 标准,> 8 提示 SAP)
- [ ] **CTSI / 改良 CTSI**(48-72h CT 形态学)

## 一线处理

### 1. 液体复苏(核心!)

> [!danger] 第一个 24-48h 决定预后
> **乳酸林格(LR)优于生理盐水**(WATERFALL trial NEJM 2022);
> 起始 5-10 mL/kg/h(目标导向);
> 第 1-2 小时较积极(若无心 / 肾功能限制),之后**按容量反应性 + 尿量调整**(避免过度复苏致 ARDS / 腹腔间隔综合征)。

**目标**:
- 尿量 ≥ 0.5 mL/kg/h
- 红细胞压积下降趋势(基线高 → 下降提示扩容有效)
- BUN / 尿素改善
- 心率下降 / MAP 改善
- 乳酸清除

> 来源:WATERFALL trial NEJM 2022;ACG 2024;中国急性胰腺炎诊治指南 2021。

### 2. 镇痛

| 药 | 用法 | 备注 |
|---|---|---|
| **吗啡** | 2-4 mg IV q4h prn | **现代证据:不显著加重 Oddi 括约肌痉挛**;过去禁忌已弱化 |
| 氢吗啡酮 / 芬太尼 | 替代 | 半衰期短,SAP 偏好 |
| 哌替啶(度冷丁) | 25-50 mg IM/SC | 中国传统选择,但代谢物毒性,**长期 / 重症不首选** |

来源:ACG 2024;中国胰腺炎指南 2021。

### 3. 营养

- **轻度**:清流质 24-48h 内尝试(若耐受,**不需禁食 5-7 天**——已被 2024 ACG 等弱化)
- **中重 / 重度**:早期肠内(EN)< 24-72h(鼻空肠或鼻胃,等效)优于全肠外(PN)— 显著降低感染、死亡
- 严重不能耐受 → 全肠外补充

### 4. 抗生素

> [!warning] **不预防性给抗生素**
> ACG 2024 / 中国 2021 / IDSA — **预防性抗生素不能降低死亡率反增耐药 / 真菌**。

抗生素**仅当**:
- **感染性坏死**(CT 见气泡 / 抽吸培养阳性 / 临床不可解释的恶化)
- 胆管炎 / 胆道感染合并
- 院外其他明确感染

经验首选:碳青霉烯(美罗 / 厄他)/ 哌拉他唑 + 必要时抗真菌。

### 5. 病因导向

| 病因 | 处置 |
|---|---|
| **胆源性 + 胆管炎 / 胆总管梗阻** | **48-72h 内 ERCP 取石 + 鼻胆引流** |
| **胆源性轻度** | 同次住院或 2-4 周内胆囊切除 |
| 高 TG | 禁食 + 胰岛素 / 肝素降脂 + 必要时血浆置换(TG > 11.3 + 重症)|
| 高钙血症 | 治疗高钙(水化 + 利尿 + 双膦酸盐) |
| 药物性 | 停可疑药物 |
| 自身免疫(IgG4) | 类固醇 |

### 6. 并发症监测

- ARDS — NIV / 机械通气
- AKI — CRRT 必要
- 腹腔间隔综合征(IAP > 20)— 减压 / 外科
- 假性囊肿 / 包裹性坏死 — 影像随访;有症状 / 感染再干预
- 胰腺脓肿 / 感染性坏死 — 介入引流(经皮 / 内镜)→ 必要时外科

## 🔺 升级标准

> [!danger]
> - 持续器官衰竭 ≥ 48h(SAP)→ ICU
> - BISAP ≥ 3 / APACHE II > 8 / CRP 48h > 150 → 高危观察
> - 血流动力学不稳 / SBP < 90 / 乳酸 > 2 不降 → 1h bundle 思路
> - 呼吸窘迫 / SpO2 < 90% → 早期 NIV / 插管
> - 少尿 / 肌酐进行性升 / 高钾 → CRRT 评估
> - **胆源性 + 胆管炎征象** → 紧急 ERCP 通道
> - 感染性坏死征象(发热不退、CT 气泡)→ 外科 / 介入会诊
> - 任何"病人在恶化"

## ⚠️ 常见坑

> [!warning]
> 1. **早期就大剂量复苏 ≠ 越多越好** — WATERFALL trial 显示过度复苏反致 ARDS / 腹腔间隔综合征
> 2. **预防性抗生素** — 已被多个指南反对,仅感染性坏死才用
> 3. **酶高就诊断胰腺炎** — 穿孔 / 缺血 / 肾衰 / 巨淀粉酶血症都可升,需 ≥ 3× + 临床 + 影像
> 4. **CT 太早做** — 48-72h 前低估坏死范围;**急诊 CT 仅用于鉴别其他急腹症**
> 5. **胆源性 + 胆管炎不及时 ERCP** → 死亡率上升
> 6. **吗啡禁忌的旧观念** — 现代证据不支持,镇痛不充分增加痛苦 + 心血管应激
> 7. **DKA 表现胰腺炎** → 别开错诊;DKA 也可有胰腺酶轻升,需酮 / 血糖 / pH 综合判断
> 8. **下壁 MI 误为胰腺炎** → 中老年上腹痛 + 出汗,**ECG 永远先做**
> 9. **持续禁食 5-7 天** — 老观念,现代提倡早期肠内(< 72h)或耐受即恢复饮食
> 10. **一发现胰腺坏死就加抗生素** → 无菌性坏死不需要抗;感染征象才用
> 11. **胆囊不切等"以后" → 反复发作复发率高**;**轻症同次住院切除**优于推迟
> 12. **高 TG 性胰腺炎漏问 / 漏查** → 急性期取空腹 TG 或入院 TG
> 13. **儿童胰腺炎漏问外伤 / 解剖异常** — 不要套用成人病因
> 14. **慢性胰腺炎急性发作 / 假性囊肿破裂误为新发**
