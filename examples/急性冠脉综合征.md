# 急性冠脉综合征(ACS)

> [!summary] 30 秒摘要
> **红旗**:持续胸痛 + 出汗 + 放射 / ECG ST 改变 / 血流动力学不稳
> **首查**:**10 分钟内 12 导联 ECG** + 高敏肌钙蛋白(0h+1h)
> **一线**:阿司匹林 300mg 嚼服 + P2Y12 + 抗凝 + 再灌注决策
> **升级**:任何 ST 抬高 / 不稳定 / VT/VF / 胸痛持续不缓解 → 立刻打电话 + 启动导管室

---

## 一句话定义

冠脉血流急性减少导致心肌缺血/坏死的临床综合征。按 ECG 和肌钙蛋白分为:
- **STEMI**:ST 段抬高型心肌梗死(完全闭塞,需立即再灌注)
- **NSTEMI**:非 ST 段抬高型心肌梗死(肌钙蛋白阳性 + 无 ST 抬高)
- **UA(不稳定型心绞痛)**:症状符合但肌钙蛋白阴性

---

## 🚩 红旗(立刻警觉)

- 持续 > 20 min 胸痛 / 压榨样 / 濒死感
- 放射至左臂、下颌、背部、上腹
- 大汗、恶心、呼吸困难伴发
- 低血压(SBP < 90)/ 心动过速 / 心动过缓 + 低血压(下壁 / RV 梗死)
- 新发心衰体征(肺水肿、奔马律、湿啰音)
- ECG 任何 ST 抬高或新发 LBBB
- **⚠️ 老年 / 糖尿病 / 女性常无典型胸痛**——可表现为乏力、气短、上腹不适、晕厥

---

## 鉴别诊断(必须排除的杀手)

| 疾病 | 区分点 | 关键检查 |
|------|--------|---------|
| [[主动脉夹层]] | 撕裂痛、向背部放射、双上肢血压差 > 20、胸片纵隔增宽 | CTA |
| [[肺栓塞]] | 突发气短、低氧、单侧腿肿、Wells 评分高 | D-dimer + CTPA |
| [[张力性气胸]] | 单侧呼吸音消失、气管偏移、突发剧痛 | 床旁查体即可,**别等 CXR** |
| [[心包炎]] | 体位相关、前倾减轻、弥漫凹面向上 ST 抬高、PR 压低 | ECG + 心包摩擦音 |
| [[心包填塞]] | 三联征:低血压 + 颈静脉怒张 + 心音遥远;奇脉 | 床旁超声 |
| [[食管破裂]] | 剧吐后剧痛 + 皮下气肿(Mackler 三联征) | CT / 食管造影 |
| 急性胃炎 / GERD | 进食相关、与体位有关、肌钙蛋白阴性 | — |

## 首查(到床后 10 分钟内)

- [ ] **12 导联 ECG**(<10min,门口立刻做)
  - 怀疑下壁:加做 V3R、V4R(RV 梗死)
  - 怀疑后壁:加做 V7-V9
- [ ] **高敏肌钙蛋白 hs-cTn**(0h + 1h 或 3h 系列)
- [ ] CBC、电解质、肌酐、凝血、血糖、血脂
- [ ] CXR(排除夹层、气胸、肺水肿)
- [ ] 床旁超声(若怀疑夹层 / RV / 心包填塞)
- [ ] 静脉通路 ×2、心电监护、血氧

## 一线处理

> 现代版,弃用 MONA-BASH 口诀(吗啡和氧疗的指征已收紧)

### 共同处理(所有 ACS)

1. **抗血小板双联(DAPT)**
   - 阿司匹林 300 mg 嚼服首剂,后 75-100 mg/d
   - P2Y12 抑制剂(选一):
     - 替格瑞洛 180 mg 负荷 → 90 mg bid(首选,STEMI 和 NSTEMI 均可)
     - 氯吡格雷 300-600 mg 负荷 → 75 mg/d(高出血风险或不能用替格瑞洛)
2. **抗凝**(根据再灌注策略)
   - PCI 路线:普通肝素 70-100 U/kg(或比伐卢定)
   - 溶栓 / 保守:依诺肝素 1 mg/kg sc q12h
3. **镇痛**:吗啡 2-4 mg IV(**仅在剧痛且其他无效时**,2023 ESC 弱化吗啡)
4. **氧疗**:**仅当 SpO2 < 90%**,常规给氧已不推荐
5. **硝酸甘油**:舌下 0.5 mg,SBP > 90 且**排除 RV 梗死、24h 内服西地那非**
6. **β受体阻滞剂**:24h 内启动,排除心衰急性期、低血压、严重心动过缓

### 再灌注决策

| 类型 | 决策 |
|------|------|
| **STEMI** | PCI < 90 min(首诊到导丝);若 > 120 min 不能 → 溶栓 < 30 min |
| **NSTEMI 极高危** | 立即 PCI(< 2 h):血流动力学不稳 / 持续胸痛 / 恶性心律失常 / 机械并发症 |
| **NSTEMI 高危** | 早期 PCI < 24 h(GRACE > 140 / 肌钙蛋白动态变化 / ST-T 动态改变) |
| **NSTEMI 中危** | 选择性介入 < 72 h |

## 🔺 升级标准(立刻叫上级)

> [!danger]
> - ECG 任何 ST 抬高 / 新发 LBBB / 后壁导联抬高
> - 血流动力学不稳:SBP < 90 / 急性肺水肿 / 心源性休克
> - 恶性心律失常:VT / VF / 完全 AVB / 反复非持续 VT
> - 持续胸痛 ≥ 20 min 不缓解
> - 肌钙蛋白进行性升高
> - 机械并发症:新发杂音(VSR / MR)、心包摩擦音
> - **任何"病人不对劲"的直觉**

---

## ⚠️ 常见坑

> [!warning]
> 1. **老年女性 / 糖尿病无典型胸痛** → 主诉是"恶心""乏力""上腹不适"。任何中老年新发不适,先做 ECG。
> 2. **肌钙蛋白单次阴性不能排除 ACS** → 必须 0h + 1h 或 0h + 3h 系列(高敏)。
> 3. **后壁 STEMI 在标准 12 导联可能只表现为 V1-V3 ST 压低** → 别漏,加做 V7-V9。
> 4. **下壁 STEMI 给硝酸甘油可能血压崩** → 必须先排除 RV 梗死(V3R-V4R 抬高 + 低血压 + 颈静脉怒张三联征)。
> 5. **西地那非 24h 内禁用硝酸甘油** → 给药前一定问。
> 6. **左主干病变 ECG 可能表现为 aVR ST 抬高 + 广泛 ST 压低** → 别当成"全导联缺血"按 NSTEMI 处理,这是极高危。
> 7. **胸痛缓解 ≠ 没事** → 可能是侧支开放或暂时再通,仍可能再闭塞。
> 8. **"年轻人不会得心梗"是错的** → 可卡因、自发夹层(SCAD,孕产妇)、川崎病后遗症、家族性高胆固醇都会。

## 关键评分

- **[[HEART]]**:急诊室快速分诊,5 项简单。≤3 分低危可考虑出院。
- **[[GRACE]]**:NSTEMI 危险分层,**决定 PCI 时机**(>140 = 24h 内)。
- **[[TIMI]]**:简单易记但精度低于 GRACE。
- **[[CRUSADE]]**:出血风险,决定抗凝策略。
- **[[Killip 分级]]**:急性心衰严重程度(I-IV 级,IV = 心源性休克)。
