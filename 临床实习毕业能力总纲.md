# 临床实习毕业能力总纲

> [!summary] 核心目标
> 毕业目标不是“知道所有疾病”，而是达到安全的低年资医生能力:识别危险、启动基础处理、清楚沟通、正确记录、及时升级。

## 使用方式

- **先学症状**:从主诉进入，训练“危险病因优先排除”的临床思维。
- **再学急症**:掌握必须立即识别、处理、升级的 must-not-miss 条目。
- **随后学系统疾病**:按心血管、呼吸、消化、肾脏、内分泌等系统补齐常见病。
- **并行学药物、检查、沟通和安全习惯**:这些决定临床执行是否安全。
- **每篇具体疾病/症状/药物笔记**:必须按 `AGENTS.md` 模板写，数值、剂量、阈值必须有指南来源或标记 `[需查证]`。

## 1. 核心临床思维

临床推理的基本顺序:

```text
主诉 → 病史 → 聚焦查体 → 问题列表 → 鉴别诊断 → 初始检查 → 初始处理 → 复评
```

每个常见症状都要能回答:

| 问题 | 临床意义 |
|---|---|
| 有哪些致命原因? | 防止漏诊 must-not-miss |
| 有哪些常见原因? | 提高日常处理效率 |
| 哪些病史问题能区分它们? | 避免泛泛问诊 |
| 哪些体征最关键? | 把查体变成决策工具 |
| 首批检查是什么? | 先做能改变处理的检查 |
| 哪些治疗不能延误? | 识别时间敏感疾病 |
| 何时叫上级? | 把不确定性及时转化为团队决策 |

- 神经: [[symptoms/头痛|头痛]] · [[symptoms/头晕|头晕]] · [[symptoms/意识改变|意识改变]] · [[抽搐]] · [[肢体无力-麻痹]]

## 2. 急诊医学:必须不漏的病

实习医生的急诊能力不是“独立完成全部治疗”，而是:

- 识别患者是否不稳定。
- 立刻做 ABCDE 评估。
- 启动基础处理和首批检查。
- 用结构化语言呼叫上级。
- 记录时间线、关键体征、处理和复评结果。

### 心血管急症

- [[diseases/急性冠脉综合征|急性冠脉综合征]]:胸痛、心电图、肌钙蛋白、抗栓风险、导管室/上级升级。
- [[急性心力衰竭]] / [[emergencies/急性肺水肿|急性肺水肿]]:呼吸窘迫、低氧、湿啰音、血压分型。
- 心律失常:[[diseases/心房颤动|心房颤动]] · [[室上性心动过速]] · [[室性心动过速]] · [[心室颤动]] · [[缓慢性心律失常]]。
- [[高血压急症]]:判断是否有靶器官损害，而不是只看血压数字。
- [[emergencies/主动脉夹层|主动脉夹层]]:突发撕裂样胸背痛、脉搏/血压差、神经或灌注异常。
- [[emergencies/肺栓塞|肺栓塞]]:胸痛、呼吸困难、低氧、DVT 线索、风险分层。
- [[diseases/休克|休克]]:低血容量性、感染性、心源性、梗阻性、过敏性。

### 呼吸急症

- [[diseases/急性呼吸衰竭|急性呼吸衰竭]]:先看氧合、通气、意识和疲劳。
- [[哮喘急性发作]] / [[COPD急性加重]]:呼吸功增加、沉默胸、CO2 潴留风险。
- [[肺炎]] / [[重症肺炎]]:感染灶、氧合、严重程度、抗菌药时机。
- [[气胸]] / [[emergencies/张力性气胸|张力性气胸]]:胸痛、呼吸困难、单侧呼吸音减弱、循环不稳。
- [[大咯血]]:先保护气道和健侧肺，再找出血来源。
- [[肺水肿]]:区分心源性、容量负荷、ARDS 等病因。

### 神经急症

- [[emergencies/急性缺血性脑卒中|急性缺血性脑卒中]]:最后正常时间、神经定位、血糖、影像和再灌注路径。
- [[diseases/脑出血|脑出血]] / [[emergencies/蛛网膜下腔出血|蛛网膜下腔出血]]:头痛、呕吐、意识、抗凝、血压和神经外科升级。
- [[癫痫持续状态]]:先保护气道、测血糖、止惊、找诱因。
- [[diseases/细菌性脑膜炎|细菌性脑膜炎]] / [[脑炎]]:发热、头痛、颈强直、意识改变，不能延误抗感染和升级。
- [[颅内压增高]]:头痛、呕吐、视乳头水肿、意识改变、脑疝风险。
- [[急性脊髓压迫]]:背痛、无力、感觉平面、尿潴留/失禁。

### 腹部和外科急症

- [[diseases/急性阑尾炎|急性阑尾炎]] · [[肠梗阻]] · [[消化道穿孔]] · [[diseases/急性胰腺炎|急性胰腺炎]]
- [[急性胆囊炎]] · [[diseases/急性胆管炎|急性胆管炎]] · [[diseases/消化道出血|消化道出血]]
- [[异位妊娠]] · [[卵巢扭转]] · [[睾丸扭转]] · [[急性尿潴留]]

### 感染急症

- [[emergencies/脓毒症|脓毒症]] / [[感染性休克]]:识别、早期评估、初始治疗、升级、感染源控制和监测是关键 (NICE Sepsis, 2025)。
- [[重症肺炎]] · [[diseases/细菌性脑膜炎|细菌性脑膜炎]] · [[急性肾盂肾炎]] · [[坏死性软组织感染]]

## 4. 外科知识

外科实习重点是识别外科急症、准备患者、减少围手术期风险:

- 术前评估:基础病、抗凝、过敏、禁食、麻醉风险、感染风险。
- 术后发热:时间线、肺部、尿路、伤口、血栓、药物。
- 伤口感染和手术部位感染:红肿热痛、渗液、全身反应。
- 液体管理:容量状态、尿量、肾功能、电解质。
- 创伤初评:气道、呼吸、循环、神经、暴露。
- 急腹症:[[diseases/急性阑尾炎|急性阑尾炎]] · [[急性胆囊炎]] · [[diseases/急性胰腺炎|急性胰腺炎]] · [[肠梗阻]] · [[嵌顿疝]]
- 基础外伤:[[烧伤]] · [[骨折]] · [[开放性骨折]]

## 5. 妇产科

必须先识别高风险主诉:

- 妊娠 + 腹痛 → [[异位妊娠]]、胎盘问题、外科急腹症。
- 妊娠 + 阴道流血 → [[流产]]、[[前置胎盘]]、[[胎盘早剥]]。
- 妊娠 + 高血压 → [[子痫前期]] / [[子痫]]。
- 产后大量出血 → [[产后出血]]。
- 产后发热 → 子宫内膜炎、乳腺炎、泌尿感染、伤口感染。

基础主题:

- 正常产检、[[妊娠期糖尿病]]、胎膜早破、盆腔炎、卵巢扭转、子宫肌瘤、宫颈癌筛查、避孕基础。

## 6. 儿科

儿科最重要习惯:不要照搬成人剂量。

- [[新生儿黄疸]] · [[新生儿感染]] · [[儿童发热]]
- 儿童肺炎、腹泻脱水、[[热性惊厥]]、儿童哮喘
- 先天性心脏病基础、生长发育里程碑、疫苗接种、儿童液体和药物剂量计算

## 7. 神经系统能力

- 卒中定位和急诊路径。
- 癫痫与癫痫持续状态。
- 脑膜炎、脑炎、帕金森病、周围神经病、重症肌无力、GBS。
- 头痛红旗、外周/中枢性眩晕鉴别。
- 查体技能:颅神经、运动、感觉、反射、小脑、脑膜刺激征、NIHSS 基础。

## 8. 骨科

- 骨折分类基础、开放/闭合骨折、神经血管评估。
- [[骨筋膜室综合征]]:疼痛不成比例、麻木、无力、脉搏异常。
- 关节脱位、[[化脓性关节炎]]、骨髓炎、腰背痛红旗、骨质疏松、术后并发症。

## 9. 感染病和抗菌药

实习医生必须会问:

- 感染源在哪里?
- 需不需要先取培养?
- 抗菌药是否必须立即给?
- 是社区获得还是医院获得?
- 是否有耐药风险、过敏史、肾功能问题?
- 什么时候需要降阶梯?

常见感染:

- [[肺炎]] · [[尿路感染]] · [[急性肾盂肾炎]] · [[蜂窝织炎]]
- [[diseases/细菌性脑膜炎|细菌性脑膜炎]] · [[emergencies/脓毒症|脓毒症]] · [[结核病]] · [[乙型肝炎]] · [[丙型肝炎]] · [[HIV感染]]

## 11. 检验和影像解释

### 血液检查

- 血常规、CRP/PCT、肝功能、肾功能、电解质、凝血、D-dimer、肌钙蛋白、BNP/NT-proBNP、血糖、HbA1c、甲功、动脉血气。

### 尿/便/体液

- 尿常规、尿培养、便潜血、脑脊液基础、胸水基础、腹水基础。

### 影像

- 胸片:肺炎、气胸、胸腔积液、肺水肿。
- 腹片:肠梗阻、穿孔线索。
- 急诊头颅 CT:出血、占位效应、脑疝线索。
- CT 胸/腹:知道要问放射科什么问题。
- 超声:胆石、肾积水、腹水、胸水、DVT 筛查概念。

## 12. 操作和床旁技能

- 静脉采血、留置针、动脉血气、血培养、导尿、胃管、换药、缝合基础。
- ECG 采集、氧疗、雾化、CPR/BLS/ACLS 原则。
- 无菌技术、手卫生、PPE。

## 13. 病历书写和沟通

必须会写:

- 入院记录、病程记录、出院小结、操作记录、会诊申请、交班记录、处方、死亡记录基础。

必须会说清楚:

- 诊断不确定性、治疗计划、风险获益、知情同意、坏消息基础、转诊理由、升级给上级、和家属沟通。

SBAR 升级模板:

```text
S: 我是谁,患者是谁,现在最担心什么
B: 关键背景:年龄/主要病史/入院原因/目前诊断
A: 当前评估:生命体征、关键检查、已处理、为什么不放心
R: 我需要你做什么:现在来看/电话指导/是否转 ICU/是否启动通道
```

## 14. 临床安全习惯

- 核对身份、过敏、妊娠可能。
- 肾毒性药物或增强 CT 前查肾功能。
- 操作前查抗凝和凝血风险。
- 给药前查剂量、适应证、禁忌证和相互作用。
- 治疗后复评并记录。
- 病情不稳定早升级。
- 不隐藏错误。
- 不确定就明确写“不确定”，并说明下一步验证计划。

- [ ] 用 SBAR 呼叫上级。
- [ ] 用患者能懂的话解释病情。
- [ ] 避免不安全处方。
- [ ] 承认不知道，并写出验证计划。

## 18. 心态

> [!note]
> 不要追求“知道所有疾病”。要追求“安全处理常见问题，迅速识别危险问题，在伤害发生前升级”。
