# 晕厥

> [!summary] 接到这个主诉,3 件事必做
> 1. 先确认是否真正短暂意识丧失:快速起病、短暂、可自行完全恢复
> 2. 立即查生命体征、卧立位血压、床旁血糖、12 导联 [[ECG]]
> 3. 任何胸痛、心悸、劳力中发作、异常 [[ECG]]、低血压、神经缺损或外伤 → 先按高危晕厥升级

## 🚨 必须排除的 Killer

> [!danger] [[心律失常]] / 心源性晕厥
> 晕厥发生在运动中或平卧位、晕厥前有心悸、存在结构性心脏病、家族猝死史或 [[ECG]] 异常时，应优先考虑 [[心律失常]] 或结构性心脏病导致的心源性晕厥。初始评估必须包含详细病史、体格检查和静息 12 导联 [[ECG]] (ACC/AHA/HRS Syncope Guideline, 2017; 晕厥诊断与治疗中国专家共识, 2018)。

> [!danger] [[diseases/急性冠脉综合征|急性冠脉综合征]]
> 晕厥合并胸痛、压榨感、出汗、呼吸困难、缺血性 [[ECG]] 改变或肌钙蛋白升高时，不能按“低血压晕倒”处理；需按 [[diseases/急性冠脉综合征|急性冠脉综合征]] 路径评估和升级。

> [!danger] [[emergencies/肺栓塞|肺栓塞]]
> 晕厥合并突发呼吸困难、胸痛、低氧、咯血、DVT 线索、近期手术/制动或恶性肿瘤风险时，需评估 [[emergencies/肺栓塞|肺栓塞]]。晕厥可能是血流动力学受影响的表现，不能被单纯解释为焦虑或迷走反射。

> [!danger] [[emergencies/主动脉夹层|主动脉夹层]]
> 晕厥合并突发胸背痛、撕裂样疼痛、神经症状、肢体缺血、脉搏差或血压差时，需立即考虑 [[emergencies/主动脉夹层|主动脉夹层]] 并升级。

> [!danger] [[emergencies/蛛网膜下腔出血|蛛网膜下腔出血]] / [[diseases/脑出血|脑出血]]
> 晕厥伴突发严重头痛、呕吐、颈强直、神经缺损、抗凝用药或头部外伤时，需排除 [[emergencies/蛛网膜下腔出血|蛛网膜下腔出血]]、[[diseases/脑出血|脑出血]] 和创伤性颅内出血。

## 常见非危急原因

- [[血管迷走性晕厥]]:站立、疼痛、情绪刺激、医疗操作、闷热环境后出现恶心、出汗、面色苍白、视物变暗等前驱症状；诊断仍需先排除高危特征。
- [[体位性低血压]]:从卧位/坐位起立后出现黑蒙、头晕或晕厥；体位性低血压可用卧立位血压评估，收缩压下降 ≥20 mmHg 或舒张压下降 ≥10 mmHg 支持诊断 (NICE TLoC CG109, 2023)。
- [[emergencies/低血糖|低血糖]]:糖尿病、进食少、饮酒、使用胰岛素或促泌剂者优先排除；意识改变或出汗心悸时先测床旁血糖。
- [[癫痫]]:舌咬伤、发作后意识混乱、持续抽搐、尿失禁或目击者描述提示时考虑；需与晕厥后短暂抽动区分。
- [[精神性非癫痫发作]] / 功能性发作:反复发作、持续时间长、生命体征稳定、发作形式不符合晕厥或癫痫时考虑；诊断前先排除高危心脑血管病因。
- 药物或容量不足:降压药、利尿剂、血管扩张剂、酒精、呕吐腹泻、出血或脱水可诱发晕厥。

## 决策树

```text
疑似晕厥 / 黑蒙 / 倒地
├── 是否真正短暂意识丧失并完全恢复?
│   ├── 否 → 考虑跌倒、眩晕、癫痫、低血糖、精神性发作
│   └── 是
├── 立即是否不稳定:低血压、胸痛、气促、低氧、持续心悸、神经缺损、严重外伤?
│   ├── 是 → ABCDE + 监护 + ECG + 血糖 + 叫上级
│   └── 否
├── 有无心源性红旗:运动中/平卧发作、无前驱、结构性心脏病、家族猝死、异常 ECG?
│   ├── 是 → 高危晕厥:心电监护 + 心内/急诊上级评估
│   └── 否
├── 是否起立相关?
│   ├── 是 → 卧立位血压 → [[体位性低血压]]
│   └── 否
├── 是否有典型迷走诱因和前驱?
│   ├── 是 → [[血管迷走性晕厥]] 倾向,仍需复核 ECG 和外伤风险
│   └── 否 → 考虑 [[癫痫]]、[[emergencies/低血糖|低血糖]]、[[emergencies/肺栓塞|肺栓塞]]、出血、药物和其他病因
```

## 必问病史

| 模块 | 必问内容 | 红旗问题 |
|---|---|---|
| O 起病 | 发作前在做什么、站立/坐位/平卧/运动中、是否突然 | 运动中或平卧位晕厥 → 心源性风险 |
| P 诱因 | 疼痛、情绪、抽血、排尿、咳嗽、吞咽、久站、闷热、起立 | 特定诱因可提示反射性或情境性晕厥 |
| Q 先兆 | 心悸、胸痛、气促、恶心、出汗、黑蒙、耳鸣 | 无前驱 + 突然倒地 → 心律失常风险 |
| R 过程 | 意识丧失持续时间、抽动、目击者描述、肤色、呼吸、受伤 | 长时间意识不清或发作后混乱 → 非单纯晕厥 |
| S 恢复 | 是否自行完全恢复、恢复后有无神经缺损、头痛、胸痛 | 恢复后仍异常需按神经/心肺急症处理 |
| T 频率 | 首次或复发、近期次数、是否越来越频繁 | 复发且无解释需专科评估 |
| 既往史 | 结构性心脏病、心衰、冠心病、先天性心脏病、癫痫、糖尿病 | 心脏病史显著提高高危概率 |
| 家族史 | 年轻猝死、遗传性心律失常、心肌病 | 家族猝死是心源性风险线索 |
| 用药 | 降压药、利尿剂、硝酸酯、抗心律失常药、QT 延长药、降糖药、抗凝药 | 药物性低血压、低血糖、出血和心律失常风险 |

## 必查体征

- 生命体征:血压、心率、呼吸频率、SpO2、体温；记录是否持续异常。
- 卧立位血压:疑似体位性低血压时测量；卧位到站立后血压下降阈值见 NICE TLoC CG109 2023。
- 心血管:心律、杂音、心衰体征、外周灌注、双上肢血压/脉搏差。
- 神经:意识、瞳孔、言语、眼球运动、面瘫、肌力、感觉、共济、脑膜刺激征。
- 外伤:头颅、颈椎、口腔舌咬伤、肢体骨折、抗凝相关出血风险。
- 容量状态:皮肤黏膜、毛细血管再充盈、尿量线索、出血或脱水体征。

## 床旁首查

- [ ] 12 导联 [[ECG]]:所有疑似晕厥患者初始评估均应做 (ACC/AHA/HRS Syncope Guideline, 2017; 晕厥诊断与治疗中国专家共识, 2018)。
- [ ] 床旁血糖:排除 [[emergencies/低血糖|低血糖]]。
- [ ] 卧立位血压:怀疑 [[体位性低血压]] 或起立相关发作者。
- [ ] 心电监护:高危特征或怀疑 [[心律失常]] 者。
- [ ] 妊娠检测:育龄女性腹痛、阴道流血或休克表现时，排除 [[异位妊娠]]。
- [ ] 血常规、电解质、肾功能、肌钙蛋白、D-dimer、血气/乳酸:根据胸痛、气促、出血、感染、脱水或 PE 风险选择；不要无差别“全套检查”替代病史和 ECG。
- [ ] 头颅影像:仅在头外伤、持续神经异常、严重头痛、抗凝或怀疑颅内病变时优先；单纯典型反射性晕厥不应依赖头颅 CT 作为常规首查。

## 🔺 立刻升级的临床表现

- 晕厥发生在运动中或平卧位。
- 晕厥前胸痛、持续心悸、呼吸困难或突发严重头痛。
- [[ECG]] 异常、持续心动过缓/过速、宽 QRS 心动过速或疑似传导阻滞。
- 既往结构性心脏病、心衰、冠心病、先天性心脏病或家族猝死史。
- 低血压、低氧、休克体征或不能快速恢复到基线意识。
- 局灶神经体征、颈强直、头部外伤或抗凝用药。
- 妊娠可能 + 腹痛/阴道流血/休克。
- 反复发作、无明确诱因、造成严重外伤或在高风险场景发生。

## ⚠️ 常见漏诊场景

- 把所有“晕倒”都当 [[血管迷走性晕厥]]，没有做 [[ECG]]。
- 只听患者描述，没问目击者，导致 [[癫痫]]、跌倒、精神性发作和真正晕厥混淆。
- 平卧或运动中晕厥被误认为“低血糖/贫血”，漏掉 [[心律失常]]。
- 晕厥后头痛或外伤没有复查神经系统，漏掉 [[emergencies/蛛网膜下腔出血|蛛网膜下腔出血]] 或颅内出血。
- 老年人晕厥只归因于“体虚”，未复核用药、心律、主动脉瓣狭窄和出血。
- 胸痛/气促伴晕厥没有评估 [[emergencies/肺栓塞|肺栓塞]]、[[diseases/急性冠脉综合征|急性冠脉综合征]]、[[emergencies/主动脉夹层|主动脉夹层]]。
- 头颅 CT 正常就认为安全，忽略心源性晕厥风险。
- 未记录发作前体位、诱因、先兆、恢复时间和目击者描述，导致后续无法分层。
