# 头晕

> [!summary] 接到这个主诉,3 件事必做
> 1. 先分清是持续性眩晕、发作性眩晕、晕厥前兆还是步态不稳
> 2. 立即查血糖、生命体征、卧立位血压、眼震、步态和局灶神经体征
> 3. 任何新发神经缺损、不能站走、垂直/方向改变眼震或新发听力下降 → 先按 [[后循环卒中]] 排除

## 🚨 必须排除的 Killer

> [!danger] [[后循环卒中]] / [[短暂性脑缺血发作]]
> 突发头晕伴复视、构音障碍、吞咽困难、肢体无力/麻木、共济失调、不能独立站走、新发耳聋或中枢性眼震时，需立即按 [[后循环卒中]] 或 [[短暂性脑缺血发作]] 处理。NICE 建议成人突发头晕合并局灶神经缺损、垂直或旋转眼震、新发不稳或新发耳聋时，在 [[良性阵发性位置性眩晕]] 和 [[体位性低血压]] 不能解释的情况下立即转诊排除后循环卒中 (NICE Suspected neurological conditions, 2019)。

> [!danger] [[emergencies/低血糖|低血糖]]
> 糖尿病、进食差、出汗、心悸、意识改变或使用降糖药者，头晕先测床旁血糖；NICE 对突发头晕合并局灶神经表现者明确要求糖尿病患者先检查并处理低血糖 (NICE Suspected neurological conditions, 2019)。

> [!danger] [[心律失常]] / 心源性晕厥前兆
> 头晕若表现为眼前发黑、快要晕倒、胸痛、心悸、劳力诱发或伴低血压，应按 [[symptoms/晕厥|晕厥]] / [[心律失常]] 路径评估，而不是先归因于“耳石症”。需床旁心电图和持续监测；具体心源性晕厥风险分层需按本院急诊流程核对。

> [!danger] [[diseases/脑出血|脑出血]] / 颅内占位
> 头晕伴突发剧烈头痛、呕吐、意识改变、抗凝用药、近期头外伤或局灶神经体征时，需按中枢神经急症处理。中国急性缺血性卒中指南强调疑似卒中早期需影像排除出血并快速评估再灌注适应证 (中国急性缺血性卒中诊治指南 2023)。

> [!danger] [[一氧化碳中毒]]
> 头晕伴头痛、恶心、乏力、多人同场所发病、燃气/取暖暴露或冬季密闭环境暴露时，需立即考虑 [[一氧化碳中毒]]；需查碳氧血红蛋白和给予氧疗，具体阈值与流程需查证本院急诊规范。

## 常见非危急原因

- [[良性阵发性位置性眩晕]]:体位改变诱发短暂旋转感，Dix-Hallpike 或滚转试验诱发特征性眼震；符合典型表现时优先复位而不是常规头颅影像 (GRACE-3, 2023; 中国 BPPV 诊治指南 2017)。
- [[前庭神经炎]]:急性持续性眩晕、恶心呕吐和不稳，可有方向固定外周性眼震；诊断前提是没有急性中枢神经或急性听觉症状/体征 (Bárány Society AUVP criteria, 2022; 前庭神经炎诊治多学科专家共识 2023)。
- [[前庭性偏头痛]]:反复眩晕或头晕伴偏头痛病史、畏光、畏声或视觉先兆；需排除 [[短暂性脑缺血发作]] 和耳科疾病。
- [[体位性低血压]]:起立后头晕、眼前发黑、乏力或晕厥前感，需卧立位血压和用药复核。
- [[精神性头晕]] / [[持续性姿势-感知性头晕]]:慢性或反复头晕，常受焦虑、环境视觉刺激、站立/行走影响；诊断前必须先排除中枢、心源性和前庭器质性病因 (精神性头晕诊疗中国专家共识 2024)。
- [[贫血]] / [[脱水]] / 电解质紊乱:乏力、心悸、体位相关或近期失血、腹泻、呕吐、进食差时考虑。

| P 诱因 | 起立、翻身、转头、咳嗽、运动、劳力、进食少 | 劳力诱发或伴胸痛/心悸 → [[心律失常]] |

| S 严重度 | 能否坐稳、站立、独立行走、是否跌倒 | 不能独立站走需升级 |

| 用药/暴露 | 降压药、降糖药、镇静药、抗癫痫药、氨基糖苷、酒精、CO 暴露 | 药物和中毒可模拟前庭/中枢头晕 |

## 必查体征

- 生命体征:血压、心率、SpO2、体温；加做卧立位血压。
- 床旁血糖:糖尿病、意识改变、出汗心悸或进食差者立即查。
- 神经查体:眼位、眼震、凝视诱发眼震、复视、构音、吞咽、面瘫、肢体肌力/感觉、指鼻试验、跟膝胫试验。
- 步态和平衡:能否坐稳、站立、直线行走；严重躯干共济失调是中枢警讯。
- 眼耳查体:听力粗测、耳痛/耳流脓、耳疱疹、耳鸣。
- 心血管:心律、杂音、颈动脉杂音、容量状态。
- HINTS:仅限受训者且只用于急性持续性眩晕伴眼震的场景；不适用于短暂位置性发作或无眼震患者 (GRACE-3, 2023)。

## 床旁首查

- [ ] 血糖:先排除 [[emergencies/低血糖|低血糖]]。
- [ ] ECG:心悸、胸痛、晕厥前感、老年或心血管病史者优先。
- [ ] 卧立位血压:起立相关头晕或降压药/利尿剂使用者。
- [ ] NIHSS / FAST 或中风 120:任何疑似 [[卒中]] 表现立即启动卒中流程 (中国急性缺血性卒中诊治指南 2023)。
- [ ] Dix-Hallpike / 滚转试验:短暂位置诱发眩晕且无中枢红旗时。
- [ ] HINTS + 听力粗测:仅由受训者用于急性持续性眩晕伴眼震；中枢或不确定结果需 MRI/MRA 确认 (GRACE-3, 2023)。
- [ ] CBC、电解质、肾功能:贫血、脱水、呕吐、腹泻、用药风险或全身病表现时。
- [ ] 头颅 CT / MRI / CTA / MRA:有卒中红旗、头痛/外伤/抗凝、HINTS 中枢或不确定、疑似 TIA 时按卒中路径选择。
- [ ] 碳氧血红蛋白:多人同场所头晕、燃气/取暖暴露或密闭空间暴露。

## 🔺 立刻升级的临床表现

- 新发局灶神经体征、复视、构音障碍、吞咽困难、单侧肢体无力/麻木。
- 不能独立站立或行走，或明显躯干共济失调。
- 垂直眼震、方向改变眼震、凝视诱发眼震、明显 skew deviation。
- 急性持续性眩晕合并新发听力下降。
- 头晕伴胸痛、持续心悸、低血压、晕厥或严重气促。
- 头晕伴突发严重头痛、意识改变、抗凝用药或近期头外伤。
- 多人同环境出现头晕、头痛、恶心或意识改变，警惕 [[一氧化碳中毒]]。

## ⚠️ 常见漏诊场景

- 把所有“天旋地转”都当 [[良性阵发性位置性眩晕]]，没有查步态、眼震方向和局灶体征。
- 只问“晕不晕”，不问时间模式和诱因，导致 [[短暂性脑缺血发作]] 与位置性眩晕混淆。
- 对急性持续性眩晕做了错误场景的 HINTS；HINTS 不是无眼震、短暂发作或未受训者的筛查工具 (GRACE-3, 2023)。
- 头颅 CT 阴性就排除 [[后循环卒中]]；GRACE-3 不建议用 CT 作为急性眩晕排除中枢病因的一线工具 (GRACE-3, 2023)。
- 新发耳聋 + 眩晕被当作单纯耳科病，漏掉 AICA 区域缺血。
- “年轻人焦虑”被过早诊断为 [[精神性头晕]]，未先排除低血糖、心律失常、前庭病和中枢病因。
- 未查用药:降压药、镇静催眠药、抗癫痫药、酒精和耳毒性药物可导致头晕或步态不稳。
- 老年人主诉“头昏”“走不稳”但无旋转感，仍可能是 [[后循环卒中]]。
