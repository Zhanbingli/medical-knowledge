# 腹痛

> [!summary] 接到这个主诉,3 件事必做
> 1. **先排除 10 个 Killer**——其中血管 / 穿孔 / 扭转有时间窗,不是慢慢分诊
> 2. 育龄女性 → **末次月经 + 妊娠试验是问诊与化验的强制项**(漏问就要漏诊宫外孕)
> 3. 老年腹痛 → 默认严重性升级一档,即使生命体征正常

---

## 一句话定义

腹腔内 / 腹膜后 / 邻近器官(胸、盆、泌尿生殖)病变引起的腹部疼痛或不适。
- **急腹症**:急性起病、需在数小时内决策"是否手术"的腹痛
- **病理生理三类**:内脏痛(模糊、定位差)、体性痛(尖锐、定位准、腹膜受累)、牵涉痛(膈下→肩、胆囊→右肩胛、心肌→上腹)

---

## 必须排除的 Killer

> [!danger] [[腹主动脉瘤破裂]](rAAA)
> 经典三联:**老年男性 + 突发腰背 / 腹痛 + 低血压**。
> - 搏动性腹部包块 + 双侧下肢脉差 = 强烈提示
> - **不稳定 → 直接送手术 / 介入,不要等 CT**
> - 稳定者床旁超声测主动脉直径(> 5.0 cm 高度警觉)→ 增强 CT
> - 漏诊为"肾绞痛"是经典误诊
> 
> 来源:ESVS Abdominal Aorto-iliac Artery Aneurysms Guideline(具体版本年份请核实);中华医学会血管外科学分会主动脉瘤相关共识 <!-- 待具体年份与杂志补充 -->

> [!danger] [[肠系膜缺血]](急性 AMI)
> **"腹痛与查体不成比例"** 是经典提示:患者主诉剧痛但腹部相对软。
> - 房颤 / 严重动脉粥样硬化 / 心衰 / 低灌注 / 高凝状态者高发
> - **乳酸进行性升高**(早期可不高)
> - **腹部增强 CT(动脉期)** 是金标准,不是常规腹部 CT
> - 延误数小时即坏死、穿孔、休克、死亡
> 
> 来源:WSES Acute Mesenteric Ischemia Guideline 2017;中国急性肠系膜血管缺血性疾病诊治专家共识 <!-- 待具体年份与杂志补充 -->

> [!danger] [[消化道穿孔]](Perforated viscus)
> **板状腹 + 全腹反跳 + 突发剧痛**。
> - 立位胸片或腹片可见膈下游离气体(敏感性约 75-80%,**正常不能排除**)
> - **腹部 CT** 敏感性更高(微小气体 / 积液也能看到)
> - 危险因素:NSAIDs、激素、消化道溃疡、肿瘤、憩室、内镜操作后
> - 立即外科会诊 + 备血 + 禁食 + 广谱抗生素覆盖革兰阴性 + 厌氧
> 
> 来源:WSES Complicated Intra-abdominal Infections Guideline 2017;中国急性腹腔感染诊治指南 <!-- 待具体年份与杂志补充 -->

> [!danger] [[异位妊娠破裂]](Ruptured ectopic pregnancy)
> **任何育龄女性腹痛 → 先做妊娠试验,任何育龄女性腹痛 → 先做妊娠试验,任何育龄女性腹痛 → 先做妊娠试验**(说三遍)。
> - HCG 阳性 + 腹痛 + 阴道流血 / 晕厥 = 直接妇产科会诊
> - 经阴道超声看不到宫内妊娠 + HCG 高于辨别阈 → 强烈提示
> - 不稳定 → 立即手术,不等 HCG 定量
> 
> 来源:NICE Ectopic Pregnancy and Miscarriage 指南 <!-- 待 NG 编号核实 -->;中国早期妊娠诊治流程 / 中华妇产科学会异位妊娠相关共识 <!-- 待具体年份与杂志补充 -->

> [!danger] [[卵巢扭转]] / [[睾丸扭转]]
> **疼痛黄金 6 小时窗,超过显著降低存活 / 保留率**。
> - 卵巢扭转:育龄女性突发下腹剧痛 + 恶心呕吐 + 卵巢肿大 / 囊肿史
> - 睾丸扭转:青少年男性突发阴囊剧痛 + 提睾反射消失 + 睾丸抬高 / 横位
> - **多普勒超声**(血流消失提示扭转,但阴性不能排除)
> - **临床高度怀疑直接手术,不应等影像**
> 
> 来源:AUA Acute Scrotum / Testicular Torsion Guideline <!-- 待具体年份核实 -->;ESHRE Adnexal Torsion 共识 <!-- 待核实 -->;中国小儿外科 / 泌尿外科相关共识 <!-- 待具体年份与杂志补充 -->

> [!danger] [[重症急性胰腺炎]](Severe acute pancreatitis, SAP)
> 上腹剧痛放射至背部 + 淀粉酶 / 脂肪酶 ≥ 3× 正常上限 + 影像支持(Atlanta 2012 修订)。
> - 早期评估:BISAP / APACHE II / Ranson;48h 内 CT 严重度评分(CTSI)
> - 重症标志:器官衰竭(呼吸、循环、肾)+ 局部并发症(坏死、感染、囊肿)
> - **24-48h 内积极液体复苏**(乳酸林格,目标尿量 0.5-1 mL/kg/h)
> - 不应预防性抗生素(轻 / 中度);感染性坏死再用
> 
> 来源:Atlanta Classification 修订版 2012;ACG Acute Pancreatitis Guideline 2024 修订;中国急性胰腺炎诊治指南 2021(中华医学会外科学分会 / 中华胰腺病杂志)

> [!danger] [[diseases/急性胆管炎|急性胆管炎]](Acute cholangitis)
> **Charcot 三联征**:发热 + 黄疸 + 右上腹痛(发生率约 50-70%)。
> **Reynolds 五联征**:三联征 + 休克 + 意识改变 = 重症胆管炎,死亡率显著升高。
> - **抗生素 + 紧急胆道引流(ERCP / PTCD)** 是核心
> - 不稳定不应延迟引流等血培养结果
> - 老年 / 免疫抑制可表现不典型
> 
> 来源:Tokyo Guidelines TG18 Acute Cholangitis;中国急性胆道系统感染诊治指南 2021(中华急诊医学杂志 / 中国医师协会急诊医师分会)

> [!danger] [[绞窄性肠梗阻]] / 肠扭转
> 单纯梗阻 → **绞窄**(血供受损)= 肠坏死 → 穿孔 → 脓毒症。
> - 警觉信号:持续剧痛(梗阻本身多为阵发)、压痛局限、发热、心率持续升、白细胞 / 乳酸升、腹腔游离液
> - **腹部 CT 增强** 是首选影像(平片仅限筛查)
> - 立即外科会诊 + 禁食 + 胃肠减压 + 液体复苏
> 
> 来源:WSES Bowel Obstruction Guideline <!-- 具体年份待核实 -->;中国成人急性肠梗阻诊治指南 <!-- 待具体年份与杂志补充 -->

> [!danger] [[下壁心肌梗死]](表现为腹痛)
> **老年人 + 糖尿病人** 上腹痛伴出汗 / 恶心 / 呼吸困难——可能是下壁 MI 牵涉痛。
> - **任何中老年新发上腹痛 → 12 导联 ECG**(成本几乎为零)
> - 加做 V3R-V4R(右室梗死)和 V7-V9(后壁)
> - 误判为"胃炎"给质子泵抑制剂错过再灌注窗
> 
> 来源:ESC ACS Guideline 2023;中国急性 ST 段抬高型心肌梗死诊断和治疗指南 2019(中华心血管病杂志)

## 常见非危急原因

- [[diseases/急性阑尾炎|急性阑尾炎]]:转移性右下腹痛、发热、Alvarado / AIR 评分
- [[急性胆囊炎]]:右上腹痛 + Murphy 征 + 超声胆囊壁厚 / 周围积液
- [[diseases/急性胰腺炎|急性胰腺炎]](轻症):上腹痛放射至背 + 酶升高
- [[肾绞痛]]:腰部 → 腹股沟放射,坐立不安(区别腹膜炎的不动)
- [[胃肠炎]]:腹痛 + 腹泻 / 呕吐;排除外科急腹症
- [[消化性溃疡]]:慢性 / 进食相关上腹痛
- [[盆腔炎]](PID):下腹痛 + 阴道分泌物 + 宫颈摇摆痛
- [[肠易激综合征]] / [[功能性腹痛]]:**排除性诊断**,绝不是首诊

## 必查体征

> [!tip] 顺序:**视 → 听 → 叩 → 触**(听诊在触诊前,避免改变肠鸣音)

- **生命体征 + qSOFA / NEWS2**:不稳定立即升级
- **一眼判断**:面色、出汗、能否平卧、是否扭曲身体
  - 腹膜炎 → 不动(怕震)
  - 肾绞痛 → 坐立不安
- **视诊**:腹胀、瘢痕、疝、皮疹、瘀斑(Cullen / Grey-Turner = 重症胰腺炎 / 腹膜后出血)
- **听诊**:亢进(梗阻早期)/ 减弱或消失(腹膜炎 / 麻痹性肠梗阻)/ 血管杂音(AAA / 夹层)
- **叩诊**:鼓音(气体)/ 浊音(包块 / 积液 / 膀胱)/ 移动性浊音(腹水)/ 肝浊音界消失(气腹)
- **触诊**:轻 → 重,从远处向痛处移动
  - 反跳痛 / 板状腹 = 腹膜炎
  - Murphy 征(胆囊炎)
  - Rovsing(右下腹痛被左下腹按压触发——阑尾炎)
  - Psoas / Obturator(后位 / 盆位阑尾)
  - Carnett 征(腹壁源 vs 腹腔源)
- **直肠指检**:盆腔脓肿、消化道出血、肿块、宫颈摇摆痛
- **盆腔检查**:育龄女性 PID / 异位妊娠
- **腹股沟 / 阴囊**:疝、睾丸扭转
- **双下肢动脉 + 主动脉搏动**:AAA / 夹层
- **心肺**:下壁 MI、心衰、肺炎牵涉

## 床旁首查

- [ ] **生命体征 + qSOFA / NEWS2 + 乳酸**
- [ ] **指尖血糖 + 12 导联 ECG**(中老年上腹痛排 MI)
- [ ] **HCG**(育龄女性强制项)
- [ ] 血常规、CRP、电解质、肝肾功、淀粉酶、脂肪酶、凝血、血型
- [ ] 乳酸:疑似肠系膜缺血 / 脓毒症必查;> 2 警觉,> 4 立即升级
- [ ] 肌钙蛋白:中老年上腹痛 / 心血管病人
- [ ] 尿常规 + 尿淀粉酶:肾结石 / 泌尿感染 / 胰腺受累
- [ ] **立位胸片 / 腹片**:气腹 / 梗阻气液平(敏感性有限)
- [ ] **床旁超声(POCUS)**:
  - **FAST**(腹腔游离液)
  - **AAA**(主动脉直径)
  - 胆囊(结石、壁厚、Murphy)
  - 肾(积水)
  - 妊娠子宫 / 异位妊娠
- [ ] **腹部 CT**(常规增强;**疑肠系膜缺血用 CTA 动脉期**;疑 AAA 用主动脉 CTA)
- [ ] 血培养:发热 / 脓毒症 / 胆管炎,抗生素前
- [ ] 妊娠保护:育龄女性影像决策权衡(超声 → MRI > CT)

## 🔺 立刻升级的临床表现

- 板状腹 / 全腹反跳 / 腹腔游离液(FAST 阳性)
- 进行性低血压 / 休克
- **乳酸 > 4 mmol/L** 或 > 2 持续不能纠正
- 搏动性腹部包块 + 低血压(警觉 [[腹主动脉瘤破裂]])
- "**腹痛与查体不成比例**"(警觉 [[肠系膜缺血]])
- 腹痛 + 育龄女性 + HCG 阳性 + 不稳定(警觉 [[异位妊娠破裂]])
- 阴囊剧痛 < 6h 或卵巢肿大 + 急性下腹痛(警觉 [[睾丸扭转]] / [[卵巢扭转]],时间窗)
- 腹痛 + 黄疸 + 发热 + 低血压(警觉重症 [[diseases/急性胆管炎|急性胆管炎]],Reynolds)
- 老年腹痛 + 房颤 / 心血管病(警觉 [[肠系膜缺血]])
- 中老年上腹痛 + 出汗 / 恶心 / 心律改变(警觉 [[下壁心肌梗死]])
- 妊娠 + 腹痛 + 阴道流血 / 子宫张力增高(警觉胎盘早剥)
- 肠梗阻 + 持续剧痛 + 局限压痛(警觉 [[绞窄性肠梗阻]])

## ⚠️ 常见漏诊场景

- **没问末次月经 / 没做 HCG** → 漏诊宫外孕(育龄女性腹痛 No.1 漏诊)
- **老年腹痛"看起来不像急腹症"** → 体征可不典型,腹膜刺激征常缺如,先按严重处理
- **"血常规正常 + 体温正常" 排除外科急腹症** → 早期可正常,反复评估比一次化验重要
- **肠系膜缺血"症状重 - 体征轻"被忽视** → 晚期才出现腹膜征,那时已坏死
- **AAA 误诊为肾绞痛** → 老年 + 腰背痛 + 低血压三联,先想 AAA
- **下壁 MI 当胃炎** → 中老年上腹痛 + 出汗 / 恶心,**ECG 几乎零成本**
- **"假愈期"陷阱** → 缺血或穿孔后疼痛短暂缓解(神经麻痹 / 压力释放),被当成好转
- **睾丸 / 卵巢扭转等待影像** → 高度怀疑直接手术,影像阴性不能排除
- **小肠梗阻 KUB 阴性继续观察** → 腹部 CT 才是金标准
- **NSAIDs / 激素史漏问** → 漏掉穿孔背景
- **给阿片止痛后症状缓解就放走** → 阿片不影响诊断性查体可给,但**疼痛缓解 ≠ 病好**
- **抗凝 / 抗血小板治疗中突发腹痛** → 警觉自发腹腔出血、肠系膜栓塞
- **妊娠期阑尾位置上移**(右上腹甚至右肋下)→ 不典型容易漏
