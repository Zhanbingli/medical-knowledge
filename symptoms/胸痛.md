# 胸痛

> [!summary] 接到胸痛主诉,3 件事必做
> 1. **生命体征**(SBP / HR / RR / SpO2)— 任一异常 → 立即抢救通道
> 2. **10 分钟内 12 导联 ECG**(无论年龄、症状是否典型)
> 3. **排除 6 个 Killer**(下方红色框)再考虑常见原因

---

## 🚨 必须排除的 Killer(漏一个就死人)

> [!danger] 胸痛 6 大杀手 · 优先用床旁手段排除
> 1. **[[急性冠脉综合征]]** — ECG + hs-cTn(0h+1h)
> 2. **[[主动脉夹层]]** — 双上肢血压差 > 20 / 撕裂痛 / 纵隔增宽 → CTA
> 3. **[[肺栓塞]]** — Wells 评分 / D-dimer / CTPA
> 4. **[[张力性气胸]]** — 单侧呼吸音消失 / 气管偏移 / 颈静脉怒张 → **不等 CXR,直接针刺减压**
> 5. **[[心包填塞]]** — Beck 三联征:低血压 + 颈静脉怒张 + 心音遥远 → 床旁超声
> 6. **[[食管破裂]]** — Mackler 三联征:剧烈呕吐 + 胸痛 + 皮下气肿 → CT

## 常见非危急原因

- [[胃食管反流]] / 食管痉挛

- [[胆囊炎]] / [[胰腺炎]](放射至胸)

- 焦虑 / 惊恐(**排除性诊断,不是首诊**)

## 决策树(关键分叉问题)

```
1. 生命体征稳定?
   ├── 否 → 抢救通道,同时做 ECG + 床旁超声
   └── 是 → 继续

2. ECG 有 ST 改变 / 新发 LBBB?
   ├── 是 → [[急性冠脉综合征|ACS]] 通道
   └── 否 → 继续

3. 撕裂样痛 + 双上肢血压差?
   ├── 是 → 怀疑 [[主动脉夹层]] → CTA
   └── 否 → 继续

4. 单侧呼吸音消失 / 气管偏移?
   ├── 是 → 怀疑 [[气胸]],张力性立刻减压
   └── 否 → 继续

5. 气短为主 + 单侧腿肿 / 危险因素?
   ├── 是 → Wells 评分 → D-dimer / CTPA
   └── 否 → 继续

6. hs-cTn 0h+1h/3h 阴性 + 上述全排除?
   └── 考虑非危急鉴别(GERD / 肌骨 / 焦虑)
```

## 必问病史(OPQRST + 红旗)

| 项 | 问什么 | 暗示 |
|---|---|---|
| **O**nset | 突发(秒)vs 渐进(分钟-小时) | 突发 → 夹层 / PE / 气胸 |
| **P**rovoke / Palliate | 用力诱发?休息缓解?进食加重?体位变化? | 劳累 → 心绞痛;前倾 → 心包炎;进食 → GERD |
| **Q**uality | 压榨 / 撕裂 / 烧灼 / 锐痛 | 撕裂 → 夹层;压榨 → ACS |
| **R**adiation | 左臂 / 下颌 / 背部 / 上腹 | 下颌 / 左臂 → ACS;背部 → 夹层 |
| **S**everity | 0-10 分;最严重的一次? | 任何"我从未这么痛过"必须警觉 |
| **T**iming | 持续多久?> 20min? | > 20min → ACS 风险升 |

**必问的红旗问题**:
- 有没有放射至**下颌或左臂**?
- 有没有**呼吸困难**伴发?
- 有没有**晕厥 / 接近晕厥**?
- 有没有**大汗 / 濒死感**?
- **危险因素**:糖尿病、高血压、吸烟、家族史、口服避孕 / 制动史(PE)
- **年龄 / 性别**(老年女性糖尿病常无典型表现)

---

## 必查体征(高 yield)

> [!tip] 不要做完整查体再处理。先做这 6 项,1 分钟内完成
> 1. **生命体征 + 双上肢血压**(差 > 20 → 夹层)
> 2. **颈静脉**(怒张 → 心包填塞 / RV 衰竭 / 张力性气胸)
> 3. **气管位置**(偏移 → 张力性气胸)
> 4. **双肺呼吸音**(单侧消失 → 气胸 / 胸腔积液)
> 5. **心音**(摩擦音 → 心包炎;遥远 → 填塞;新杂音 → ACS 并发症)
> 6. **下肢**(单侧肿 + 压痛 → DVT → 警惕 PE)

---

## 床旁首查(10 分钟内)

- [ ] **12 导联 ECG**(下壁可疑加 V3R-V4R,后壁加 V7-V9)
- [ ] **hs-cTn**(0h + 1h 系列)
- [ ] **CXR**(气胸、纵隔、肺水肿、骨折)
- [ ] **静脉通路 + 心电监护 + SpO2**
- [ ] **D-dimer**(若 Wells / Geneva 评分中-高度怀疑 PE)
- [ ] **床旁超声(POCUS)**:疑夹层 / 填塞 / RV 衰竭时
- [ ] **ABG**(低氧 / 不稳定时)

---

## 🔺 立刻升级的临床表现

> [!danger]
> - SBP < 90 或较基线下降 > 40
> - SpO2 < 90% 即使吸氧
> - 任何 ECG ST 改变 / 新发 LBBB / 严重心律失常
> - 单侧呼吸音消失(警惕张力性气胸)
> - Beck 三联征(警惕心包填塞)
> - 双上肢血压差 > 20 + 撕裂痛
> - 持续胸痛 ≥ 20min
> - 晕厥伴发
> - 任何"病人不对劲"的直觉

## ⚠️ 常见漏诊场景

> [!warning]
> 1. **老年女性 / 糖尿病不典型 ACS** → 主诉是"恶心""乏力""上腹不适"。中老年新发不适 = 先 ECG。
> 2. **后壁 STEMI** 在标准 12 导联仅表现为 V1-V3 ST 压低 → 加做 V7-V9。
> 3. **主动脉夹层误诊为 ACS** → 给抗凝 / 溶栓 = 灾难。任何 ECG 不典型 + 撕裂痛 + 高血压病史 → 想夹层。
> 4. **D-dimer 阴性不能排除高危 PE**(Wells 高度怀疑时直接 CTPA)。
> 5. **早期带状疱疹** 皮疹出现前可纯胸痛 → 单侧、皮节分布、烧灼感。
> 6. **冠脉痉挛 / SCAD(自发夹层)** → 年轻人、孕产妇、可卡因史 → ECG 可正常。
> 7. **食管破裂** 剧烈呕吐后剧痛常被当 ACS → 听皮下气肿。
> 8. **"年轻人不会得心梗"是错的** → 可卡因、SCAD、川崎后遗症、家族性高胆固醇都会。
