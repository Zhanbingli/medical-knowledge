# 呕血-黑便

> [!summary] 接到这个主诉,3 件事必做
> 1. 先判断是否休克/活动性出血:气道、意识、血压、心率、皮肤灌注、尿量
> 2. 立刻建立静脉通路、抽血交叉配血、停抗栓/NSAIDs、通知上级和消化内镜团队
> 3. 有肝硬化/门脉高压线索时，先按 [[食管胃静脉曲张破裂出血]] 处理，不能等胃镜才启动路径

## 🚨 必须排除的 Killer

> [!danger] [[上消化道大出血]] / 失血性休克
> 呕血、咖啡渣样呕吐、黑便或暗红/鲜红便伴低血压、心动过速、晕厥、意识改变、少尿或皮肤湿冷时，先按失血性休克处理。ACG 建议上消化道出血住院患者采取限制性输血策略，Hb 阈值 7 g/dL (ACG Upper GI and Ulcer Bleeding, 2021)；具体输血需结合休克、活动性出血和心血管病。

> [!danger] [[食管胃静脉曲张破裂出血]]
> 肝硬化、门静脉高压、腹水、脾大、黄疸、既往静脉曲张或大量呕血者，优先考虑 [[食管胃静脉曲张破裂出血]]。Baveno VII 建议疑似静脉曲张出血时尽早启动血管活性药物并持续 2–5 天，Hb 目标 7–8 g/dL；意识改变或活动性呕血者内镜前考虑气管插管 (Baveno VII, 2022)。中国指南同样强调肝硬化门静脉高压食管胃静脉曲张出血的早期识别和内镜/综合治疗 (中华医学会肝硬化门静脉高压食管胃静脉曲张出血指南, 2022)。

> [!danger] [[消化性溃疡出血]]
> 黑便、呕血合并 NSAIDs、阿司匹林、抗凝药、既往溃疡或幽门螺杆菌风险时，需考虑 [[消化性溃疡出血]]。ACG 建议上消化道出血患者入院后 24 小时内行内镜检查 (ACG Upper GI and Ulcer Bleeding, 2021)。

> [!danger] [[主动脉肠瘘]]
> 既往主动脉瘤、主动脉支架/手术、腹部搏动性包块、腹痛/背痛伴消化道出血时，需考虑 [[主动脉肠瘘]]。若出现“先少量出血后大出血”模式，应立即叫上级并联系血管外科/介入。

> [!danger] [[下消化道大出血]]
> 鲜红或暗红便伴休克可来自下消化道，也可来自快速上消化道出血；不能只凭颜色定位。大量便血、持续血流动力学不稳或抗凝背景需同步考虑下消化道出血和上消化道快速出血。

## 常见非危急原因

- [[消化性溃疡]]:黑便或呕血，常有上腹痛、NSAIDs/阿司匹林/抗凝药、幽门螺杆菌或既往溃疡线索。
- [[急性胃黏膜病变]]:重症应激、酒精、NSAIDs、肾衰、肝衰或 ICU 背景；需同时评估凝血和血小板。
- [[Mallory-Weiss综合征]]:剧烈呕吐后呕血；诊断前仍需评估出血量和休克。
- [[食管炎]] / [[胃炎]]:少量呕血或咖啡渣样物，仍需排除高危用药和肿瘤。
- [[胃癌]] / [[食管癌]]:贫血、消瘦、进行性吞咽困难、黑便或隐匿出血；急性出血稳定后需内镜明确。
- 食物/药物假黑便:铁剂、铋剂、动物血制品可使大便变黑；但有头晕、乏力、Hb 下降或粪便潜血阳性时不能归因于食物。

```text
│   ├── 是 → 侧卧/吸引/备插管 + 叫上级 + 内镜/麻醉/ICU
│   └── 否

│   ├── 是 → ABCDE + 可靠静脉通路 + 交叉配血 + 快速复评 + 上级
│   └── 否
├── 是否肝硬化/门脉高压/腹水/既往静脉曲张?

│   └── 否
├── 是否 NSAIDs/抗血小板/抗凝/既往溃疡?

```

## 必查体征

- ABCDE:气道保护能力、呼吸、循环、意识、暴露查体。
- 生命体征:血压、心率、呼吸频率、SpO2、体温；反复测量并看趋势。
- 灌注:皮肤湿冷、毛细血管再充盈、尿量、意识、四肢温度。
- 腹部:压痛、反跳痛、肝脾大、腹水、腹壁静脉、手术瘢痕、搏动性包块。
- 肝硬化体征:黄疸、蜘蛛痣、肝掌、腹水、下肢水肿、意识改变。
- 直肠指检:确认黑便/血便，评估粪便性状；不可替代病情分层。
- 皮肤黏膜:瘀斑、出血点、贫血貌，提示凝血/血小板问题。

## 床旁首查

- [ ] 评估气道风险并备吸引；活动性呕血或意识改变时叫上级评估插管。
- [ ] 建立可靠静脉通路或中心静脉通路；抽血交叉配血。
- [ ] CBC、凝血、肝肾功能、电解质、乳酸/血气、血型和交叉配血。
- [ ] ECG 和肌钙蛋白:老年、胸痛、休克或冠心病风险者，出血可诱发心肌缺血。
- [ ] Glasgow-Blatchford Score:ACG 建议 GBS 0–1 的极低风险上消化道出血患者可考虑门诊随访而非住院 (ACG Upper GI and Ulcer Bleeding, 2021)。
- [ ] 肝硬化/静脉曲张线索阳性:启动静脉曲张出血路径，联系消化内镜，使用血管活性药和抗菌药需按本院规范。
- [ ] 抗凝/抗血小板用药:记录最后服药时间，叫上级评估停药、逆转和血栓风险。
- [ ] 内镜:ACG 建议上消化道出血住院患者入院后 24 小时内内镜检查 (ACG Upper GI and Ulcer Bleeding, 2021)；休克未纠正时先复苏和气道保护。
- [ ] CTA / 介入 / 外科:持续大出血、内镜不可及或疑似 [[主动脉肠瘘]] / 下消化道大出血时由上级决定。

## 🔺 立刻升级的临床表现

- 低血压、休克、晕厥、意识改变、少尿、皮肤湿冷。
- 持续呕血、喷射样呕血、反复暗红/鲜红便或 Hb 快速下降。
- 活动性呕血伴误吸风险或不能保护气道。
- 肝硬化/门脉高压患者出现呕血或黑便。
- 抗凝、双联抗血小板、血小板减少或凝血异常合并出血。
- 胸痛、气短、缺血性 ECG 改变或高龄心血管病患者出血。
- 主动脉瘤/支架/手术史合并腹痛、背痛或消化道出血。
- 妊娠、儿童、重症基础病或无法获得可靠病史者。

## ⚠️ 常见漏诊场景

- 黑便被解释为“吃了铁剂/动物血”，没有查生命体征、Hb 和粪便潜血。
- 只看入院 Hb 正常就放心；急性大出血早期 Hb 可滞后，需看休克和动态变化。
- 肝硬化患者呕血未按静脉曲张出血路径处理，延误血管活性药、抗菌药和内镜。
- 只补液不叫内镜/上级，导致止血时机延误。
- 忽略抗凝/抗血小板和 NSAIDs，未记录最后服药时间。
- 活动性呕血患者平卧转运或检查，误吸风险未处理。
- 鲜红便只当下消化道出血，漏掉快速上消化道大出血。
- 主动脉支架/手术史患者少量消化道出血未警惕 [[主动脉肠瘘]]。
