# 气短(Dyspnea)

> [!summary] 接到气短主诉,3 件事必做
> 1. **生命体征 + SpO2**:RR > 30 或 < 8 / SpO2 < 90% / 三脚架 / 辅助肌 / 发绀 → 抢救通道
> 2. **快速排除 7 个 Killer**(下方红色框)
> 3. **床旁:ECG + 听诊 + POCUS** 三件套,诊断率 80%

---

## 🚨 必须排除的 Killer

> [!danger] 气短 7 大杀手
> 1. **[[上气道梗阻]]**(异物 / 喉头水肿 / 会厌炎)— 听**喘鸣 stridor**,立刻气道
> 2. **[[过敏反应]]**(anaphylaxis)— 荨麻疹 + 喉头水肿,**肾上腺素 IM 0.3-0.5 mg**
> 3. **[[张力性气胸]]** — 单侧呼吸音消失 + 气管偏移 → **针刺减压,不等 CXR**
> 4. **[[严重哮喘发作]]** — 沉默胸 = 极危 → 雾化 + 类固醇 + 准备插管
> 5. **[[急性肺水肿]]** — 端坐呼吸 + 双肺湿啰音 + 粉红泡沫痰 → 利尿 + 扩血管 + NIV
> 6. **[[肺栓塞]]** — 突发 + 心动过速 + 低氧 + 单侧腿肿
> 7. **[[急性冠脉综合征]]**(气短可为唯一表现,尤其老年女性糖尿病)— ECG + hs-cTn

---

## 常见非危急原因

- [[肺炎]] / [[COPD急性加重]] / [[哮喘]] 一般发作
- [[心力衰竭]] 慢性失代偿
- [[胸腔积液]]
- [[贫血]]
- 焦虑 / 过度通气(**排除性诊断**,绝不是首诊)

---

## 决策树(听诊定位 + 床旁超声)

```
1. 喘鸣(stridor)?
   └─ 是 → 上气道梗阻 / anaphylaxis → 立刻气道,肾上腺素

2. 哮鸣(wheeze)?
   ├─ 弥漫双侧:哮喘 / COPD / 心源性 / anaphylaxis
   └─ 局限:异物 / 肿瘤压迫

3. 双侧湿啰音?
   ├─ 重力依赖性(底):肺水肿(CHF)
   └─ 弥漫:ARDS / 严重肺炎

4. 单侧呼吸音消失?
   ├─ 叩诊鼓音 + 气管偏移 → 张力性气胸(立刻减压)
   ├─ 叩诊浊音 → 大量胸腔积液 / 肺不张
   └─ 局部捻发音 + 实变 → 肺炎

5. 听诊清晰 + 心动过速 + 低氧?
   └─ → 肺栓塞 / 心内分流 / 早期 ARDS

6. 气短 + 端坐呼吸 + JVP 升 + S3 + 下肢肿?
   └─ → 急性心衰

7. 气短 + 心绞痛特征(放射、出汗)?
   └─ → ACS(气短可为唯一表现)
```

## 必查体征(高 yield 60 秒)

> [!tip] 别做完整查体再处理
> 1. **生命体征 + SpO2 + 呼吸节律和深度**
> 2. **能讲完整句子吗?**(讲不完 = 重度)
> 3. **辅助肌 / 三脚架 / 鼻翼煽动 / 发绀 / 反常呼吸**
> 4. **气管位置**(偏移 → 张力性气胸)
> 5. **听诊**(哮鸣 / 湿啰音 / 单侧消失 / 喘鸣)
> 6. **JVP / 心音(S3 / 杂音)**
> 7. **下肢:对称肿(CHF) / 单侧肿(DVT)**
> 8. **皮肤**:荨麻疹(过敏)、紫绀

---

## 床旁首查(10 分钟内)

- [ ] **SpO2 + ABG**(低氧 / 高碳酸 / 酸碱)
- [ ] **12 导联 ECG**(ACS、PE 的 RV 应变、心律失常)
- [ ] **CXR**(气胸、肺水肿、肺炎、心影、纵隔)
- [ ] **POCUS**:
  - 肺(B 线 = 肺水肿;sliding 消失 = 气胸)
  - 心(EF、RV 大小、心包积液、IVC)
- [ ] **BNP / NT-proBNP**(<100 强烈反对 CHF;>500 强烈支持)
- [ ] **hs-cTn**(ACS 排除)
- [ ] **D-dimer**(PE 评估后)
- [ ] **CBC**(贫血、感染)
- [ ] 静脉通路 ×2、心电监护

## 🔺 立刻升级的临床表现

> [!danger]
> - RR > 30 或 < 8
> - SpO2 < 90% 即使吸氧 / 高流量
> - 三脚架 / 辅助肌 / 反常呼吸
> - **沉默胸**(严重哮喘 → 即将呼吸停止)
> - 神志改变(CO2 潴留 / 缺氧脑病)
> - 单侧呼吸音消失(警惕张力性气胸)
> - SBP < 90 + 气短(休克)
> - 喘鸣 stridor(上气道,即将关闭)
> - 任何"病人吸不上气"的眼神

## ⚠️ 常见漏诊场景

> [!warning]
> 1. **老年"哮喘"其实是 CHF**(cardiac asthma)→ 听 S3、查 BNP。
> 2. **气短是 ACS 唯一表现** → 老年女性糖尿病常如此。
> 3. **沉默胸是哮喘最危险信号**——不是好转。
> 4. **PE 听诊清晰,常被忽视** → 任何不能解释的低氧 + 心动过速 = PE 直到证实。
> 5. **过度通气归因焦虑** → DKA、败血症、阿司匹林中毒、PE 都可表现过度通气,先排器质病因。
> 6. **胸腔积液被当肺炎"治疗效果不好"** → CXR + 超声看叶间裂积液。
> 7. **气胸 CXR 正常**(小、隐匿)→ 床旁超声看 lung sliding。
> 8. **重度 COPD 给高浓度氧** → CO2 潴留加重。控制 SpO2 88-92%。
> 9. **"没有水肿就不是 CHF"** → 急性心衰可无周围水肿,只有肺水肿。
