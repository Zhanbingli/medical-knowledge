# 意识改变(Altered Mental Status, AMS)

> [!summary] 接到这个主诉,3 件事必做
> 1. **ABC + 指尖血糖**(永远先做血糖,2 分钟内)
> 2. 评估 **GCS / FOUR / AVPU + 瞳孔 + 神经定位 + 脑膜刺激**
> 3. 立即查 **ECG + 静脉血气 + 电解质 + 氨 + 毒物筛查 + 头 CT 决策**

---

## 一句话定义

任何**警觉度(arousal)**或**内容(content)**的急性 / 亚急性改变。包括:
- **谵妄(Delirium)**:急性、波动、注意力障碍 + 思维混乱(可表现亢进 / 抑制 / 混合)
- **嗜睡 / 昏睡(Drowsy / Stupor)**:警觉度下降但可被唤醒
- **昏迷(Coma)**:不能被刺激唤醒、无目的反应

来源:DSM-5 谵妄诊断标准;Plum & Posner Stupor and Coma 经典定义;中国谵妄诊治专家共识 <!-- 待具体年份与杂志补充 -->。

## 必须排除的 Killer(AEIOU TIPS 助记 + 现代扩充)

> [!danger] 第一关:**测血糖**(即使你不怀疑)
> [[低血糖]] 漏诊 = 不可逆脑损伤。任何意识改变 → 指尖血糖 < 2 分钟内。
> - 处理:50% 葡萄糖 20-40 mL IV(成人);儿童 0.5-1 g/kg
> - 反复发作 / 长期用磺脲 → 长效葡萄糖 + 监护
> - 来源:中国 2 型糖尿病防治指南 2020(中华糖尿病杂志);ADA Standards of Care 2024

> [!danger] [[脑卒中]] / 大动脉闭塞
> 突发意识改变 + 局灶神经定位(偏瘫、失语、偏盲)→ **卒中通道 < 4.5h** 评估
> - 头 CT 平扫排出血,然后 CTA / MRI 决定取栓 / 溶栓
> - "醒后卒中"用 DWI-FLAIR 错配评估
> - 来源:中国急性缺血性脑卒中诊治指南 2018(中华神经科杂志);AHA / ASA Acute Ischemic Stroke 2019 + 2024 update

> [!danger] [[蛛网膜下腔出血]](SAH)
> **"我一生中最严重的头痛"突然发作** + 意识改变 → SAH 直到证实排除。
> - 头 CT 6h 内敏感性 > 95%(BMJ 多项 meta);超过 6h 阴性 → LP 排除黄变
> - Hunt-Hess / WFNS 分级
> - 来源:AHA / ASA SAH Guideline 2023;中国蛛网膜下腔出血诊治指南 2019(中华神经外科杂志)

> [!danger] [[细菌性脑膜炎]] / [[脑膜炎球菌病]]
> 发热 + 头痛 + 颈强直 + 意识 → 经典四联(全部具备 < 50%,任意 2 项即筛查)
> - **1 小时内经验性抗生素 + 地塞米松**
> - 来源:NICE Bacterial Meningitis 2024;中国成人社区获得性细菌性脑膜炎诊治专家共识 2015

> [!danger] [[阿片中毒]] / 镇静药过量
> 三联征:**意识抑制 + 呼吸抑制(RR < 8)+ 瞳孔针尖样**
> - 立即 [[纳洛酮]] 0.04-0.4 mg IV / IM,可重复;阿片依赖者从小剂量起防戒断
> - BZD 中毒 [[氟马西尼]] 谨慎(有癫痫风险,长期 BZD 使用者禁用)
> - 来源:中国药物中毒诊治专家共识 <!-- 待具体年份补充 -->;UpToDate Opioid intoxication

> [!danger] [[严重低钠血症]](< 120 mmol/L 或快速下降)
> 头痛、恶心、抽搐、昏迷;过快纠正 → **渗透性脱髓鞘综合征(ODS)**
> - 急性症状性 < 48h:3% 高渗盐水 100 mL IV bolus + 必要时重复(2-3 次)
> - 24h 纠正 ≤ 8-10 mmol/L,慢性更严
> - 来源:European Hyponatremia Guideline 2014;中国低钠血症诊治专家共识 <!-- 待具体年份补充 -->

> [!danger] [[糖尿病酮症酸中毒]] / [[HHS]]
> 高血糖 + 酮 / 高渗 + 意识改变 → **静脉补液 + 胰岛素 + 钾 + 找诱因**
> - DKA:血糖 > 13.9 + pH < 7.3 + 阴离子隙↑ + 酮阳性
> - HHS:血糖 > 33 + 渗透压 > 320 + 无酮 / 轻酮
> - 来源:ADA Hyperglycemic Crises 2024;中国 2 型糖尿病防治指南 2020

> [!danger] [[肝性脑病]] / 尿毒症脑病 / 高碳酸血症
> 慢性肝病 + 意识改变 / 扑翼样震颤 → 肝性脑病
> - 找诱因:GI 出血、感染、电解质、便秘、镇静药
> - 处理:乳果糖 + 利福昔明 + 诱因纠正
> - 慢性 COPD CO2 潴留 → 高流量氧加重(目标 SpO2 88-92%)
> - 来源:AASLD Hepatic Encephalopathy 2014;中国肝性脑病诊疗专家共识 2018(中华消化杂志)

> [!danger] [[Wernicke 脑病]]
> 三联征:**意识改变 + 眼肌麻痹 + 共济失调**(全具备 < 33%,任一项 + 危险因素即治疗)
> - 危险因素:酗酒、长期呕吐、营养不良、孕吐、术后、肿瘤化疗、长期 PPN
> - **任何意识不清且可能营养不良者:维生素 B1(硫胺素)100-500 mg IV 经验性给**
> - **必须先于葡萄糖给**(葡萄糖会消耗剩余 B1 加重 Wernicke)
> - 来源:EFNS Wernicke 指南;中国维生素 B1 缺乏诊治共识 <!-- 待具体年份补充 -->

> [!danger] [[非惊厥性癫痫持续状态]](NCSE)
> 持续意识不清不能解释 + 既往癫痫 / 中风 / 脑炎史 → 怀疑 NCSE
> - **EEG 是诊断金标准**(临床表现可极隐匿)
> - 经验性 [[苯二氮䓬类]]:试验性 [[劳拉西泮]] 0.1 mg/kg 或 [[地西泮]] 0.15 mg/kg
> - 来源:ILAE Status Epilepticus 2015;中国癫痫持续状态诊治专家共识 2018(中国神经精神疾病杂志)

> [!danger] [[CO 中毒]] / 缺氧
> 集体起病、火灾后、密闭空间、家用煤炉
> - 普通 SpO2 不可靠(SpO2 测的是 COHb + O2Hb)→ 需测**碳氧血红蛋白(COHb)**
> - 处理:100% 高流量氧;严重(意识 / 心肌缺血 / 孕妇)→ **高压氧**
> - 来源:中国一氧化碳中毒诊治专家共识 2018;UHM Carbon Monoxide Poisoning Practice Recommendations

> [!danger] [[脓毒症]] / 严重感染
> qSOFA 中"意识改变"是关键一项;免疫抑制 / 老年常以谵妄起病而无典型发热
> - 见 [[脓毒症]] 1h bundle
> - 来源:SSC 2021;中国脓毒症 / 脓毒性休克急诊治疗指南 2018

## 常见原因

- [[ICU谵妄]] / 老年谵妄(住院相关,**80% 老年术后 / ICU 出现**)
- [[药物性谵妄]]:抗胆碱药、苯二氮䓬、阿片、糖皮质激素、抗 Parkinson 药、免疫抑制剂
- [[尿路感染老年]](唯一表现可能就是谵妄)
- [[脱水]] / 电解质紊乱(钠、钙、镁)
- [[酒精戒断]](DT,常 48-96h 后)
- 甲状腺功能异常(粘液水肿性昏迷 / 甲亢危象)
- 肾上腺危象
- 高血压脑病
- PRES(可逆性后部白质脑病)
- 心律失常 / 心源性休克(灌注不足)
- 自身免疫性脑炎(NMDA 等)

## 必查体征

- **生命体征 + SpO2 + 体温 + 指尖血糖 + GCS / AVPU**
- **瞳孔**:大小 / 对称性 / 光反射 / 眼位
  - 针尖瞳孔 → 阿片 / 桥脑出血
  - 散大固定 → 抗胆碱 / 脑疝 / 死亡
  - 不等大 + 一侧光反射消失 → 颞叶钩回疝
- **眼球运动**:Doll's eyes(脑干完整性)
- **角膜反射 / 咽反射**(脑干)
- **运动反应**:对称性、去皮质 / 去大脑姿势
- **脑膜刺激**:颈强直、Kernig、Brudzinski
- **皮肤**:针孔(IVDU)、瘀斑(SAH 发生时罕见)、皮疹(脑膜炎球菌)、樱桃红(CO)
- **口气**:酒、酮、肝臭、尿臭
- **听诊心肺**:杂音(IE、栓子)、肺部、节律
- **腹部**:肝大 / 腹水(肝性脑病)、膀胱(尿潴留谵妄)
- **下肢**:水肿、DVT(警觉 PE 致缺氧脑病)

## 床旁首查

- [ ] **指尖血糖**(<2 分钟,优先于一切)
- [ ] **静脉通路 ×2 + 心电监护 + SpO2**
- [ ] **12 导联 ECG**(心律失常 / 心肌缺血灌注)
- [ ] **ABG + 静脉血气**(pH / pCO2 / 乳酸 / COHb 若怀疑)
- [ ] **CBC + 电解质 + 钙 + 磷 + 镁 + 肝肾功 + 凝血 + 氨**
- [ ] **TSH + 维生素 B12 / 叶酸**(亚急性 AMS)
- [ ] **肌钙蛋白 + BNP**(心源性灌注)
- [ ] **毒物筛查**(乙醇、阿片、苯二氮䓬、安非他明、TCA)
- [ ] **尿常规 + 尿培养**(老年 UTI 谵妄)
- [ ] **头 CT 平扫**(局灶定位 / 头外伤 / 高度怀疑结构性)
- [ ] **腰穿**(怀疑感染 / 出血 / 自身免疫脑炎,**先 CT 排除占位 / 移位**)
- [ ] **EEG**(疑似 NCSE / 不能解释的持续不清)
- [ ] **MRI**(CT 阴性但持续异常,尤其后循环卒中、脑炎、PRES、Wernicke)

## 一线处理(经验性"昏迷鸡尾酒"现代版)

> [!tip] 顺序很重要
> **先 B1,再葡萄糖**(防 Wernicke);疑似阿片先纳洛酮;有目标地经验性,不是无差别全给。

| 经验性给药 | 适应证 | 剂量参考 |
|---|---|---|
| **维生素 B1(硫胺素)** | 任何可能营养不良 / 酗酒 / 长期呕吐 | 100-500 mg IV |
| **50% 葡萄糖** | 血糖 < 3.9 mmol/L | 20-40 mL IV(成人) |
| **纳洛酮** | 呼吸抑制 + 瞳孔针尖 / 阿片史 | 0.04-0.4 mg IV / IM,可重复 |
| **氟马西尼** | 单纯 BZD 过量(谨慎) | 0.2 mg IV,可逐步重复;长期 BZD / 共服 TCA / 癫痫者禁用 |
| **氧** | SpO2 < 90% / CO 中毒 | 高流量;COPD 注意目标 88-92% |
| **苯二氮䓬** | 抽搐 / NCSE / 酒精戒断 | 劳拉西泮 / 地西泮按癫痫剂量 |

来源:SSC 2021;ADA 2024;ILAE 2015;中国相关共识(具体年份待补充);具体本院剂型与规格 [需查证]。

## 🔺 立刻升级的临床表现

> [!danger]
> - GCS ≤ 8 → 气道保护(气管插管)
> - 呼吸抑制 RR < 8 / SpO2 < 90% / pCO2 > 60
> - 瞳孔不等大 + 一侧光反射消失(脑疝先兆)
> - 库欣三联征(高血压 + 心动过缓 + 不规则呼吸)→ 颅高压
> - 持续抽搐 > 5 min(癫痫持续状态)
> - 局灶神经定位 + 起病 < 4.5h(卒中通道)
> - 雷击样头痛 + 意识(SAH 通道)
> - 集体起病(CO / 食物 / 化学暴露)
> - 体温 < 35 / > 40
> - 不能解释的休克
> - 任何"病人意识在持续恶化"的趋势

## ⚠️ 常见漏诊场景

> [!warning]
> 1. **没查血糖** — 最低成本的检查,漏诊代价最高
> 2. **酗酒病人直接给葡萄糖未先给 B1** → 急发 / 加重 Wernicke
> 3. **老年 UTI 唯一表现就是谵妄**,未做尿检
> 4. **慢性硬膜下血肿** 老人跌倒后数周隐匿出现 AMS,病史易漏
> 5. **非惊厥性癫痫持续状态** 看似嗜睡,EEG 才能发现
> 6. **CO 中毒**普通 SpO2 不可靠,需 COHb;集体起病 + 头痛 + 恶心要警觉
> 7. **低钠血症纠正过快** → 渗透性脱髓鞘综合征(锁定综合征样)
> 8. **抗胆碱中毒** 老年 AMS + 高热 + 皮肤干 + 心动过速 + 散瞳 → "热如兔、红如甜菜、干如骨、瞎如蝙蝠、疯如帽匠"
> 9. **5-HT 综合征 / NMS**:精神药物联用 + 高热 + 强直 + 自主神经紊乱
> 10. **甲状腺危象 / 粘液水肿性昏迷** 老人罕见但漏诊致命
> 11. **HIV / 免疫抑制者机会感染**:Toxoplasma / PML / 隐球菌 → MRI + LP
> 12. **自身免疫性脑炎(NMDA 等)** 年轻女性 + 精神症状 + 抽搐 → 抗 NMDA 抗体
> 13. **Locked-in syndrome 误为昏迷** 桥脑梗死 → 患者意识清但仅能眼球运动
> 14. **精神 / 心因诊断作为首诊** → 必须排除全部器质方可
