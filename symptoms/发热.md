# 发热

> [!summary] 接到这个主诉,3 件事必做
> 1. 先判断是不是 [[emergencies/脓毒症|脓毒症]] / [[中性粒细胞减少性脓毒症]] / [[diseases/细菌性脑膜炎|细菌性脑膜炎]] / [[感染性心内膜炎]] / [[疟疾]]
> 2. 生命体征 + 意识 + 皮疹 + 脑膜刺激征 + 感染灶查体
> 3. 根据暴露史 + 定位症状决定首查:血常规、**乳酸**、尿常规、影像、血培养、病原学

---

## 一句话定义

正常体温调节范围被疾病重置,导致核心体温超出正常波动:
- **临床发热**:口腔体温 ≥ 38.0℃ / 腋温 ≥ 37.5℃ / 直肠 ≥ 38.3℃
- **不明原因发热(FUO)**:≥ 3 周、多次 ≥ 38.3℃、规范初步诊查仍未明确(Petersdorf 1961 经典定义,Durack-Street 1991 修正,中国发热待查共识 2017 沿用)

## 必须排除的 Killer

> [!danger] [[emergencies/脓毒症|脓毒症]] / 脓毒性休克
> 任何疑似感染 + 器官功能障碍即为脓毒症。
> - 急诊筛查:[[scores/qSOFA|qSOFA]] ≥ 2(意识改变 / SBP ≤ 100 / RR ≥ 22)→ 高度警觉
> - **乳酸 > 2 mmol/L** = 隐匿低灌注;**> 4** = 立即升级
> - **1 小时 bundle**:乳酸、血培养、广谱抗生素、晶体复苏、必要时升压
> - 老年、免疫抑制、肿瘤治疗、婴幼儿、脊髓损伤者**可无典型发热**——警觉低体温 / 异常生命体征
> 
> 来源:Surviving Sepsis Campaign 2021;中国脓毒症 / 脓毒性休克急诊治疗指南 2018(中华急诊医学杂志);NICE Suspected Sepsis 指南 2024(成人非妊娠急性医疗场景推荐 NEWS2)

> [!danger] [[中性粒细胞减少性脓毒症]](FN)
> 化疗后 30 天内或其他原因致 ANC < 0.5 × 10⁹/L(或预测 48h 内降至此水平),发热 ≥ 38.3℃ 单次 / ≥ 38.0℃ 持续 1h。
> - **1 小时内广谱抗生素**(经验首选 piperacillin-tazobactam 等抗假单胞 β-内酰胺,具体本院方案)
> - 不等培养结果;不退热前不停药
> - 高危(MASCC < 21)→ 住院 IV;低危可考虑口服
> 
> 来源:中国中性粒细胞缺乏伴发热抗菌药物临床应用指南 2020(中华医学会肿瘤学分会);IDSA Febrile Neutropenia 2018;NICE Sepsis 2024

> [!danger] [[diseases/细菌性脑膜炎|细菌性脑膜炎]] / [[脑膜炎球菌病]]
> 经典四联征:**发热 + 头痛 + 颈强直 + 意识或认知改变**(全部具备 < 50%,但任意 2 项即应考虑)。
> - **不褪色瘀点 / 紫癜** = [[脑膜炎球菌病]],数小时内致死
> - **经验性抗生素 1 小时内**:头孢曲松 2g IV q12h ± 万古(地区耐药)± 阿莫西林(老 / 免疫抑制覆盖 Listeria)
> - **+ 地塞米松 0.15 mg/kg**(IV,与首剂抗生素同时或之前)
> - LP 前 CT 指征:免疫抑制、新发癫痫、神经定位、意识显著下降 — 否则不应延误
> 
> 来源:NICE Bacterial Meningitis 指南 2024;IDSA Bacterial Meningitis 2017;中国成人社区获得性细菌性脑膜炎诊治专家共识 2015(中华神经科杂志)

> [!danger] [[感染性心内膜炎]](IE)
> 任何**发热 + 新发心脏杂音** / 长期 IV 用药史 / 静脉吸毒 / 人工瓣 / 起搏器 / 既往 IE 史 → 必查。
> - 周围体征(Janeway 病变、Osler 结节、Roth 斑、瘀点)出现率 < 25% 但有指向性
> - **3 套不同部位血培养**(间隔至少 30 min,抗生素前)
> - 经胸超声 → 经食道超声(TEE 敏感性更高)
> - Duke 标准分层
> 
> 来源:ESC Infective Endocarditis 指南 2023;AHA IE 科学声明 2015(2024 update);中国感染性心内膜炎诊断与治疗专家共识 2014(中华心血管病杂志)

> [!danger] [[疟疾]](中国学生面对**所有**输入性发热的优先排查)
> 近期来自疟疾流行区(非洲、东南亚、南亚、南美)+ 发热,**数小时内可恶化为脑型疟 / 重症疟**。
> - **薄、厚血涂片 ×3**(不同时间点,不应仅凭一次阴性排除)
> - 快速诊断试纸(RDT)作为辅助
> - 恶性疟(P. falciparum)**重症标志**:意识改变、抽搐、严重贫血、肾功能、肺水肿、休克 → 立即 IV 青蒿琥酯
> - 中国仍偶发输入性死亡病例,**漏诊代价极重**
> 
> 来源:WHO Guidelines for the Treatment of Malaria 2023;中国疟疾诊断标准 WS 259-2015(国家卫健委);中国寄生虫病防治年度报告

> [!danger] [[重症肺炎]]
> 发热 + 新发咳嗽 / 脓痰 / 胸痛 / 气促 / 氧合下降 → 评估肺炎严重程度。
> - **CURB-65** ≥ 2 考虑住院,≥ 3 考虑 ICU
> - PSI 风险分层(更细)
> - 流感高发季 + 影像快速进展 → 警惕病毒性肺炎 / ARDS
> 
> 来源:中国成人社区获得性肺炎诊断和治疗指南 2016(中华结核和呼吸杂志);IDSA / ATS CAP 2019;NICE 成人肺炎诊治指南(请验证最新版本号:NG138 系 CAP 抗菌处方 2019,核心 CAP 临床指南待查)

> [!danger] [[坏死性筋膜炎]]
> 发热 + **痛感不成比例**(查体看似无大事但患者疼痛剧烈) + 皮肤异色 / 水疱 / 捻发音 / 触诊硬结。
> - LRINEC 评分 ≥ 6 提示
> - 影像可耽误 → **临床高度怀疑直接外科探查**
> - 延误每小时死亡率 +9%
> 
> 来源:IDSA Skin and Soft Tissue Infections 2014;中国坏死性筋膜炎诊治专家共识(待补充)

## 常见非危急原因

- [[上呼吸道感染]]:鼻塞、咽痛、咳嗽、流涕——需排除 [[肺炎]] 与隔离相关传染病
- [[尿路感染]]:尿频、尿急、尿痛、肋脊角叩痛;老年 / 免疫受损者定位症状不典型
- [[胃肠炎]]:腹泻、呕吐、腹痛;问饮食、聚集发病、旅行、抗菌药暴露
- [[药物热]]:新启用药物后发热,停药前先排除感染、血栓、自身炎症病
- [[深静脉血栓]] / [[emergencies/肺栓塞|肺栓塞]]:发热可伴胸痛 / 气促 / 下肢肿——"有热"不等于排除血栓
- [[自身炎症性疾病]] / [[风湿免疫病]]:反复发热伴皮疹 / 关节痛 / 浆膜炎 / 血细胞异常
- [[甲亢危象]] / [[药物性发热反应]]:非感染性体温升高,需鉴别

## 必查体征

- **生命体征**:体温、HR、RR、BP、SpO2、意识状态;成人非妊娠急性医疗场景使用 [[NEWS2]] / [[scores/qSOFA|qSOFA]]
- **一眼判断**:精神反应、寒战、发绀、花斑、皮肤湿冷、脱水、呼吸窘迫
- **皮肤黏膜**:不褪色瘀点 / 紫癜、蜂窝织炎、伤口、压疮、导管口、口腔溃疡、Koplik 斑、Janeway 病变 / Osler 结节(IE)
- **头颈**:颈强直、Kernig / Brudzinski 征、畏光、鼻窦压痛、咽峡、扁桃体渗出、淋巴结
- **胸肺**:呼吸音减弱、湿啰音、胸膜摩擦音、喘鸣
- **心血管**:**新发杂音**(IE 关键)、外周灌注、末梢温度、下肢肿痛
- **腹部 / 泌尿**:腹膜刺激征、Murphy 征、肋脊角叩痛、膀胱区压痛、肝脾大
- **关节 / 软组织**:**痛感不成比例 + 红斑 / 水疱 / 捻发音**(警惕 [[坏死性筋膜炎]])
- **神经**:意识、定向力、局灶体征、脑膜刺激征

## 床旁首查

- [ ] **复测生命体征 + [[NEWS2]] / [[scores/qSOFA|qSOFA]]**;休克或意识改变立即升级
- [ ] **指尖血糖**:意识改变、出汗、震颤、糖尿病或进食差时优先
- [ ] **乳酸**:疑似脓毒症 1 小时内必查(SSC 2021 / 中国脓毒症共识 2018)
  - > 2 mmol/L = 隐匿低灌注
  - > 4 mmol/L = 立即升级 ICU + 复苏
- [ ] **血常规 + 分类**(中性粒细胞、淋巴细胞、血小板)
- [ ] **CRP / PCT**(细菌 vs 病毒辅助;PCT 升高支持细菌)
- [ ] 肝肾功能、电解质、凝血(疑似脓毒症必查)
- [ ] **血培养 ≥ 2 套**(不同部位,IE 怀疑时 3 套间隔 30min,抗生素前)
- [ ] **尿常规 + 尿培养**:尿路症状、老年、妊娠、导尿或不明感染灶时
- [ ] **胸部影像**:咳嗽 / 胸痛 / 气促 / 低氧 / 肺部体征 / 高风险宿主
- [ ] **病原学检测**:按流行季节选 [[流感]] / [[COVID-19]] / [[麻疹]] / [[登革热]] / [[疟疾血涂片]]
- [ ] 妊娠检测:有妊娠可能者(影响影像、用药、感染风险)
- [ ] 询问过去 18 小时排尿;少尿提示灌注风险(NICE Sepsis 2024)
- [ ] **感染源全身查体**:皮肤、导管、伤口、口腔、会阴、肛周、关节

## 🔺 立刻升级的临床表现

- 发热合并血流动力学不稳、皮肤花斑 / 湿冷、意识改变、呼吸窘迫或氧合下降
- **乳酸 > 4 mmol/L** 或 > 2 mmol/L 不能纠正
- 不褪色瘀点 / 紫癜、颈强直、畏光、头痛伴意识改变
- 免疫抑制 / 化疗 30 天内 / 长期激素 / 无脾 / 留置中心静脉导管者出现发热或感染表现
- 年龄 > 75 或非常虚弱者出现感染表现,即使体温不高也警惕脓毒症
- 妊娠 / 产后 6 周内发热合并腹痛、阴道异常分泌物、持续出血或全身不适
- 发热 + 新发心脏杂音 / 周围栓塞征(警惕 [[感染性心内膜炎]])
- **痛感不成比例**于查体所见 + 皮肤改变 + 发热(警惕 [[坏死性筋膜炎]])
- 疟疾流行区返回 + 发热 + 任何重症征象(意识、呼吸、休克)— 立即 IV 青蒿琥酯
- 发热 + 皮疹 + 咳嗽 / 流涕 / 结膜炎(尤其未免疫 / 旅行史)→ 隔离 + 报告疑似 [[麻疹]]
- 登革热流行区暴露后发热合并腹痛、持续呕吐、黏膜出血、嗜睡 / 烦躁、呼吸急促或冷湿皮肤(WHO 警示征)

## ⚠️ 常见漏诊场景

- 把"退热后精神好转"当安全信号,漏掉早期脓毒症"暖休克 → 冷休克"过渡
- 老年 / 肿瘤治疗 / 免疫抑制者**没有高热**就被误判为轻症感染
- **没问旅行史漏掉疟疾**——中国实习医生最易栽的坑,漏诊死亡率极高
- 发热 + 头痛只按上呼吸道感染处理,未查颈强直、意识、不褪色皮疹
- 发热 + 新杂音未做血培养与超声,漏诊 IE
- 发热 + 皮疹未问旅行 / 疫苗 / 接触史,未及时隔离麻疹等传染病
- 发热 + 腹痛只考虑胃肠炎,漏诊胆道感染、阑尾炎、盆腔炎、腹腔脓肿
- 发热 + 胸痛 / 气促只按肺炎处理,漏掉肺栓塞 / 心内膜炎 / 心肌炎
- **痛感不成比例**被忽视,漏诊坏死性筋膜炎(早期 6 小时是黄金窗口)
- 留置导管 / 伤口 / 压疮 / 牙源性感染 / 肛周感染没做全身暴露查体
- 抗菌药前未采培养——降低后续病原学诊断率(但**病情危重不应因等检查延误升级和治疗**)
- 退热药遮蔽热型 → 间日疟、伤寒等热型诊断价值消失
