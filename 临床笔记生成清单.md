# 临床笔记生成清单

> [!summary] 生成原则
> 先补症状入口和急症，再补系统疾病、药物、评分、检查和操作。每篇具体临床笔记必须按 `AGENTS.md` 写:不确定就 `[需查证]`，临床数字必须有来源。

## 优先级说明

| 优先级 | 含义 | 生成策略 |
|---|---|---|
| P0 | 当班必须能识别和升级 | 先生成症状/急症笔记 |
| P1 | 常见病房问题 | 生成疾病 + 药物 + 评分联动笔记 |
| P2 | 系统复习必备 | 生成疾病概要和鉴别框架 |
| P3 | 进阶或轮转相关 | 有对应轮转时补齐 |

- [ ] 没有未引用的剂量、阈值、时间窗、百分比。
- [ ] 指南编号已核验；不确定编号就只写机构 + 主题 + 年份。
- [ ] 中国医疗场景有国际指南 + 中国指南；缺中国指南时标注 `<!-- 待中国指南补充 -->`。
- [ ] `我的临床观察` 只保留空骨架。
