# Wells PE 评分

> [!summary]
> **用途**:疑似肺栓塞临床预测概率分层
> **适用人群**:急诊 / 门诊疑似 PE 成人,血流动力学稳定
> **不适用**:不稳定 / 已确诊 PE / 妊娠(妊娠用 YEARS / 修订标准)/ 儿童

---

## 评分项目

| 项目 | 分值 |
|---|---|
| **DVT 临床体征**(单侧腿肿 + 沿深静脉走行压痛) | 3 |
| **PE 是 / 与 PE 同样可能的诊断**(主观判断) | 3 |
| HR > 100 bpm | 1.5 |
| **近期制动 ≥ 3 天 或 4 周内手术** | 1.5 |
| 既往 DVT / PE 史 | 1.5 |
| 咯血 | 1 |
| 活动性肿瘤(治疗中、6 个月内治疗或姑息) | 1 |

来源:Wells PS et al. Thromb Haemost 2000(原始版);ESC Pulmonary Embolism Guideline 2019(沿用)。

---

## 解读

### 三层分类(经典)

| 总分 | 风险层 | PE 患病率(参考) | 处置 |
|---|---|---|---|
| < 2 | 低 | ~3% | D-dimer(年龄校正);阴性可排除 |
| 2-6 | 中 | ~20% | D-dimer;阳性 → CTPA |
| > 6 | 高 | ~65% | **直接 CTPA**,不靠 D-dimer 排除 |

### 二层分类(简化版,SSCM 与多数现代指南采用)

| 总分 | 分层 | 处置 |
|---|---|---|
| ≤ 4 | PE 不太可能(unlikely) | D-dimer(年龄校正);阴性可排除 |
| > 4 | PE 可能(likely) | **直接 CTPA** |

> 二层分类更便于临床操作。来源:ESC PE Guideline 2019;中华医学会呼吸病学分会 / 心血管病学分会肺血栓栓塞症诊治指南 2018(中华医学杂志)。

### 与 D-dimer 配合(重要)

- **低 / 不太可能 + D-dimer 阴性**(年龄校正:> 50 岁阈值 = 年龄 × 10 µg/L FEU)→ 排除 PE
- **低 / 不太可能 + D-dimer 阳性** → CTPA
- **高 / 可能** → **直接 CTPA**,D-dimer 不足以排除

## 临床应用场景

1. **急诊室胸痛 / 气短分诊**:配合 [[scores/HEART|HEART]](心源性)、Wells DVT、PERC 一起用
2. **术后 / 制动后突发气短**
3. **怀疑 [[emergencies/肺栓塞|肺栓塞]] 但 CTPA 资源有限时**:Wells 分层避免不必要的 CTPA
4. **门诊 D-dimer 决策**:低度怀疑 + D-dimer 阴性 + Wells 低分 → 安全排除

---

## 局限性 / 不适用情况

> [!warning]
> 1. **"PE 是 / 与 PE 同样可能"是主观项**(3 分),实习医生评分波动大;同一病人不同医生可差 3 分
> 2. **不能替代临床判断** — 评分低但临床高度怀疑就继续查
> 3. **不适用**:
>    - 血流动力学不稳(直接 CTPA / 床旁超声)
>    - 妊娠(用 [[YEARS]] 或妊娠修订标准)
>    - 儿童
> 4. **D-dimer 在妊娠 / 肿瘤 / 术后 / 老年常假阳性** → 高假阴性率前应用 PERC 或临床判断
> 5. **复发 PE 评估** Wells 不如客观影像
> 6. **慢性血栓栓塞性肺动脉高压(CTEPH)** Wells 不适用
