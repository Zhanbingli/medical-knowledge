# 阿司匹林(Aspirin / ASA)

> [!summary] 给药前 5 秒检查清单
> - [ ] 过敏史(哮喘三联征:鼻息肉 + 哮喘 + ASA 过敏 = 禁用)
> - [ ] 24h 内有无活动性出血(消化道、颅内)
> - [ ] 是否儿童 + 病毒感染(Reye 综合征风险)
> - [ ] 妊娠状态(3rd trimester 全剂量禁用)
> - [ ] 是否合用其他抗血小板 / 抗凝 / NSAID

---

## 类别 / 机制

**抗血小板药物 / 非甾体抗炎药(NSAID)**。

机制:**不可逆**乙酰化血小板 COX-1 → 抑制 TXA2 合成 → 抑制血小板聚集。
- 血小板无核 → 不可逆抑制持续血小板寿命(7-10 天)
- 高剂量同时抑制 COX-2 → 抗炎、退热、镇痛
- **低剂量(75-100 mg/d)= 抗血小板**;**高剂量(≥ 500 mg)= 抗炎 / 退热**

## 适应证

| 用途 | 证据等级 |
|---|---|
| **[[diseases/急性冠脉综合征|急性冠脉综合征]] 急性期 + 长期** | I 级,首选 (ESC 2023, AHA, 中国 STEMI 2019) |
| **[[卒中]] 二级预防(非心源性)** | I 级 |
| **急性缺血性卒中(早期)** | 24-48h 内 162-325 mg(若不溶栓 / 取栓后) |
| **[[房颤]] 抗栓**(替代不能用 DOAC / 华法林时) | 弱推荐,**首选仍为口服抗凝药** |
| **PCI / CABG 术后长期** | 标准方案 |
| **川崎病急性期 + 后期** | 抗炎 + 抗血小板 |
| **子痫前期预防(妊娠 11-14 周起,75-150 mg/d)** | I 级 (NICE / ACOG / 中国妇产科共识) |
| 退热镇痛(成人) | 不再首选(对乙酰氨基酚 / 布洛芬更安全) |

> [!warning] **不再推荐用于一级心血管预防**(ASCEND / ARRIVE / ASPREE 三项 RCT 后)。仅高血栓风险 + 低出血风险特定人群个体化。来源:USPSTF 2022;ESC 心血管预防 2021。

## 用法用量

| 适应证 | 剂量 | 备注 |
|---|---|---|
| **ACS 负荷** | 150-300 mg 嚼服 | 首剂嚼服加快吸收(ESC ACS 2023) |
| **ACS / 卒中长期** | 75-100 mg/d 口服 | 中国 75-100 mg;美国 81 mg |
| **急性缺血性卒中** | 162-325 mg 单剂 24-48h 内启动 | 若 rt-PA 后,等 24h |
| **子痫前期预防** | 75-150 mg/d 口服,夜间服 | 11-14 周起,持续至 36 周或分娩前 |
| **川崎病急性期** | 30-50 mg/kg/d 分次(高剂量抗炎) | 退热后改 3-5 mg/kg/d 抗血小板 |

> 来源:ESC ACS Guideline 2023;中国急性 ST 段抬高型心肌梗死诊断和治疗指南 2019;中国缺血性卒中和短暂性脑缺血发作二级预防指南 2022(中华神经科杂志);NICE Hypertension in pregnancy <!-- 待具体编号核实 -->;中国妊娠期高血压疾病诊治指南 2020(中华妇产科学杂志)。具体剂型和规格 [需查证: 本院药品目录]。

## 禁忌证

> [!danger] 绝对禁忌
> - 已知阿司匹林 / NSAID 严重过敏
> - **哮喘三联征**(Samter 三联征:鼻息肉 + 哮喘 + ASA / NSAID 过敏)
> - 活动性消化道出血
> - 颅内出血(出血性卒中、近期颅内手术)
> - 严重血小板减少 < 50 × 10⁹/L
> - **妊娠 3rd trimester 全剂量**(导管早闭、出血风险);低剂量预防子痫前期不在此列
> - **儿童病毒感染**(水痘、流感)→ Reye 综合征
> - 严重未控制高血压(伴出血风险时)
> - 凝血功能障碍 / 血友病

> [!warning] 相对禁忌
> - 消化道溃疡病史(可加 PPI 保护)
> - 严重肝肾功能不全
> - 哺乳期(可分泌至乳汁,小剂量短期一般可用,长期不推荐)
> - 围术期(择期手术前 5-7 天评估,心血管高危者权衡)

## 常见不良反应

- **消化道**:消化不良、恶心、上腹不适、糜烂、溃疡(剂量依赖)
- **出血**:鼻衄、皮下淤血、月经量增加
- 耳鸣 / 听力下降(高剂量 / 中毒征)
- 哮喘恶化(已知敏感者)
- 转氨酶升高
- 高尿酸血症 / 痛风加重(低剂量)

---

## ⚠️ 危险不良反应

> [!danger]
> **1. 上消化道出血 / 穿孔**
> - 风险与剂量、年龄、合用 NSAID / 抗凝、Hp 感染、溃疡史相关
> - 高危(年龄 > 65 / 既往溃疡 / 合用其他抗血小板 / 抗凝)→ 加 PPI
> - 临床表现:呕血、黑便、低血压、HR 升、Hb 下降
> - 处理:立即停药、输血、内镜止血、PPI

> [!danger]
> **2. 颅内出血**
> - 高血压未控、抗凝合用、外伤后 → 风险升
> - 任何抗血小板治疗中突发头痛 / 意识改变 → 立即头 CT

> [!danger]
> **3. Samter 三联征 / 严重过敏**
> - 哮喘急性发作、血管性水肿、休克
> - 用药史 + 鼻息肉 + 哮喘 = 高危
> - 任何阿司匹林后呼吸困难 → 急救

> [!danger]
> **4. Reye 综合征**(儿童)
> - 病毒感染(水痘、流感)+ 阿司匹林 → 急性脑病 + 肝衰
> - **儿童病毒感染禁用阿司匹林**
> - 川崎病例外(已确诊,获益 > 风险)

> [!danger]
> **5. 阿司匹林过量 / 中毒**
> - 急性大量(> 150 mg/kg)→ 代谢性酸中毒 + 呼吸性碱中毒(混合酸碱)
> - 表现:耳鸣、过度换气、高热、意识改变、肺水肿、酸中毒
> - 处理:活性炭、碱化尿液、严重时血液透析

## 肾 / 肝功能调整

- **肾**:轻中度不需调整;严重(CrCl < 10)避免长期高剂量,警觉 GI 出血与肾损伤
- **肝**:严重肝功能不全慎用(凝血障碍)

## 监测指标

- **临床出血**(皮下、鼻衄、月经、消化道、颅内)
- **CBC**(若怀疑慢性失血查 Hb)
- **大便潜血**(疑似消化道出血)
- 无需常规凝血监测(不延长 PT / APTT)
- 长期用 + 高危 → 定期评估 GI 风险 + Hp 检测 / 治疗

## 关键相互作用

> [!warning] 出血风险叠加
> - **其他抗血小板**(P2Y12:[[替格瑞洛]] / [[氯吡格雷]])— ACS 中常规联用,接受
> - **抗凝**:[[drugs/肝素|肝素]] / [[华法林]] / [[利伐沙班]] / [[阿哌沙班]] — 需评估三联抗栓必要性,大幅提升出血
> - **NSAIDs**(布洛芬等)— 不仅出血叠加,**布洛芬可竞争性占用 COX-1 抗血小板效应**(若用药顺序错误)
> - **SSRI**(氟西汀等)— 增加 GI 出血
> - 银杏 / 大蒜 / 鱼油 / 银杏叶提取物
> - 系统溶栓(rt-PA)— 联合时严格按 ACS / 卒中方案

> [!warning] 其他重要相互作用
> - **甲氨蝶呤** — 阿司匹林减少肾排泄 → MTX 中毒
> - **丙磺舒** — 阿司匹林抵消其降尿酸作用
> - **ACEI** — 阿司匹林大剂量轻度削弱 ACEI 降压效果(临床意义小)
> - **酒精** — GI 出血叠加
