# 蛛网膜下腔出血(SAH)

> [!summary] 30 秒摘要
> **红旗**:**雷击样头痛 + 颈强直 + 意识 / 神经定位**——"我一生中最严重的头痛"
> **首查**:**头 CT 平扫 < 6h(敏感性 > 95%)**;阴性 + 持续怀疑 → **腰穿看黄变**
> **一线**:控血压(SBP < 140-160 直至处理)+ 尼莫地平 60 mg q4h × 21 天 + 神经外科 / 介入紧急评估
> **升级**:任何怀疑都立刻打电话——**前 24h 再出血率最高**

---

## 一句话定义

血液进入蛛网膜与软脑膜之间的蛛网膜下腔,常因**囊状动脉瘤破裂**(75-85%)。

- **死亡率 25-50%**(包括院前)
- **再出血(24h 内 ~ 4-13%,2 周内累计 30%)= 死亡核心驱动**
- **早期闭塞动脉瘤(< 24-72h)**= 当前标准

来源:AHA / ASA Guidelines for Aneurysmal SAH 2023;中国蛛网膜下腔出血诊治指南 2019(中华神经外科杂志 / 中华医学会神经外科学分会)。

## 🚩 红旗(立刻警觉)

- **"雷击样头痛"**(thunderclap headache):突然爆发,**几秒钟内达到峰值**;患者描述"我一生中最严重的头痛"
- **前哨头痛**(sentinel headache):破裂前数日-数周,小渗漏所致 → 漏诊代价高
- **运动 / Valsalva 诱发**(咳嗽、用力、性活动)
- 颈强直 + 畏光(脑膜刺激)
- 一过性意识丧失 / 抽搐
- 局灶神经定位(动眼神经麻痹 → 后交通动脉瘤压迫)
- 视网膜下出血(玻璃体下出血,Terson 综合征)
- 高血压病史 + 上述任一

## 鉴别诊断(雷击样头痛全部需要排查)

| 疾病 | 区分点 | 关键检查 |
|------|--------|---------|
| **偏头痛** | 既往典型发作、先兆、缓慢进展 | 病史 + CT 阴性 + LP 阴性 |
| **颈动脉 / 椎动脉夹层** | 颈部疼痛 + 卒中表现 | CTA / MRA |
| **静脉窦血栓(CVST)** | 亚急性头痛 + 凝血高危 | MRV / CTV |
| **可逆性脑血管收缩综合征(RCVS)** | 反复雷击样 + 触发(产后 / 用药)+ "串珠"血管 | CTA / MRA + DSA |
| 高血压脑病 / PRES | 严重高血压 + 后部白质 | MRI |
| 颈源性头痛 | 慢性、活动 / 体位相关 | 病史 |
| 急性闭角青光眼 | 眼痛 + 视力下降 + 瞳孔散大 | 眼压 |
| 急性鼻窦炎 / 牙源性 | 局部压痛、感染征 | 临床 |
| 脑膜炎 / 脑炎 | 发热 + 颈强 + 意识 | LP |
| 占位性病变出血 | CT 见占位伴出血 | MRI |

> [!warning] **前哨头痛**(sentinel headache)漏诊是 SAH 漏诊的主要场景
> 任何"我从来没有过的最严重头痛"+ 突然 → 必须 CT + LP 排除

---

## 首查(到院后 25 分钟内 CT)

- [ ] **头 CT 平扫**(< 6h 内敏感性 > 95-99%,Perry et al BMJ 2011)
- [ ] CT 阴性 + 起病 ≥ 6h 或临床高度怀疑 → **腰穿(LP)**:
  - **黄变(xanthochromia)**:出血后 12h 内出现,最敏感
  - 红细胞计数 / 三管比较(创伤性穿刺会逐管减少;SAH 不变)
- [ ] CTA(确诊 SAH 后立即,定位动脉瘤)
- [ ] DSA(全脑血管造影 — 介入或手术决策金标准)
- [ ] CBC、电解质、肌酐、凝血、肌钙(应激性心肌损伤常见)
- [ ] 12 导联 ECG(神经源性心肌损伤 — ST 改变、长 QT、Takotsubo)
- [ ] 胸片(神经源性肺水肿)

## 临床分级(决定预后)

### Hunt-Hess(经典)

| 级 | 描述 |
|---|---|
| I | 无症状或轻头痛、轻颈强 |
| II | 中-重头痛 / 颈强,无神经定位(除外动眼麻痹) |
| III | 嗜睡 / 混乱或轻定位 |
| IV | 木僵 / 中-重偏瘫 / 早期去脑强直 |
| V | 深昏迷 / 去脑强直 / 濒死 |

### WFNS(基于 GCS,客观)

| 级 | GCS | 主要定位 |
|---|---|---|
| I | 15 | 无 |
| II | 13-14 | 无 |
| III | 13-14 | 有 |
| IV | 7-12 | 有 / 无 |
| V | 3-6 | 有 / 无 |

### 影像分级

- **Fisher**(0-4):基于 CT 出血量
- **改良 Fisher**(0-4):额外考虑脑室出血,**预测血管痉挛更准**

## 一线处理

### 0. 通用稳定

- ABC + 静脉通路 + 心电监护
- 禁食水(可能手术 / 介入)
- 抬头 30°(降低 ICP)
- 镇痛 + 止吐(吗啡 / 芬太尼;昂丹司琼)

### 1. 控血压(再出血最大可控因素)

| 期 | 目标 | 用药 |
|---|---|---|
| **未处理动脉瘤** | **SBP < 140-160 mmHg**(AHA 2023 II 类推荐) | [[拉贝洛尔]] / [[尼卡地平]] IV 滴注 |
| 处理后(夹闭 / 介入) | 个体化,通常允许较高(避免血管痉挛缺血) | — |

来源:AHA / ASA SAH 2023;中国 SAH 诊治指南 2019。具体药物剂量 [需查证: 本院 ICU 方案]。

### 2. 早期闭塞动脉瘤(< 24-72h)

| 方式 | 适应证 |
|---|---|
| **介入栓塞(coiling)** | ISAT trial 后多数动脉瘤首选 — 后循环 / 解剖适合 / 老年高危 |
| **外科夹闭(clipping)** | 解剖不适合介入(MCA 分叉)/ 大血肿需清除 / 介入失败 |
| 流转分流 / 支架 / 球囊辅助 | 复杂动脉瘤 |

### 3. 预防迟发性脑缺血(DCI)/ 血管痉挛

- **[[尼莫地平]] 60 mg q4h PO × 21 天**(I 类推荐)
  - 给药途径以**口服 / 鼻饲**为主,IV 仅特殊情况(低血压风险)
  - 来源:NEJM 1989(Pickard);AHA 2023;中国 SAH 2019
- 维持等容 / 轻度高容(避免低血容)
- TCD(经颅多普勒)+ CT 灌注 / DSA 监测血管痉挛
- DCI 临床发生:升压维持脑灌注(Triple-H 已不再标准,改为 normovolemic + 升压)
- 球囊 / 化学血管成形术(顽固痉挛)

### 4. 并发症管理

| 并发症 | 处理 |
|---|---|
| **再出血** | 早期闭塞 + 血压控制 + 抗纤溶(氨甲环酸 短期可考虑,< 72h,长期增加 DCI) |
| **脑积水**(急性 ~ 20%) | 脑室外引流(EVD) |
| **抽搐** | 苯二氮䓬终止 + 短期抗癫痫预防;**长期抗癫痫只用于明确发作** |
| **低钠血症**(SIADH / CSWS) | 区分 SIADH(限水)vs CSWS(补钠 + 容量) — **CSWS 在 SAH 更常见** |
| **神经源性心肌损伤** | ECG / 肌钙蛋白 / 心超随访;支持治疗 |
| **神经源性肺水肿** | NIV / 机械通气支持 |
| **VTE** | 机械预防;肝素 2-3 天后启动(评估出血风险) |

## 🔺 升级标准

> [!danger]
> - 任何怀疑 SAH(雷击样头痛)→ 立刻 CT + 神外会诊
> - Hunt-Hess III-V 或 WFNS III-V → ICU + EVD 评估
> - GCS 突然下降 ≥ 2 → 警觉再出血 / 脑积水 → 急 CT
> - 新发抽搐 / 局灶定位
> - 血压急剧波动不响应控制
> - 低钠血症 < 130 + 神经恶化
> - 脑积水征(心室扩大、瞳孔异常、双侧上视障碍)
> - 任何"病人在恶化"

## ⚠️ 常见坑

> [!warning]
> 1. **雷击样头痛被当偏头痛 / 紧张性** → 漏诊 SAH 是急诊医师最常见严重错误
> 2. **CT 阴性就排除** → > 6h CT 敏感性下降,**起病 ≥ 6h 必须 LP 看黄变**
> 3. **前哨头痛(渗漏)漏诊** → 数日-周后大破裂
> 4. **运动 / 性活动 / Valsalva 诱发头痛** → 警觉,不要当"用力性头痛"
> 5. **CT 阴性 LP 黄变阳性当"创伤性穿刺"** → 黄变需离心分光,RBC 三管比较
> 6. **血压控制延迟** → 再出血 24h 内最高;立即降至 SBP < 140-160
> 7. **尼莫地平没给 / 给错途径** → 21 天 PO/NG q4h,IV 易低血压
> 8. **抗纤溶用 > 72h** → 长期增加 DCI / 缺血卒中
> 9. **常规预防性抗癫痫** → 已弱化推荐,仅有发作或高危
> 10. **低钠血症一律按 SIADH 限水** → SAH 中 CSWS 更常见,**CSWS 限水会更糟**(应补钠 + 容量)
> 11. **动眼神经麻痹漏诊** → 后交通动脉瘤压迫,定位价值高
> 12. **Terson 综合征**(玻璃体下出血)漏诊
> 13. **认为"PNSAH 影响小不查血管造影"** → 仍需排除;典型 PNSAH 才安全
> 14. **再出血后果 = 死亡** → 任何 GCS 下降立即 CT
> 15. **AVM / RCVS 等其他原因漏诊** → CTA + DSA 全面评估
