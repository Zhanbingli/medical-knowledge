# 急性肺水肿(Acute Pulmonary Edema)

> [!summary] 30 秒摘要
> **红旗**:端坐呼吸 + 双肺湿啰音 + 粉红泡沫痰 + SpO2 ↓ + 出汗烦躁
> **首查**:ECG(排 ACS)+ BNP + hs-cTn + CXR + 床旁超声(B 线)
> **一线**:**坐位 + NIV / CPAP** + IV 硝酸甘油(SBP > 110)+ IV 呋塞米;**寻找诱因优先**
> **升级**:不响应 NIV / 休克 / SpO2 < 90% → ICU + 气管插管 / 机械通气 + 升压药

---

## 一句话定义

肺毛细血管液体进入肺间质和肺泡腔的急性事件,导致严重低氧血症、通气 / 换气衰竭。

- **心源性**(本文重点):左室充盈压急剧升高 → 肺静脉淤血(LVEF 任意)
- **非心源性**(此文不展开):ARDS、神经源性、TRALI、复张性、高原性等

---

## 病因 / 诱因(找到诱因 = 治疗一半)

| 类别 | 具体 |
|---|---|
| **缺血** | [[diseases/急性冠脉综合征|急性冠脉综合征]] — 任何急性肺水肿都先排 STEMI |
| **后负荷骤升** | 高血压急症、肾性高血压、嗜铬细胞瘤、停降压药 |

## 🚩 红旗(立刻警觉)

- **端坐呼吸 + 不能平卧**
- **双肺底湿啰音迅速向上扩展**(从底到全肺)
- **粉红泡沫痰**(经典晚期表现)
- 突发严重气促 + 大汗 + 烦躁 + 濒死感
- SpO2 < 90% 即使吸氧
- SBP 极高(高血压急症性)或极低(心源性休克)
- 三脚架、辅助肌、紫绀
- **奔马律 S3** + JVP 升高 + 下肢水肿
- 突发新发心脏杂音 → 警觉机械并发症
- ECG ST 改变 / 新发心律失常

## 首查(到床后 10 分钟内)

- [ ] **生命体征 + SpO2 + ECG(< 10 min)**
- [ ] **12 导联 ECG**(STEMI / NSTEMI / 心律失常 / RV 应变)
- [ ] **hs-cTn**(0h + 1h / 3h)
- [ ] **BNP / NT-proBNP**(< 100 强烈反对急性 HF;> 500 强烈支持)
- [ ] **CXR**(肺水肿 Kerley B 线、蝶翼影、心影增大、胸腔积液)
- [ ] **床旁超声(POCUS)**:
  - 肺(B 线 ≥ 3 个 / 区域 ≥ 2 区域 = 间质 / 肺泡水肿)
  - 心(LVEF、室壁运动、瓣膜、心包积液)
  - IVC(容量评估)
- [ ] CBC + 电解质 + 肾功 + 肝功 + 凝血 + 乳酸 + ABG
- [ ] 血气优先动脉(PaO2 / PaCO2 / pH)
- [ ] 静脉通路 ×2 + 心电监护 + 持续 SpO2

## 一线处理(modern 版,弃用 LMNOP 经典记忆)

### 0. 立即定位

| 表型 | 重点 |
|---|---|
| **高血压性 / Flash** APE(SBP > 140) | 扩血管 + 利尿 + NIV |
| 正常血压 APE | 利尿为主 + 谨慎扩血管 |
| **低血压 / 心源性休克 APE** | **不给硝酸甘油 / 利尿过急**;升压 + 正性肌力 + 寻找机械并发症 |

### 1. 体位 + 氧 + NIV / CPAP(每例必做)

- **坐位**(双脚下垂如条件允许)
- 氧:SpO2 < 90% 高流量
- **NIV / CPAP 早期启动**:CPAP 5-10 cmH2O 或 BiPAP — **降低气管插管率与 ICU 死亡率**(ESC 2021 I 类推荐)
- 不能配合 / 意识差 / 呕吐 / 不稳定 → 直接气管插管

### 2. 扩血管(SBP > 110 时首选)

| 药 | 起始 | 备注 |
|---|---|---|
| **硝酸甘油 IV** | 10-20 µg/min,每 5min 增 10-20,最大 200 µg/min | SBP > 110 时首选;头痛常见 |
| **硝普钠 IV** | 0.25-10 µg/kg/min | 强效双向扩血管,长期 / 大剂量警觉氰中毒 |
| ACEI / ARB / ARNI | 后期慢性管理,**不在急性期 IV 给** | — |

> 来源:ESC HF 2021;中国心力衰竭诊治指南 2018;具体剂量按本院方案 [需查证]。

### 3. 利尿(容量过负荷的核心)

| 药 | 用法 | 备注 |
|---|---|---|
| **呋塞米 IV** | 平时未用利尿者:20-40 mg IV;原已用口服利尿者:**等效或加倍 IV** | 单次或持续输注 |
| 持续输注 | 5-20 mg/h | 顽固性水肿可加用 |
| 联合 | 噻嗪类(美托拉宗)序贯阻断 | 顽固病例 |

> 来源:DOSE trial(NEJM 2011)— 大剂量 vs 小剂量、bolus vs 持续等效;ESC HF 2021;中国 HF 2018。

### 4. 心律失常处理

- 房颤 RVR + 不稳定 → 同步电复律
- 房颤 RVR + 稳定 → 控率(EF 保留用 β / 地尔硫卓;EF 降低用 β / 地高辛 / 胺碘酮)
- 高度 AVB → 阿托品 / 临时起搏

### 5. 病因治疗(决定预后)

- **STEMI / 高危 NSTEMI** → PCI / 再灌注
- 高血压急症 → IV 扩血管控制
- 急性瓣膜事件 → 紧急手术 / 介入
- 房颤 RVR → 控率 / 复律

### 6. 心源性休克 APE(注意!逻辑相反)

- **不给硝酸甘油 / 大剂量利尿**(已经低灌注)
- **升压药首选 [[drugs/去甲肾上腺素|去甲肾上腺素]]**(SSC / ESC HF 2021)
- 正性肌力:[[多巴酚丁胺]] 2-20 µg/kg/min(心肌抑制为主)
- 机械支持:IABP / VA-ECMO / Impella(本院条件)
- 立即心内 / ICU 会诊

### 7. 弱化 / 不再常规推荐

- **吗啡** — ESC 2021 弱化(**OPERA-HF 等观察性提示死亡率上升**),仅极焦虑且其他无效时考虑
- 大剂量利尿一开始就上 — 个体化,小剂量起效不差

## 🔺 升级标准

> [!danger]
> - NIV / CPAP 30-60min 不响应 → **气管插管**
> - SpO2 < 90% 即使高流量氧
> - 意识恶化 / pH < 7.25 / pCO2 进行性升
> - 心源性休克(SBP < 90 + 灌注差 + 乳酸 ↑)
> - 急性 STEMI / 大面积 NSTEMI → 立即导管室
> - 急性瓣膜并发症(新杂音 + APE)→ 心外科会诊
> - 顽固水肿不响应大剂量利尿 → CRRT / 超滤
> - 任何"病人在恶化"

## ⚠️ 常见坑

> [!warning]
> 1. **没排除 ACS** — 任何急性 APE 必查 ECG + 肌钙蛋白,STEMI 假装是"心衰"再灌注就晚
> 2. **低血压 APE 给硝酸甘油 / 大剂量利尿** → 灾难;先想心源性休克
> 3. **太晚启动 NIV** → 等病人喘到不能配合再插管,死亡率显著高于早 NIV
> 4. **吗啡常规给** → 现代证据不支持,可能增加死亡(谨慎)
> 5. **房颤 RVR + APE 不积极控率 / 复律** → 节律本身是诱因,光利尿无效
> 6. **慢性 HF 漏服药突发 APE,只利尿不重启长期治疗** → 出院再发反复
> 7. **老年女性 + 急性 APE + 突发新杂音漏听** → 急性 MR(腱索 / 乳头肌断裂)外科急诊
> 8. **BNP 受年龄、肾功能、肥胖影响** — 不能单凭 BNP 判定;NT-proBNP 在房颤升高
> 9. **ARDS 当 APE 给利尿** → 容量耗竭 + 不解决炎症,病情恶化
> 10. **嗜铬细胞瘤 / 肾动脉狭窄等可逆性高血压性 APE 漏诊** → 反复发作或顽固
> 11. **高原 / TRALI / 神经源性等非心源性肺水肿误处置** → 病因导向治疗才行
