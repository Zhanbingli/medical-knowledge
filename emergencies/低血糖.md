# 低血糖(Hypoglycemia)

> [!summary] 30 秒摘要
> **红旗**:任何意识 / 行为改变 + 糖尿病用药史 / 酗酒 / 长期禁食
> **首查**:**指尖血糖 < 2 分钟**(永远先于一切检查);严重 / 不能解释 → 抽"hypoglycemia panel"前先治
> **一线**:意识清 → 口服 15-20 g 糖;不清 / 不能吞咽 → **50% 葡萄糖 20-40 mL IV** 或 **胰高血糖素 1 mg IM**
> **升级**:磺脲类 / 长效胰岛素 / 反复发作 / 不明原因 → ICU + 持续输糖 + 内分泌

---

## 一句话定义

血糖低至引起症状或体征的水平。

- **生化阈值**:静脉血糖 < 3.9 mmol/L(< 70 mg/dL,ADA 1 级);< 3.0(< 54 mg/dL,2 级,需立即处理);**< 2.8 mmol/L 或意识改变 = 严重低血糖(3 级)**
- **Whipple 三联征**(诊断金标准):症状 + 实验室低血糖 + 升糖后症状缓解
- 糖尿病人即使血糖未到 3.9,**症状性下降快**也算

来源:ADA Standards of Care 2024 / Hypoglycemia Definition;Endocrine Society Hypoglycemia 2009(2023 update);中国 2 型糖尿病防治指南 2020(中华糖尿病杂志)。

## 病因(临床导向分类)

### 糖尿病人(占绝大多数)

| 原因 | 机制 |
|---|---|
| **胰岛素过量 / 错误注射** | 短效误为长效、剂量计算错、运动后未减量 |
| **磺脲类**(格列苯脲 / 格列美脲 / 格列齐特) | **长效,可持续 24-72h,易反复**;老年 / 肾功能差风险显著 |
| **格列奈类**(瑞格列奈) | 短效但仍可致 |
| **进餐错过 / 延迟** | 用药后未及时进食 |
| 运动量增加 | 胰岛素敏感性升高 |
| 肾功能下降 | 胰岛素 / 磺脲清除减慢 |
| 慢性肝病 | 糖原储备降低 |
| 联合用药 | β 阻滞剂(掩盖症状)、ACEI、磺胺、酒精 |

### 非糖尿病人

| 原因 | 提示 |
|---|---|
| **酒精** | 抑制糖异生;空腹饮酒后 6-36h |
| **败血症 / 严重感染** | 多器官衰竭 → 糖储备耗 |
| **肝衰竭 / 肝硬化** | 糖异生 / 糖原贮备 ↓ |
| **肾上腺功能不全 / Addison 危象** | 皮质醇缺乏 → 反向调节失败 |
| **垂体功能减退 / Sheehan** | 同上 |
| **甲状腺功能减退**(粘液水肿) | 慢性低血糖 |
| **胰岛素瘤** | 反复禁食 / 运动后,胰岛素 / C 肽不当升高 |
| **非胰岛细胞肿瘤**(IGF-2 分泌) | 大间叶肿瘤 / 肝细胞癌 |
| **胃旁路术后倾倒综合征** | 餐后 1-3h |
| **自身免疫性低血糖** | 抗胰岛素抗体 |
| **药物性**(意外 / 自伤) | 老人 / 心理疾患;查看家人药盒 |

## 临床表现(Cushing 三联机制 → 双系统症状)

### 自主神经(交感)— 早期警报

- **心动过速、心悸**
- **出汗**(冷汗、面色苍白)
- **震颤**
- **饥饿感、焦虑**
- **苍白、瞳孔散大**

> [!warning] **β 阻滞剂使用者**和 **长病程糖尿病(自主神经病变 / hypoglycemia unawareness)**自主神经症状可严重抑制 → 直接表现为意识改变

### 神经低血糖 — 严重 / 持续

- 意识模糊、行为异常(可如醉酒)
- 视物模糊、复视
- 头痛、构音障碍
- 局灶神经定位(可拟卒中)
- 抽搐
- 昏迷

## 鉴别诊断

| 疾病 | 区分点 |
|---|---|
| [[脑卒中]] | 局灶定位 + CT/MRI;**血糖即刻区分** |
| [[癫痫]] | 抽搐史、EEG;**测血糖排除** |
| [[阿片中毒]] | 瞳孔针尖 + 呼吸抑制 |
| [[酒精中毒]] | 呼气酒精;**酒精经常 + 低血糖共存** |
| 焦虑 / 惊恐发作 | 客观无低血糖,主观相似 |
| 心律失常 / 心源性晕厥 | ECG 异常 |
| [[败血症]] | 感染征象 + 低血糖可同时(感染消耗) |
| 低钠血症 / 高钠血症 | 电解质 |

> **临床准则**:任何意识改变 / 抽搐 / 局灶定位 → **指尖血糖优先于一切**

## 首查

### 立即(< 2 min)

- [ ] **指尖血糖 / POC glucose**(即使你"不怀疑")
- [ ] 静脉通路 + 抽血(后述 panel,**抽完才给糖**——**仅在病人不严重时**;严重直接给糖,事后查)

### 严重 / 不能解释 / 非糖尿病低血糖 → "Hypoglycemia panel"

> [!warning] **抽血必须在症状性低血糖时进行**,纠正后送检不准

- [ ] 静脉血糖(实验室确认)
- [ ] 胰岛素
- [ ] **C 肽**(区分外源性 vs 内源性)
- [ ] 胰岛素原(proinsulin)
- [ ] β-羟丁酸(酮体)
- [ ] **磺脲药 / 格列奈代谢物筛查**(尿 / 血;意外 / 自伤识别)
- [ ] 皮质醇 + ACTH(肾上腺功能)
- [ ] TSH + FT4
- [ ] GH / IGF-1 / IGF-2(怀疑非胰岛细胞肿瘤)
- [ ] 抗胰岛素抗体(自身免疫低血糖)

### 决定病因后的影像

- 胰岛素瘤:腹部 CT / MRI / 超声内镜 / 68Ga-DOTATATE PET
- 肾上腺:CT / MRI;垂体 MRI

## 一线处理

### 1. 意识清 + 能吞咽 — 口服

- **15-20 g 速效糖**:葡萄糖片、果汁 150 mL、糖水、蜂蜜
- 15 分钟后复测;未达标重复
- 改善后吃**复合碳水 + 蛋白质**(防反跳)

### 2. 意识不清 / 不能吞咽 / 严重 — 静脉

> [!danger]
> **50% 葡萄糖 20-40 mL IV**(成人;儿童 0.5-1 g/kg = 25% 葡萄糖 2-4 mL/kg)
> 
> - 起效 1-3 min
> - 大量重复必要时
> - **必须经粗静脉**(高渗,外渗致组织坏死)
> 
> 来源:ADA 2024 / 中国 2 型糖尿病 2020;具体本院规格 [需查证]。

### 3. 没有静脉通路 — IM / SC

- **胰高血糖素 1 mg IM / SC**(成人;儿 < 25 kg 0.5 mg)
- 起效 5-15 min
- **不适用磺脲低血糖**(胰岛素分泌仍被刺激)
- 不适用酒精 / 严重肝病 / 长期饥饿(糖原耗尽)

### 4. 病因导向(关键!)

| 病因 | 后续 |
|---|---|
| **磺脲类** | **持续 D5W / D10W 静脉输注 24-48h+ 监测,可达 72h** + **奥曲肽 50-100 µg SC q6-8h**(抑制胰岛素分泌)+ ICU 监护 |
| **长效胰岛素**(甘精 / 地特) | 持续静脉糖输注直至胰岛素效应消退(可 24-36h) |
| **酒精性** | 持续糖输注 + B1(防 Wernicke)+ 肝功支持 |
| 严重感染 | 抗生素 + 病因 + 持续监测 |
| Addison / 肾上腺危象 | **氢化可的松 100 mg IV**(经验)+ 容量复苏 |
| 胰岛素瘤 / 非胰岛细胞肿瘤 | 内分泌会诊 + 影像 + 手术评估 |

### 5. 持续监测

- 30-60 min 复测血糖直到稳定
- 严重 / 反复 / 长效原因 → ICU 监护
- 反向调节钝化者(自主神经病变)→ **较高目标血糖**(避免反复发作)

## 🔺 升级标准

> [!danger]
> - 磺脲类 / 长效胰岛素低血糖 → 持续输糖 + ICU(可反复 24-72h)
> - 反复纠正后又下降 → 警觉胰岛素瘤 / 非胰岛细胞肿瘤 / 故意中毒
> - 抽搐 / 持续意识不清不能纠正
> - 不明原因低血糖 + 非糖尿病 + Whipple 三联 → 诊断性筛查 + 内分泌会诊
> - Addison 危象怀疑 → 经验性 [[氢化可的松]] 不等结果
> - 故意 / 自伤(磺脲滥用)→ 心理 + 安全干预
> - 反复发作的糖尿病人 → 胰岛素方案重审 / 教育不足

## ⚠️ 常见坑

> [!warning]
> 1. **没第一时间测血糖** — 任何意识改变 / 抽搐 / 局灶 / 行为异常 → 指尖血糖永远先于 CT
> 2. **磺脲类病人纠正一次就放走** → 长效 24-72h,**必须留观 + 持续输糖 24-48h+**
> 3. **长效胰岛素病人观察时间太短** → 同上,可反复
> 4. **β 阻滞剂使用者无典型自主神经症状** → 直接昏迷
> 5. **糖尿病自主神经病变 / hypoglycemia unawareness** 无预警 → 调高血糖目标
> 6. **酒精性低血糖给糖未先给 B1** → 加重 Wernicke
> 7. **酗酒患者次日"宿醉"** → 隐藏低血糖
> 8. **胰高血糖素用于磺脲低血糖** → 无效;且消耗肝糖原后回落更糟
> 9. **非糖尿病人低血糖不抽 panel** → 错过诊断窗口,胰岛素瘤诊断延误数月-数年
> 10. **C 肽不查** → 区分内源 vs 外源关键(自伤性胰岛素中毒)
> 11. **儿童不明原因低血糖**漏掉先天代谢病(脂肪酸氧化障碍、糖原贮积病)
> 12. **昏迷糖尿病人血糖正常**,以为不是低血糖 → 可能纠正过的余像;问家属、护士 / 药盒
> 13. **故意 / 自伤** → 心理疾患糖尿病患者反复严重低血糖
> 14. **新生儿 / 婴儿低血糖**阈值不同 + 后果严重(脑损伤),专儿科处理
> 15. **不查皮质醇 / TSH** → 漏 Addison / 严重甲减

## 关键监测目标

| 人群 | 目标血糖 |
|---|---|
| 普通糖尿病(住院非危重) | 7.8-10 mmol/L(140-180 mg/dL,ADA 2024) |
| 危重 / ICU | 6.1-10 mmol/L(110-180 mg/dL) — 不严格 < 6.1 |
| 老年 / hypoglycemia unawareness | **较宽松目标**,如 8-12 mmol/L |
| 妊娠糖尿病 | 餐前 < 5.3 / 餐后 1h < 7.8 |

来源:ADA Standards of Care 2024;中国 2 型糖尿病防治指南 2020。
