# 肺栓塞(PE / PTE)

> [!summary] 30 秒摘要
> **红旗**:突发气短 + 心动过速 + 低氧 + 单侧腿肿 / DVT 危险因素
> **首查**:Wells/Geneva 评分 → D-dimer → CTPA;床旁:ECG + 心脏超声
> **一线**:吸氧 + 抗凝(LMWH/UFH);**massive PE → 溶栓**
> **升级**:massive PE(SBP<90 或休克)立即 ICU + 溶栓决策

---

## 一句话定义

栓子(多为下肢深静脉血栓)阻塞肺动脉系统,导致 V/Q 失调、低氧、右心后负荷骤增甚至休克。

---

## 风险分层(决定治疗激进度)

| 等级 | 标准 | 30 天死亡率 | 处理 |
|---|---|---|---|
| **High-risk(massive)** | SBP<90 持续 ≥15min 或 SBP 较基线降>40 / 休克 | 25-50% | **溶栓 / 取栓** |
| **Intermediate-high(submassive)** | RV 功能不全(超声) **+** 心肌损伤标志物升高 | 5-15% | 抗凝 + 监护;恶化 → 溶栓 |
| **Intermediate-low** | RV 功能不全 **或** 标志物升高(单一) | 3-5% | 抗凝 + 监护 |
| **Low-risk** | 无 RV 不全、无标志物升高、sPESI=0 | <1% | 抗凝(可门诊) |

---

## 🚩 红旗(立刻警觉)

- **突发**气短(分钟级)
- 胸痛(胸膜性,深呼吸加重)
- 心动过速(**最常见体征**)
- SpO2 下降 / 低氧不能解释
- 咯血(<30%)
- 晕厥 + 低氧 = **大面积 PE 警报**
- **单侧下肢肿胀 / 压痛**(DVT)
- 颈静脉怒张 + 心动过速 + 低血压 = RV 衰竭三联征

---

## 危险因素(Virchow 三联征)

| 类别 | 具体 |
|---|---|
| **血液高凝** | 肿瘤、口服避孕、HRT、妊娠/产褥、肾病综合征、易栓症 |
| **血流淤滞** | 制动 ≥3 天、长途旅行 ≥4h、术后(尤其骨科)、瘫痪 |
| **血管损伤** | 创伤、手术、中心静脉置管、骨折 |

## 首查 + 决策路径

### 临床预测(先做这个)

> [!tip] 流程顺序很关键
> 不要见胸痛就 D-dimer / CTPA。先**评分 + 低度怀疑用 PERC** 排除。

```
血流动力学稳定?
├── 否 → 立即床旁超声(看 RV 功能)→ CTPA / 床旁诊断 → 溶栓
└── 是 → Wells/Geneva 评分
        ├── 低度怀疑 + PERC 全部阴性 → 不需要 D-dimer,排除
        ├── 低/中度怀疑 → D-dimer
        │   ├── 阴性(年龄校正)→ 排除
        │   └── 阳性 → CTPA
        └── 高度怀疑 → 直接 CTPA
```

> **年龄校正 D-dimer**:>50 岁,阈值 = 年龄 × 10 (µg/L FEU)。例:70 岁阈值 700。

### 床旁检查

- [ ] ECG:窦速最常见;典型 **S1Q3T3**(<25%);RBBB 新发;V1-V4 T 波倒置(RV 应变)
- [ ] CXR:多正常;Westermark 征(局部少血)、Hampton 驼峰(楔形浸润)罕见但提示
- [ ] ABG:低氧 + 低 CO2 + A-a 梯度增大(正常 ABG 不能排除)
- [ ] **床旁心脏超声**:RV 扩张/低动 / McConnell 征 / D 形左室——massive PE 床旁可立即诊断
- [ ] D-dimer:RULE OUT 用,不能 RULE IN
- [ ] **CTPA**:确诊金标准
- [ ] 替代:V/Q 扫描(肾功能差、孕妇、造影剂禁忌)
- [ ] 风险分层:**肌钙蛋白 + BNP**(intermediate 与 low 区分)
- [ ] 下肢静脉超声(DVT 阳性 + 临床符合 = 等同于 PE 确诊)

## 一线处理

### 共同处理

1. **氧疗**:SpO2 < 90% 时给氧
2. **静脉通路 + 监护 + ABG**
3. **维持血压**:晶体补液 ≤500 mL(谨慎,RV 衰竭过补会更糟)→ 升压药首选**去甲肾上腺素**
4. **抗凝**(无禁忌即给,等待影像不要等)

### 抗凝选择

| 情境 | 首选 | 备注 |
|---|---|---|
| 稳定、不计划溶栓 | LMWH(依诺肝素 1 mg/kg sc q12h) | 最常用 |
| 不稳定、可能溶栓、术前 | **UFH IV**(80 U/kg → 18 U/kg/h,APTT 1.5-2.5x) | 半衰期短可逆 |
| 严重肾功能不全(CrCl<30) | UFH 首选 | LMWH 蓄积 |
| 妊娠 | LMWH | 华法林禁用 |
| 长期 | DOAC(利伐沙班 / 阿哌沙班)或华法林(VTE 抗凝 ≥3 个月) | DOAC 简便 |

### Massive PE / 不稳定 → 溶栓

| 适应证 | 用药 |
|---|---|
| SBP < 90 持续 / 休克 / 心搏骤停 | **阿替普酶 100 mg over 2h**(或心搏骤停时 50 mg bolus) |
| 替代 | 链激酶 / 尿激酶 |

**禁忌**:活动性出血、近期颅内手术 / 卒中、近期重大创伤、严重未控制高血压。

> [!warning] **Submassive PE 是否溶栓有争议**——给上级判断,通常需要血流动力学恶化倾向再考虑。

### 介入 / 外科

- 溶栓禁忌或失败 → **导管取栓(CDT)** 或外科取栓
- 抗凝禁忌 + 复发风险高 → **IVC 滤器**

---

## 🔺 升级标准

> [!danger]
> - SBP < 90 / 持续休克 → ICU + 溶栓决策
> - SpO2 < 90% 不能纠正 → 高级气道
> - RV 功能恶化(超声、肌钙蛋白上升)
> - 反复晕厥
> - 抗凝下仍有血栓进展
> - 任何"病人在恶化"的征象

---

## ⚠️ 常见坑

> [!warning]
> 1. **气短归因为"焦虑"**——年轻女性 + 突发气短 + 心动过速,排 PE 再说焦虑。
> 2. **正常 ABG / CXR 不能排除 PE**。
> 3. **D-dimer 在妊娠、肿瘤、术后、感染都假阳性**——只能用来 RULE OUT,且必须结合预测概率。
> 4. **Saddle PE 起初血流动力学可稳定**,但随时崩——别因稳定就掉以轻心。
> 5. **晕厥 + PE = 高死亡率标志**,特别警觉。
> 6. **下肢超声阴性不能排除 PE**(已有栓子脱落上肺去了)。
> 7. **抗凝足量下复发** → 排查肿瘤(尤其腺癌)。
> 8. **慢性血栓栓塞性肺动脉高压(CTEPH)** 急性 PE 后 6 周仍气短 → 心超 + V/Q 扫描。
> 9. **心搏骤停 + 怀疑 PE** → CPR 中可经验性溶栓 50mg。
> 10. **孕妇 PE** D-dimer 几乎都高,流程不同(下肢超声优先;V/Q 比 CTPA 辐射更友好)。

## 关键评分

- **[[scores/Wells-PE|Wells-PE]]**:0-12.5,>4 中-高度怀疑(简化版 ≥2)
- **[[Geneva]]**(修订简化版):无主观判断项,适合不熟病史者
- **[[PERC]]**:8 项全阴 + 临床低度怀疑 → 排除,不需 D-dimer
- **[[sPESI]]**:0 分 = 30 天死亡率 <1%(可门诊抗凝)
- **[[PESI]]**:I-V 级,完整版死亡率分层
