# 急性缺血性脑卒中(AIS)

> [!summary] 30 秒摘要
> **红旗**:突发局灶神经定位 — FAST / BE-FAST 任一阳性
> **首查**:**血糖 + 头 CT < 25 分钟**(door-to-CT)+ NIHSS + 末次正常时间(LKW)
> **一线**:**4.5h 内静脉溶栓** / **6-24h 大动脉闭塞取栓**(影像选择)
> **升级**:Door-to-needle ≤ 60 min 是质控指标——**任何怀疑立刻启动卒中通道**

---

## 一句话定义

脑动脉急性闭塞 / 严重狭窄导致脑组织缺血、神经功能急性丧失。
- **占所有卒中 ~ 80%**
- **每延误 1 分钟约死亡 190 万神经元 / 14 亿突触**(Saver 2006 经典数据)
- **时间就是大脑(Time is brain)**

来源:AHA / ASA Acute Ischemic Stroke 2019(2024 update);中国急性缺血性脑卒中诊治指南 2018(中华神经科杂志)。

## 分型(决定治疗方向)

### 按病因(TOAST 分型)

| 类型 | 占比 | 提示 |
|---|---|---|
| **大动脉粥样硬化** | ~ 30% | 颈动脉 / 椎基底狭窄;CTA / MRA 评估 |
| **心源性栓塞** | ~ 25% | 房颤、IE、心内膜血栓、卵圆孔未闭 |
| **小动脉闭塞**(腔隙) | ~ 25% | 高血压 + 糖尿病背景;深部小灶 |
| 其他确定病因 | ~ 5% | 夹层、血管炎、易栓症 |
| 不明原因 | ~ 15% | 称 ESUS;需进一步评估 |

### 按部位

| 部位 | 临床特征 |
|---|---|
| **前循环 — MCA**(最常见)| 对侧偏瘫(面 / 上肢重于下肢)、失语(优势侧)、忽视(非优势侧)、同向偏盲 |
| 前循环 — ACA | 对侧下肢偏瘫(下肢重于上肢)、尿失禁、淡漠 |
| **后循环 — 基底动脉** | 意识改变、四肢瘫、眼球运动 / 瞳孔异常、共济失调、**Locked-in 综合征** |
| 后循环 — PCA | 同向偏盲、视幻觉、记忆障碍 |
| 小脑 / 脑干 | 头晕、共济失调、眼震、构音 / 吞咽困难、对侧感觉障碍 |
| 腔隙 | 5 大经典综合征(纯运动 / 纯感觉 / 共济偏瘫 / 构音手笨拙 / 感觉运动) |

## 🚩 红旗(立刻警觉)

> [!danger] FAST → BE-FAST(增加后循环识别)
> - **B**alance — 突发头晕 / 共济失调
> - **E**yes — 突发视野缺损 / 复视
> - **F**ace — 面部偏垂
> - **A**rms — 上肢无力 / 偏侧
> - **S**peech — 言语含糊 / 失语
> - **T**ime — 立即记录末次正常时间(LKW),启动通道

**其他红旗**:
- 突发剧烈头痛 + 意识改变 → 警觉 [[蛛网膜下腔出血]] + 椎基底闭塞
- 突发眩晕 + 共济失调 + 眼震 → 后循环卒中(常被误为前庭外周)
- 老年突发"乏力"或"不愿讲话"→ 优势半球失语
- **醒后卒中 / 不能确定起病时间** → DWI-FLAIR 错配影像评估
- 房颤 / 机械瓣 / 近期 MI 患者突发神经症状

## 鉴别诊断(Stroke Mimics ~ 20-30% 急诊"卒中")

| 疾病 | 区分点 | 关键检查 |
|------|--------|---------|
| **[[低血糖]]** | 意识 / 局灶可拟卒中,血糖即刻发现 | **每例必查血糖,<2 分钟** |
| [[癫痫后状态]](Todd 麻痹) | 抽搐史 + 麻痹缓慢消退 | 病史 + EEG |
| 偏头痛伴先兆 | 闪光暗点 → 神经症状,既往偏头痛史 | 病史 |
| **[[蛛网膜下腔出血]]** | "雷击样"剧烈头痛 + 颈强 | CT(6h 内 > 95% 敏感)+ LP |
| 颅内出血 | 不影像无法区分 | CT 平扫 |
| 脑炎 / 脑膜炎 | 发热 + 头痛 + 颈强 + 意识 | LP |
| 中枢神经系统肿瘤 | 亚急性进展、影像占位 | MRI |
| 转换障碍(功能性) | 不符合解剖、检查不一致 | **排除性诊断,必须先 CT + MRI** |
| [[心源性晕厥]]后定位假象 | 短暂 | ECG + 心电监护 |
| Wernicke 脑病 | 营养不良 + 三联征 | B1 经验性给 |
| **[[主动脉夹层]]累及颈动脉** | 撕裂胸背痛 + 卒中表现 + 双臂血压差 | **CTA**(任何 ACS 不典型 + 神经定位) |

## 首查(到院后 25 分钟内 CT 完成)

> [!warning] AHA / ASA 时间目标
> - Door-to-CT ≤ 25 min
> - Door-to-CT 报告 ≤ 45 min
> - **Door-to-Needle ≤ 60 min**(国家级卒中中心 ≤ 45 min)
> - Door-to-Groin(取栓)≤ 90 min

### 立即(< 5 min)

- [ ] **指尖血糖**(< 2.8 严重低血糖,纠正再评估)
- [ ] 生命体征 + SpO2 + 心电监护
- [ ] 静脉通路 ×2 + 抽血(CBC、生化、凝血、肌钙、血型)
- [ ] **NIHSS 评分**(基线,沟通预后)
- [ ] **末次正常时间(Last Known Well, LKW)**——决定时间窗
- [ ] 简明既往史:既往卒中、抗凝、近期手术 / 出血、肿瘤、孕妇

### 影像(< 25 min)

- [ ] **头 CT 平扫**(排除出血)
- [ ] **CT 灌注 / CTA**(评估大动脉闭塞 LVO + 缺血核心 / 半暗带)
- [ ] 醒后 / 时间不明 → MRI DWI-FLAIR 错配
- [ ] 颈动脉 + 椎基底动脉 CTA(必要时延伸主动脉弓)
- [ ] **ASPECTS 评分**(MCA 区域早期缺血)

### 实验室

- 血糖、电解质、肌酐、肝功、PT / INR / APTT、肌钙蛋白、ABO 血型
- DOAC 监测(达比加群 dTT;Xa 抑制剂 anti-Xa)
- 妊娠试验(育龄女性溶栓决策)

## 一线处理

### 1. 时间窗内静脉溶栓(IV-tPA)

> [!danger] 时间窗
> - **0-4.5h**:大多数患者(2018 ECASS-IV 等扩展)
> - **4.5-9h** 或 醒后卒中 + DWI-FLAIR 错配:特定患者(EXTEND, WAKE-UP)— 经验中心
> - **替奈普酶 (TNK)** 在 NEJM 2022 等多 RCT 后已广泛替代 alteplase(EXTEND-IA TNK, ATTEST-2)

**剂量(以阿替普酶为代表)**:
- **总量 0.9 mg/kg(最大 90 mg)**
- 10% bolus 1 min IV → 余量 60 min IV
- 替奈普酶:0.25 mg/kg(最大 25 mg)单次 IV bolus

来源:AHA / ASA AIS 2019 + 2024 update;中国急性缺血性脑卒中诊治指南 2018(中华神经科杂志)。

### 2. 大动脉闭塞(LVO)血管内取栓

> [!danger] 时间窗
> - **0-6h** 标准:CT + CTA 确认 LVO + ASPECTS ≥ 6
> - **6-24h** 选择性:DAWN / DEFUSE-3 标准(影像 / 临床错配)
> - 优先取栓 + 同步 IV 溶栓 / 或单纯取栓(SWIFT-DIRECT, MR-CLEAN-NOIV 等)

### 3. 血压管理

| 情境 | 目标 |
|---|---|
| 溶栓前 / 溶栓中 | **SBP < 185 / DBP < 110** |
| 溶栓后 24h | **SBP < 180 / DBP < 105** |
| 不溶栓 | 急性期"许可性高血压"(SBP < 220 / DBP < 120 通常不降) |

降压药:**[[拉贝洛尔]] 10-20 mg IV q10min**(最大 300mg)/ **[[尼卡地平]] 5 mg/h IV**(每 5min 加 2.5 mg/h,最大 15 mg/h)。

来源:AHA / ASA AIS 2019;中国 AIS 2018;具体本院药品规格 [需查证]。

### 4. 抗血小板 / 抗凝

- **溶栓后 24h** 内**禁用抗血小板和抗凝**
- 24h 复查 CT 无出血 → 阿司匹林 + 氯吡格雷(高危)/ 阿司匹林单药
- 心源性栓塞 → 长期口服抗凝(房颤用 DOAC 或华法林)
- **小卒中 / TIA + ABCD2 高 → 双抗 21 天**(CHANCE / POINT trials)

### 5. 支持

- SpO2 < 94% → 给氧
- 体温 > 38℃ → 退热(高热加重缺血)
- 血糖目标 7.8-10 mmol/L(避免 < 3.9 或 > 11.1)
- VTE 预防、DVT 物理预防
- 吞咽筛查 → 防误吸

## 🔺 升级标准(立刻叫上级 / 启动通道)

> [!danger]
> - 任何怀疑卒中 → **立刻启动卒中绿色通道**,不要走常规挂号 / 候诊
> - NIHSS ≥ 6 → 怀疑 LVO,准备取栓评估
> - 醒后 / 时间不明卒中 → MRI 错配评估通道
> - 后循环卒中(基底动脉)→ 时间窗放宽但预后差,立即上级
> - 溶栓后 24h 内任何神经恶化 / 头痛 / 呕吐 / 高血压 → 警觉**症状性颅内出血**,立即头 CT
> - 溶栓相关血管性水肿(舌 / 唇)→ 停溶栓 + 抗组胺 + 类固醇 + 准备气道
> - 大面积脑梗 + 占位 / 意识恶化 → 警觉脑疝 → 神外评估去骨瓣
> - 任何"病人在恶化"

## ⚠️ 常见坑

> [!warning]
> 1. **没问 LKW(末次正常时间)** → 时间窗误判,患者错失溶栓
> 2. **没查血糖** → 低血糖可拟卒中,溶栓下去就出事
> 3. **后循环卒中漏诊** → 头晕 / 共济失调 / 复视常被当外周性眩晕(HINTS 检查可助识别)
> 4. **CT 阴性就排除** → 早期 CT 多正常,**临床高度怀疑 + 时间窗内仍可溶栓**
> 5. **基底动脉血栓不识别** → 时间窗放宽但预后差,需 CTA 主动看
> 6. **溶栓前没核对绝对禁忌**(近期颅内出血、3 月内重大手术、活动性出血、INR > 1.7、血小板 < 100、AVM / 瘤、夹层等)
> 7. **降压目标错** → 急性期普遍卒中过度降压加重缺血;但**溶栓相关目标必须严**
> 8. **溶栓后 24h 内给抗栓** → 出血风险陡增
> 9. **房颤新发卒中早期就给抗凝** → 出血转化风险高,2-14 天后启动按面积权衡
> 10. **dissection / 主动脉夹层未识别** → 颈动脉 / 椎动脉夹层是中青年卒中重要原因;溶栓前 CTA 检查
> 11. **DOAC 患者溶栓** → 末次给药 < 48h 多视为禁忌;特定逆转剂(idarucizumab / andexanet)+ 个体化决策
> 12. **吞咽未筛查就喂食** → 误吸性肺炎
> 13. **小脑梗死 + 占位** → 后颅窝有限空间,水肿压迫脑干 → 神外去骨瓣指征
> 14. **Stroke mimic 误溶栓** ~ 1-3% 出现,影响低但应识别(癫痫、转换、低血糖)

## 关键评分

- **[[NIHSS]]**:0-42,基线 + 随访;> 6 强烈提示 LVO;> 25 严重大面积
- **[[ASPECTS]]**(0-10):MCA 早期缺血改变,< 6 取栓获益减
- **[[mRS]]**:出院 / 90 天功能预后(0-6)
- **[[CHA2DS2-VASc]]**:房颤患者卒中风险(评估抗凝)
- **[[ABCD2]]**:TIA 后短期卒中风险
