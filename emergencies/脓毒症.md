# 脓毒症(Sepsis)

> [!summary] 30 秒摘要
> **红旗**:疑似感染 + qSOFA ≥ 2 / 乳酸 > 2 / 低血压 / 意识改变
> **首查**:乳酸 + 血培养 ×2 + 感染源筛查 + CBC/生化/凝血
> **一线**:**1 小时 bundle**——乳酸 / 血培养 / 广谱抗生素 / 30 mL/kg 晶体(休克或乳酸 ≥ 4)/ 升压药维持 MAP ≥ 65
> **升级**:任何疑似 → 立刻打电话,不等培养结果

---

## 一句话定义

**Sepsis-3(JAMA 2016)定义**:宿主对感染的失调反应导致危及生命的器官功能障碍。
- 临床操作:疑似感染 + **SOFA 较基线增加 ≥ 2 分** = 脓毒症
- 床旁筛查:**qSOFA ≥ 2**(意识改变 / SBP ≤ 100 / RR ≥ 22)= 高风险

**脓毒性休克(Septic Shock)**:充分液体复苏后仍需升压药维持 MAP ≥ 65 + 乳酸 > 2 mmol/L,死亡率 ≥ 40%。

来源:Sepsis-3 共识(JAMA 2016);Surviving Sepsis Campaign 2021;中国脓毒症 / 脓毒性休克急诊治疗指南 2018(中华急诊医学杂志)。

## 分型

| 阶段 | 定义 | 临床意义 |
|---|---|---|
| **疑似感染** | 临床有感染倾向 | 启动筛查 |
| **脓毒症** | 感染 + SOFA Δ ≥ 2 | 1h bundle 启动 |
| **脓毒性休克** | 复苏后仍需升压药 + 乳酸 > 2 | ICU + 多器官支持 |

> [!note] 旧的 SIRS / 严重脓毒症 概念已被 Sepsis-3 弃用,但中国部分医院仍沿用。临床流程可能两种术语混用。

## 🚩 红旗(立刻警觉)

- 疑似感染 + 任何 qSOFA 异常(意识 / SBP / RR)
- **乳酸 > 2** 不能用其他原因解释;**> 4** 立刻 ICU
- SBP < 90 或较基线下降 > 40
- MAP < 65
- 尿量 < 0.5 mL/kg/h 持续 > 2h
- 皮肤花斑、湿冷、毛细血管再充盈 > 3 秒
- 新发意识改变 / 烦躁 / 嗜睡
- **老年 / 免疫抑制 / 化疗后可无发热**——警觉低体温 < 36℃
- 不褪色瘀点 / 紫癜 → 警觉 [[脑膜炎球菌病]]
- 中性粒细胞缺乏 + 任何"不对劲"= [[中性粒细胞减少性脓毒症]]

## 首查(到床后 1 小时内,即 1h bundle 内涵)

- [ ] **生命体征 + qSOFA + 意识 + 尿量**(立即)
- [ ] **乳酸**(< 1h);若 > 2 → 复测 q2-4h 直至正常化
- [ ] **血培养 ×2 套**(不同部位,**抗生素前**;不应因等待延误抗生素 > 1h)
- [ ] **CBC + 分类、CRP / PCT、肝肾功、电解质、凝血、血糖**
- [ ] **ABG + 乳酸 + 静脉血气**
- [ ] 感染源定向培养:尿培养 / 痰培养 / 脑脊液 / 伤口分泌物 / 导管尖端
- [ ] 影像:CXR / 胸腹 CT / 床旁超声(POCUS:胆 / 肾 / 心 / 腹腔)
- [ ] 妊娠试验(育龄女性)
- [ ] 心电图 + 肌钙蛋白(鉴别 / 心肌损伤)
- [ ] **2 条静脉通路 + 心电监护 + SpO2 + 中心静脉(若需升压药)**

## 一线处理(1 小时 Bundle)

> [!danger] SSC 2021 1-Hour Bundle(脓毒症诊断起 1 小时内必须完成)

### 1. 测乳酸
- 起始 + 每 2-4h 复测直至正常化
- > 4 mmol/L = 重度低灌注 → 立即升级
- > 2 mmol/L 在复苏后未降 → 强烈支持脓毒性休克

### 2. 血培养(抗生素前)
- 2 套不同部位
- **但不应因等待培养延误抗生素 > 1h**

### 3. 广谱抗生素(经验性,1h 内)
| 感染源 | 经验首选(具体方案按本院 / 当地耐药) |
|---|---|
| 来源不明 + 社区 | β-内酰胺 / β-内酰胺酶抑制剂(如哌拉西林他唑巴坦)± 万古(警觉 MRSA) |
| 来源不明 + 院内 / 长期住院 | 碳青霉烯 ± 万古;高危再加抗真菌 |
| 中性粒缺(FN) | 抗假单胞 β-内酰胺(哌拉他唑 / 头孢他啶 / 美罗培南); 不稳定加万古 |
| 疑似 [[diseases/细菌性脑膜炎|细菌性脑膜炎]] | 头孢曲松 + 万古 + 地塞米松(± 阿莫西林覆盖 Listeria) |
| 疑似 [[腹腔感染]] | 哌拉他唑 / 厄他培南 / 头孢 + 甲硝唑 |
| 疑似 [[尿源性]] | 头孢 / 哌拉他唑 / 碳青霉烯 |
| **抗生素覆盖原则** | 确诊前广,**24-48h 内根据培养降阶梯** |

> 具体药物选择 [需查证: 本院抗菌药物分级管理 + 当地耐药数据]

### 4. 液体复苏
- **休克 / 乳酸 ≥ 4** → 30 mL/kg 晶体(乳酸林格优于生理盐水)首 3 小时内
- 后续按动态监测(被动抬腿试验、SVV、IVC 变异、超声)指导
- 避免过度补液(肺水肿、腹腔间隔综合征)

### 5. 升压药(液体不足维持 MAP 时)
| 药 | 剂量起始 | 备注 |
|---|---|---|
| **去甲肾上腺素**(首选) | 0.05-0.1 µg/kg/min IV,滴定 | MAP 目标 ≥ 65 |
| **加压素** | 0.03 U/min(联用,非首线) | 减去甲剂量 |
| **肾上腺素** | 0.05-0.5 µg/kg/min | 第二线;心源性叠加 |
| **多巴酚丁胺** | 2-20 µg/kg/min | 心肌抑制为主时加 |

> 来源:SSC 2021;中国脓毒症共识 2018;具体起始剂量与滴定速度 [需查证: 本院 ICU 方案]。

### 6. 后续(non-bundle 但同样重要)
- **感染源控制**:导管 / 脓肿 / 坏死组织 → 24h 内尽早处理
- **氢化可的松 200 mg/d** IV(分次或持续)— 升压药足量仍不响应时(SSC 2021 弱推荐)
- 血糖目标 < 10 mmol/L(180 mg/dL),避免严格控糖
- VTE 预防 + 应激溃疡预防 + 营养支持
- 机械通气(肺保护性通气 6 mL/kg PBW + plateau ≤ 30)

## 🔺 升级标准(立刻叫上级 / ICU)

> [!danger]
> - 脓毒性休克:升压药维持 MAP < 65 / 乳酸 > 2 不降
> - 多器官功能障碍(SOFA Δ ≥ 2 多脏器)
> - 急性肾损伤 / 少尿 / 无尿
> - ARDS / SpO2 < 90% 高流量氧仍不能维持
> - 弥散性血管内凝血(DIC):血小板 < 100、PT 延长、FDP / D-dimer 升、纤维蛋白原 < 1.5
> - 意识进行性下降 / GCS ≤ 13
> - 严重酸中毒 pH < 7.25 不能纠正
> - 任何"病人在恶化"的临床直觉
> - 中性粒缺 + 不稳定 → ICU + FN 流程
> - 怀疑感染源需手术控制(脓肿 / 坏死 / 穿孔 / 胆管炎)

## ⚠️ 常见坑

> [!warning]
> 1. **没及时给乳酸 + 血培养** → 1h bundle 启动失败,后续治疗效果差
> 2. **等培养结果再给抗生素** → 每延误 1h 死亡率上升,**不可接受**
> 3. **老年 / 免疫抑制无典型发热** → 低体温 < 36℃ 同样是红旗;NEWS2 / qSOFA 比体温更敏感
> 4. **SIRS 当脓毒症** → 胰腺炎、创伤、烧伤、自身免疫激活也 SIRS;需要"感染"+"器官功能障碍"
> 5. **过度补液** → 30 mL/kg 是初始目标,不是无限,反复评估容量反应性
> 6. **去甲肾上腺素延迟启动** → 液体复苏中或后立即启动,不等"补够"
> 7. **没找感染源** → 抗生素再多也救不了未控制的源(脓肿、坏死、感染导管)
> 8. **没降阶梯** → 培养出敏感菌后仍用广谱 → 二重感染、艰难梭菌、耐药菌
> 9. **氢化可的松滥用** → SSC 2021 仅在升压药足量仍不响应时弱推荐,不应一开始就给
> 10. **冷休克被忽视**(儿科 / 部分老年) → 末梢冷 + 灌注差但血压可暂正常
> 11. **HCG 漏问** → 育龄女性脓毒症 + 妊娠改变抗生素与影像决策
> 12. **导管相关血流感染漏诊** → 长期留置导管 + 不明原因脓毒症 → 拔管 + 导管尖端培养
> 13. **暖休克(早期高排低阻)被当"还行"** → 末梢温暖、皮肤发红、SBP 暂正常但乳酸已升

## 关键评分

- **[[scores/qSOFA|qSOFA]]**:床旁筛查,3 项各 1 分(SBP ≤ 100 / RR ≥ 22 / 意识改变),≥ 2 = 高风险
- **[[SOFA]]**:6 系统器官功能(呼吸 / 凝血 / 肝 / 心血管 / 神经 / 肾),Δ ≥ 2 = 脓毒症诊断
- **[[NEWS2]]**:英国早期预警,7 项体征综合,5 分以上启动加强观察
- **[[APACHE II]]**:ICU 入科评估死亡率
- **[[MEWS]]**:中国部分医院使用的改良早期预警
- **[[Lactate]]**:不算评分但作为器官灌注代理指标:> 2 警觉,> 4 立即升级
